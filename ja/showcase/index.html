Guide

スクリプトでワンクリックインストール

スクリプトコマンドで Qwen Code を素早くインストール、数秒で環境設定完了。

# Linux/macOS curl -fsSL https://qwen-code.oss-cn-beijing.aliyuncs.com/install.sh | bash -s -- --install # Windows (PowerShell) curl -fsSL -o %TEMP%install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%install-qwen.bat
Guide

最初の会話を始める

インストール完了後、Qwen Code との最初の AI 会話を開始。

what is qwen code? Please give me a brief introduction about it.

Guide

Headless モード

GUI なしの環境で Qwen Code を使用、リモートサーバーや CI/CD シナリオに最適。

Guide

@file 参照機能
