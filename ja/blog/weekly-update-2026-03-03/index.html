Qwen Code 週報：Insight 分析、クリップボード画像、Terminal Capture

Qwen Team
2026-03-03

今週はv0.10.0、v0.11.0の主要機能バージョンと5つのバグ修正バージョンをリリースしました。

  • 2つの主要機能バージョンv0.10.0とv0.11.0、および5つのバグ修正バージョンをリリース
  • OAuth認証、モデルをqwen3.5-plusにアップグレード、1日1000回の無料呼び出し
  • Insight分析を追加
  • クリップボード画像サポート
  • Terminal Capture自動検証とスクリーンショット生成

✨ 新機能

Insight 分析

ローカルSessionレコードを深く分析し、パーソナライズされたプログラミングインサイトと使用状況分析レポートを提供する公式組み込みコマンド。

使用方法：/insightと入力して並列LLM分析を開始

7つのセクションに分かれた詳細なHTMLレポートを生成：

  • What You Work On - あなたの作業焦点
  • How You Use Qwen Code - Qwen Codeの使用方法
  • Impressive Things - 注目すべき成果
  • Where Things Go Wrong - 問題のある領域
  • Features to Try - 試す価値のある機能
  • New Usage Patterns - 新しい使用パターン
  • On the Horizon - 将来の展望

最後に、ソーシャルメディア共有用の画像をエクスポートできます。

クリップボード画像サポート

クロスプラットフォーム対応：macOS / Windows / Linux。PR #1612 を参照

機能：

  • 画像キャプチャ：システムクリップボードから画像をキャプチャ
  • 一時保存：クリップボード一時ディレクトリに保存
  • LRUクリーンアップ：100枚を超えると最も古い50枚を自動削除
  • 添付UI：↑で添付モード、←/→でナビゲーション、Deleteで削除

Terminal-Capture

用途：CLI視覚検証の自動化された端末スクリーンショットツール、機能を自動検証しデモスクリーンショットを生成

アーキテクチャ：node-pty（疑似端末）→ xterm.js（レンダリング）→ Playwright（スクリーンショット）、PR #1840 を参照

👇 機能デモ

MCPツール進捗更新

機能：MCPツールがリアルタイム進捗通知をサポート

  • TUIモード：⏳ [50/100] Processing...を表示
  • SDKモード：新しいtool_progressストリームイベントタイプ

PR #1756 を参照
