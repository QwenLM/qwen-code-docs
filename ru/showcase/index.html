Быстрый опыт: Смена темы терминала

Ежедневное использование

Автоматическая генерация еженедельных отчётов

Пользовательский навык—команда в одну строку для автоматического получения обновлений недели и написания еженедельных отчётов обновлений продукта на основе заданных шаблонов.

Ежедневное использование

Презентация: Создание PPT

Создайте PPT из скриншотов демо продукта—именно то, что вы сейчас видите. Просто предоставьте скриншоты, введите команду, подождите, и готово—bling bling завершено.

Программирование

Чтение статей

Читайте и скачивайте статьи напрямую из интернета, например, эту статью 'attention is all you need'. Затем вы можете вести ИИ-разговоры для обучения и даже генерировать понятные стили карточек ключевых вопросов для более интуитивного понимания.

Программирование

Изучение кода

Клонируйте репозитории напрямую для изучения и понимания—пусть Qwen Code расскажет, как вносить вклад в open source проекты, пошаговые операции, полное отслеживание процесса.

Программирование

Решение issues

На основе вышеуказанного плана вы можете начать решать issues для open source проектов—это также можно сделать с Qwen Code, и вы даже можете симулировать загрузку скриншотов терминала.

Программирование

PR Review

После решения issue и автоматической отправки PR вы также можете использовать Qwen Code напрямую для тестирования и ревью PR. Если это ваш проект и кто-то отправил PR, вы также можете поручить ревью Qwen Code. Значительно повысьте эффективность итерации проекта.
