Guide

Установка скриптом в один клик

Быстро установите Qwen Code с помощью скрипта, настройка среды за несколько секунд.

# Linux/macOS curl -fsSL https://qwen-code.oss-cn-beijing.aliyuncs.com/install.sh | bash -s -- --install # Windows (PowerShell) curl -fsSL -o %TEMP%install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%install-qwen.bat
