Qwen Code: Условия использования и политика конфиденциальности

Qwen Code — это инструмент с открытым исходным кодом для помощи в программировании с использованием ИИ, который поддерживается командой Qwen Code. В этом документе изложены условия использования и политики конфиденциальности, применимые при использовании методов аутентификации и сервисов ИИ-моделей Qwen Code.

Как определить ваш метод аутентификации

Qwen Code поддерживает два основных метода аутентификации для доступа к моделям ИИ. Ваш метод аутентификации определяет, какие условия использования и политики конфиденциальности применяются к вашему использованию:

  1. Qwen OAuth — Вход через учетную запись qwen.ai
  2. OpenAI-совместимый API — Использование ключей API от различных поставщиков моделей ИИ

Для каждого метода аутентификации могут применяться различные Условия использования и Положения о конфиденциальности в зависимости от базового поставщика услуг.

Qwen OAuthQwen AIУсловия использования Qwen Политика конфиденциальности Qwen 
OpenAI-совместимый APIРазные поставщикиЗависит от выбранного вами поставщика API (OpenAI, Alibaba Cloud, ModelScope и др.)Зависит от выбранного вами поставщика API

1. Если вы используете аутентификацию через OAuth Qwen

При аутентификации с помощью вашей учетной записи qwen.ai применяются следующие документы: Условия использования и Уведомление о конфиденциальности:

  • Условия использования: Ваше использование регулируется Условиями использования Qwen .
  • Уведомление о конфиденциальности: Сбор и использование ваших данных описаны в Политике конфиденциальности Qwen .

Подробности о настройке аутентификации, квотах и поддерживаемых функциях см. в разделе Настройка аутентификации.

2. Если вы используете аутентификацию через OpenAI-совместимый API

При аутентификации с использованием API-ключей от провайдеров, совместимых с OpenAI, применимые Условия использования и Уведомление о конфиденциальности зависят от выбранного вами провайдера.

Important

При использовании аутентификации через OpenAI-совместимый API вы подчиняетесь условиям и политикам конфиденциальности вашего выбранного API-провайдера, а не условиям Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего провайдера для получения конкретной информации об использовании данных, их хранении и практике обеспечения конфиденциальности.

Qwen Code поддерживает различных провайдеров, совместимых с OpenAI. Пожалуйста, обратитесь к условиям использования и политике конфиденциальности вашего конкретного провайдера для получения подробной информации.

Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные телеметрии для улучшения пользовательского опыта и качества продукта. Сбор этих данных является необязательным и может быть настроен через параметры конфигурации.

Какие данные собираются

При включении Qwen Code может собирать:

  • Анонимную статистику использования (выполняемые команды, метрики производительности)
  • Отчеты об ошибках и данные о сбоях
  • Паттерны использования функций

Сбор данных в зависимости от метода аутентификации

  • Qwen OAuth: Статистика использования регулируется политикой конфиденциальности Qwen. Вы можете отказаться от сбора данных через настройки конфигурации Qwen Code.
  • Совместимый с OpenAI API: Qwen Code не собирает дополнительных данных сверх того, что собирает выбранный вами провайдер API.

Часто задаваемые вопросы (FAQ)

1. Используется ли мой код, включая запросы и ответы, для обучения моделей искусственного интеллекта?

Используется ли ваш код, включая запросы и ответы, для обучения моделей искусственного интеллекта, зависит от метода аутентификации и конкретного поставщика услуг искусственного интеллекта, которого вы используете:

  • Qwen OAuth: Использование данных регулируется Политикой конфиденциальности Qwen . Пожалуйста, ознакомьтесь с их политикой для получения конкретной информации о сборе данных и практике обучения моделей.

  • Совместимый с OpenAI API: Использование данных полностью зависит от выбранного вами поставщика API. У каждого поставщика есть свои собственные политики использования данных. Пожалуйста, ознакомьтесь с политикой конфиденциальности и условиями предоставления услуг вашего конкретного поставщика.

Важно: Сам Qwen Code не использует ваши запросы, код или ответы для обучения моделей. Любое использование данных в обучающих целях будет регулироваться политиками поставщика услуг искусственного интеллекта, через которого вы проходите аутентификацию.

2. Что такое статистика использования и за что отвечает управление отказом от нее?

Настройка Статистика использования управляет сбором дополнительных данных Qwen Code с целью улучшения пользовательского опыта и качества продукта.

Если эта настройка включена, Qwen Code может собирать следующие данные:

  • Анонимную телеметрию (запускаемые команды, метрики производительности, использование функций)
  • Отчеты об ошибках и данные о сбоях
  • Общие шаблоны использования

Какие данные НЕ собираются Qwen Code:

  • Содержимое вашего кода
  • Запросы, отправляемые моделям ИИ
  • Ответы от моделей ИИ
  • Персональные данные

Настройка статистики использования влияет только на сбор данных самим Qwen Code. Она не регулирует то, какие данные могут собираться выбранным вами поставщиком ИИ-сервисов (Qwen, OpenAI и т. д.) в соответствии со своей политикой конфиденциальности.

3. Как переключаться между методами аутентификации?

Вы можете переключаться между аутентификацией через Qwen OAuth и аутентификацией, совместимой с OpenAI API, в любое время:

  1. При запуске: Выберите предпочитаемый метод аутентификации при появлении запроса
  2. Внутри CLI: Используйте команду /auth для изменения метода аутентификации
  3. Переменные окружения: Настройте файлы .env для автоматической аутентификации через API, совместимый с OpenAI

Подробные инструкции см. в документации по настройке аутентификации.
