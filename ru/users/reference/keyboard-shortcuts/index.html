Клавиатурные сокращения Qwen Code

Этот документ содержит список доступных клавиатурных сокращений в Qwen Code.

Общие

Комбинация клавишОписание
EscЗакрыть диалоги и предложения.
Ctrl+CОтменить текущий запрос и очистить ввод. Нажмите дважды для выхода из приложения.
Ctrl+DВыйти из приложения, если ввод пустой. Нажмите дважды для подтверждения.
Ctrl+LОчистить экран.
Ctrl+OПереключить отображение консоли отладки.
Ctrl+SПозволяет выводу длинных ответов отображаться полностью, отключая усечение. Используйте прокрутку терминала для просмотра всего вывода.
Ctrl+TПереключить отображение описаний инструментов.
Shift+TabПереключение режимов одобрения (plan → default → auto-edit → yolo).

Входной запрос

Комбинация клавишОписание
!Переключение режима оболочки при пустом вводе.
?Переключение отображения сочетаний клавиш при пустом вводе.
\ (в конце строки) + EnterВставить новую строку.
Стрелка внизПеремещение по истории ввода вниз.
EnterОтправить текущий запрос.
Meta+Delete / Ctrl+DeleteУдалить слово справа от курсора.
TabАвтозаполнение текущего предложения, если оно существует.
Стрелка вверхПеремещение по истории ввода вверх.
Ctrl+A / HomeПереместить курсор в начало строки.
Ctrl+B / Стрелка влевоПереместить курсор на один символ влево.
Ctrl+CОчистить поле ввода
Esc (двойное нажатие)Очистить поле ввода.
Ctrl+D / DeleteУдалить символ справа от курсора.
Ctrl+E / EndПереместить курсор в конец строки.
Ctrl+F / Стрелка вправоПереместить курсор на один символ вправо.
Ctrl+H / BackspaceУдалить символ слева от курсора.
Ctrl+KУдалить от курсора до конца строки.
Ctrl+Стрелка влево / Meta+Стрелка влево / Meta+BПереместить курсор на одно слово влево.
Ctrl+NПеремещение по истории ввода вниз.
Ctrl+PПеремещение по истории ввода вверх.
Ctrl+RОбратный поиск по истории ввода/оболочки.
Ctrl+Стрелка вправо / Meta+Стрелка вправо / Meta+FПереместить курсор на одно слово вправо.
Ctrl+UУдалить от курсора до начала строки.
Ctrl+VВставить содержимое буфера обмена. Если буфер содержит изображение, оно будет сохранено, и ссылка на него будет вставлена в запрос.
Ctrl+W / Meta+Backspace / Ctrl+BackspaceУдалить слово слева от курсора.
Ctrl+X / Meta+EnterОткрыть текущий ввод во внешнем редакторе.

Предложения

Сочетание клавишОписание
Стрелка внизПеремещение вниз по списку предложений.
Tab / EnterПринять выбранное предложение.
Стрелка вверхПеремещение вверх по списку предложений.

Выбор переключателя

ЯрлыкОписание
Стрелка вниз / jПереместить выбор вниз.
ВводПодтвердить выбор.
Стрелка вверх / kПереместить выбор вверх.
1-9Выбрать элемент по его номеру.
(многозначный)Для элементов с номерами больше 9, нажмите цифры подряд, чтобы выбрать соответствующий элемент.
