✨ Новые Функции

Аналитика Insight

Официальная встроенная команда для глубокого анализа локальных записей Session, предоставляющая персонализированные программные инсайты и отчеты об использовании.

Использование: Введите /insight, чтобы начать параллельный LLM-анализ

Генерирует подробный HTML-отчет, разделенный на 7 разделов:

  • What You Work On - Ваш фокус работы
  • How You Use Qwen Code - Как вы используете Qwen Code
  • Impressive Things - Заметные достижения
  • Where Things Go Wrong - Проблемные области
  • Features to Try - Функции, которые стоит попробовать
  • New Usage Patterns - Новые паттерны использования
  • On the Horizon - Будущие перспективы

Наконец, вы можете экспортировать изображение для публикации в социальных сетях.

👇 Демонстрация Функции

Поддержка Изображений из Буфера Обмена

Кроссплатформенная поддержка: macOS / Windows / Linux. См. PR #1612 

Функции:

  • Захват Изображений: Захват изображений из системного буфера обмена
  • Временное Хранение: Сохранение во временный каталог буфера обмена
  • LRU Очистка: Автоматическое удаление 50 старейших изображений при превышении 100
  • UI Вложений: ↑ для входа в режим вложений, ←/→ для навигации, Delete для удаления

Terminal-Capture

Назначение: Автоматизированный инструмент скриншотов терминала для визуальной верификации CLI, автоматически проверяет функции и создает демонстрационные скриншоты.

Архитектура: node-pty (псевдотерминал) → xterm.js (рендеринг) → Playwright (скриншот), см. PR #1840

Обновления Прогресса Инструментов MCP

Функция: Инструменты MCP поддерживают уведомления о прогрессе в реальном времени

  • Режим TUI: Отображение ⏳ [50/100] Processing...

См. PR #1756
