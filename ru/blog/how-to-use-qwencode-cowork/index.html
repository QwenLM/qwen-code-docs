Я использовал ИИ для наведения порядка на хаотичном рабочем столе, полный список сценариев использования Qwen Code Cowork

Бесплатный ИИ-ассистент с открытым исходным кодом, который поможет вам организовать файлы, массово переименовывать, освобождать место на диске и автоматизировать повседневные задачи.

Ваш рабочий стол тоже выглядит так? 👇

  • Снимок_экрана_20231015, Снимок_экрана_20231016, Снимок_экрана_20231017…
  • Новая_папка, Новая_папка (1), Новая_папка (2)…
  • WeChat_image_20240101, WeChat_image_20240102…

Каждый раз, когда я хочу найти файл, это похоже на поиски сокровищ на свалке — я точно помню, что сохранил его, но найти не могу. Ещё более удручает то, что каждые несколько месяцев я получаю системное уведомление: Недостаточно места на диске.

Я знаю, что нужно навести порядок, но —

Организация файлов вечно висит в списке TODO, но никогда не выполняется.

Пока я не открыл для себя Qwen Code Cowork — ИИ-ассистента, который действительно помогает работать.

Что такое Qwen Code Cowork?

Проще говоря, это бесплатный ИИ-ассистент с открытым исходным кодом для рабочего стола, построенный на основе Qwen Code SDK.

Но это не просто окно чата — это ИИ, который действительно может выполнять работу за вас:

ВозможностьОписание
📝 Написание и редактирование кодаПоддержка любых языков программирования
📁 Управление файламиСоздание, перемещение, переименование, организация
⚡ Выполнение командСборка, тестирование, развёртывание
💬 Ответы на вопросыО вашей кодовой базе или любых других темах
🔧 Автоматизация задачВсё, что вы можете описать на естественном языке

Самое главное: полностью бесплатно, открытый исходный код, данные хранятся локально.

Для сравнения: Claude Code доступен только подписчикам Max по цене $100-200 в месяц, а Qwen Code Cowork обеспечивает аналогичные возможности бесплатно.

Полный список сценариев использования

Ниже приведены несколько сверхполезных сценариев, которые я обнаружил в процессе использования, с демонстрацией конкретных действий.

Сценарий 1: Организация хаотичного рабочего стола / папки загрузок

Рабочий стол и папка загрузок — это настоящие цифровые свалки: различные скриншоты, документы, установочные файлы и временные файлы смешаны вместе. Каждый раз, когда я хочу что-то найти, приходится долго копаться, это раздражает, но ручная организация занимает слишком много времени.

Как Cowork может помочь:

Просто скажите ему: «Помоги мне организовать рабочий стол, классифицируй файлы по типу, помести скриншоты в папку скриншотов, документы в папку документов, установочные файлы в папку установочных файлов»

Он выполнит следующие действия:

  1. Сначала просканирует ваш рабочий стол и составит список всех файлов
  2. Предоставит вам предварительный просмотр плана организации
  3. После вашего подтверждения автоматически выполнит классификацию и перемещение

Полезный совет: Используйте режим Plan, чтобы ИИ сначала спланировал, а затем выполнил после подтверждения плана — это более безопасно и контролируемо.

Сценарий 2: Массовое переименование файлов (фотографии по дате съёмки)

Фотографии, экспортированные с телефона, имеют беспорядочные названия: IMG_0001, IMG_0002… Совершенно непонятно, когда они были сделаны. Я хочу переименовать их в формате «2024-01-15_закат_на_море», но ручное переименование сотен фотографий займёт целую вечность.

Как Cowork может помочь:

Скажите ему: «Переименуй фотографии в этой папке по дате съёмки в формате ГГГГ-ММ-ДД_порядковый номер»

Он выполнит следующие действия:

  1. Прочитает EXIF-информацию каждой фотографии (дату съёмки)
  2. Сгенерирует новые имена файлов в указанном формате
  3. Предварительно покажет все операции переименования
  4. После подтверждения выполнит массовое переименование

Сценарий 3: Проверка локального дискового пространства и поиск больших файлов

Компьютер снова сообщает о недостатке места на диске, но я не знаю, чем оно занято. Я хочу очистить, но боюсь случайно удалить важные файлы, а при использовании различных программ очистки беспокоюсь о безопасности данных.

Как Cowork может помочь:

Скажите ему: «Проверь использование дискового пространства на моём компьютере, найди 10 папок и файлов, занимающих больше всего места, кратко опиши их содержимое, а также можно ли их удалить и какие последствия это может иметь»

Он выполнит следующие действия:

  1. Просканирует использование диска
  2. Составит список файлов и папок, занимающих больше всего места
  3. Проанализирует, что можно безопасно удалить (кеш, временные файлы)
  4. Даст рекомендации по очистке

Внимание: Перед удалением файлов обязательно убедитесь в их ненужности! Рекомендуется сначала переместить их в корзину и окончательно удалить только после подтверждения.

Сценарий 4: Интеграция и форматирование заметок

Заметки разбросаны по разным местам: избранное WeChat, заметки, Notion, Obsidian… Я хочу объединить их в одно место, но форматы не унифицированы, а ручное копирование и вставка с последующим форматированием — слишком хлопотно.

Как Cowork может помочь:

Скажите ему: «Объедини все файлы docx в этой папке в один Markdown-файл, расположи по времени создания, унифицируй формат, отображая имена файлов как заголовки второго уровня»

Он выполнит следующие действия:

  1. Прочитает содержимое всех файлов
  2. Распознает и унифицирует формат
  3. Отсортирует и объединит по указанным правилам
  4. Сгенерирует Markdown-файл с правильным форматированием

Сценарий 5: Массовая обработка документов (извлечение содержимого PDF)

Я скачал кучу PDF-материалов и хочу извлечь из них текст для заметок, но открывать и копировать по одному слишком медленно. Некоторые PDF являются отсканированными, и при копировании получается сплошной бред.

Как Cowork может помочь:

Скажите ему: «Извлеки содержимое всех PDF-файлов в этой папке и сохрани в соответствующие markdown-файлы»

Он выполнит следующие действия:

  1. Переберёт все PDF в папке
  2. Извлечёт текстовое содержимое
  3. Для каждого PDF создаст соответствующий txt-файл
  4. Сообщит о результатах обработки

Прямая установка Qwen Code (рекомендуется)

Просто скажите Qwen Code, и всё будет готово за 3 минуты.

Установка из исходного кода (продвинутый)

Если вы хотите собрать из исходного кода, сначала нужно установить Bun:

# Установка Bun
curl -fsSL https://bun.sh/install | bash
exec /bin/zsh
 
# Клонирование репозитория
git clone https://github.com/QwenLM/qwen-code-examples
 
# Переход в директорию и установка
cd qwen-code-examples/apps/qwen-cowork
bun install
 
# Запуск
bun run dev

Полезные советы по использованию

1. Используйте режим Plan

Для необратимых операций с файлами рекомендуется сначала попросить ИИ спланировать решение:

«Сначала проанализируй ситуацию с этой папкой и дай мне план организации, не выполняй сразу»

После подтверждения плана можно попросить его выполнить.

2. Указывайте рабочую директорию

При создании сессии можно указать рабочую директорию, чтобы ИИ работал только в этой директории — это безопаснее.

3. Требуйте подтверждения для чувствительных операций

Cowork запрашивает подтверждение для чувствительных операций, таких как удаление и перемещение, поэтому случайные операции невозможны.

Другие возможности

Помимо перечисленных выше сценариев, Qwen Code Cowork также может помочь вам:

  • 🔄 Автоматизация повседневных задач (резервное копирование по расписанию, синхронизация файлов)
  • 💻 Управление проектами кода (инициализация проекта, установка зависимостей, запуск тестов)
  • 📊 Обработка данных (конвертация CSV, очистка данных, преобразование форматов)
  • 📝 Генерация документации (генерация документации по коду, создание README)
  • 🔍 Рецензирование кода (анализ качества кода, поиск потенциальных проблем)

Всё, что вы можете описать на естественном языке, он может реализовать за вас.
