Qwen Code: AI-ассистент по программированию, который думает как программист

Qwen Team
30 января 2026 г.

Если AI изменит мир, первым шагом станет трансформация цифрового мира. Qwen Code — это не просто инструмент командной строки; это компаньон-программист, которого вы можете вызвать из терминала в любое время.

За последний год парадигма программирования с AI кардинально изменилась. От раннего режима Chat до режима автодополнения Copilot, а теперь до автономного режима Agent. Сегодня мы с гордостью представляем Qwen Code — инструмент для программирования с AI, разработанный лабораторией Alibaba Tongyi.

Он имеет открытый исходный код, бесплатный и работает на мощной модели Qwen3-Coder. Он не только может писать код через разговорные взаимодействия в терминале, чтобы мгновенно воплощать ваши идеи в жизнь, но и глубоко интегрируется в ваш рабочий процесс разработки для исправления ошибок, рефакторинга кода и легкого выполнения сложных задач. Более того, это универсальный помощник, который может писать статьи, рисовать диаграммы и легко вписываться в вашу повседневную работу.

Что такое Qwen Code?

Мы создали сайт для представления продукта Qwen Code, просто разговаривая. На протяжении всего процесса Qwen Code думает и действует как настоящий программист:

  • Разбиение задач: Создает комплексный список задач.
  • Написание кода: Автоматически читает/пишет файлы и выполняет скрипты.
  • Самокоррекция: Автоматически отлаживает при возникновении ошибок.
  • Доставка результатов: Доставляет не только код, но и целые веб-приложения или документацию.

Вам больше не нужно шаг за шагом направлять его инструкциями типа «напиши эту функцию» или «измени эту переменную». Это больше не инструмент, требующий обширных руководств; это настоящий партнер по программированию, автономный и эффективный.

Почему стоит выбрать Qwen Code?

1. Мощное ядро: на базе Qwen3-Coder

Qwen Code построен на мощной модели Qwen3-Coder и глубоко оптимизирован для нее. На этапе обучения мы научили модель понимать инструменты, а фреймворк был адаптирован под модель. Он демонстрирует исключительные возможности в генерации кода, рассуждении и понимании длинных контекстов.

2. Настоящий Agentic Workflow

Qwen Code имеет такие расширенные возможности, как Skills, SubAgents и Plan Mode, делая Agent более автономным и надежным. Режим утверждения обеспечивает безопасность работы, позволяя пошаговое подтверждение, или вы можете включить режим YOLO, чтобы предоставить AI полную автономию.

3. Полное покрытие сценариев: везде, где это нужно

Хотя Qwen Code родился в терминале, он не ограничен им. Он достаточно мал, но при этом достаточно мощен, чтобы существовать в пяти различных средах:

  • Среда терминала: Выбор для гиков. Запускайте его в терминале для программирования и творчества в элегантном интерактивном интерфейсе.
  • Среда IDE: Предпочтительный выбор для большинства разработчиков. Поддерживает популярные редакторы, такие как VS Code, Qoder, Zed и JetBrains, предлагая как удобство GUI, так и мощь CodeAgent.
  • Среда CI/CD: Используйте qwen -p для запуска Headless Mode. Вы можете упомянуть @QwenCode в GitHub Actions, чтобы он автоматически отвечал на issues, проводил ревью кода или отправлял PRs.
  • Среда браузера: Экспериментальная функция, интегрирующая Qwen Code в расширение браузера, позволяющая ему воспринимать состояние веб-страниц и анализировать параметры — отличный помощник для отладки и сбора данных.
  • Среда программирования: Используйте SDK Qwen Code, чтобы привнести возможности интеллектуального программирования в вашу собственную среду, в настоящее время поддерживается Node.js и Java.

4. Приверженность открытому исходному коду и бесплатному доступу

Мы привержены стратегии открытого исходного кода и бесплатного доступа. В эпоху быстрой итерации программирования с AI мы верим, что открытый исходный код обеспечивает спокойствие, а бесплатный доступ побуждает пользователей исследовать больше возможностей.

  • Бесплатное использование: Войдите через Qwen OAuth, чтобы получить дневную квоту в 2000 бесплатных запросов.
  • Совместимость протоколов: Как проект с открытым исходным кодом, мы совместимы с основными протоколами, такими как Anthropic, Google Gemini и OpenAI.

Начните прямо сейчас

Qwen Code теперь полностью имеет открытый исходный код на GitHub с 17,4k звездами. Начните работу в три простых шага:

  1. Установка (требуется Node.js 20+): 
    npm install -g @qwen-code/qwen-code@latest
  2. Запуск: 
    qwen
  3. Аутентификация: Введите /auth в сессии и выберите Qwen OAuth, чтобы начать свое путешествие в мир программирования с AI.
