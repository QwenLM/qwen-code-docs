жҸ’д»¶иҒҢиҙЈпјҡsenderId еҝ…йЎ»зЁіе®ҡдё”е”ҜдёҖпјӣchatId йңҖеҢәеҲҶз§ҒиҒҠдёҺзҫӨиҒҠпјӣеёғе°”ж Үеҝ—еҝ…йЎ»еҮҶзЎ®д»Ҙж”ҜжҢҒ Gate йҖ»иҫ‘пјӣtext дёӯйңҖеүҘзҰ» @жҸҗеҸҠ еҶ…е®№гҖӮ

ж¶ҲжҒҜжөҒиҪ¬

Inbound:  User message вҶ’ Adapter вҶ’ GroupGate вҶ’ SenderGate вҶ’ Slash commands вҶ’ SessionRouter вҶ’ AcpBridge вҶ’ Agent
Outbound: Agent response вҶ’ AcpBridge вҶ’ SessionRouter вҶ’ Adapter вҶ’ User

ж–ңжқ е‘Ҫд»ӨпјҲ/clearгҖҒ/helpгҖҒ/statusпјүдјҡеңЁеҲ°иҫҫ Agent д№ӢеүҚз”ұ ChannelBase еӨ„зҗҶгҖӮ

дјҡиҜқз®ЎзҗҶ

еҚ•дёӘ qwen-code --acp иҝӣзЁӢжүҝиҪҪеӨҡдёӘ ACP дјҡиҜқгҖӮжҜҸдёӘ Channel зҡ„дҪңз”ЁеҹҹпјҡuserпјҲй»ҳи®ӨпјүгҖҒthread жҲ– singleгҖӮи·Ҝз”ұй”®йҮҮз”Ё <channelName>:<key> е‘ҪеҗҚз©әй—ҙж јејҸгҖӮ

й”ҷиҜҜеӨ„зҗҶ

  • иҝһжҺҘеӨұиҙҘ вҖ”вҖ” и®°еҪ•ж—Ҙеҝ—пјӣеҸӘиҰҒиҮіе°‘жңүдёҖдёӘ Channel жҲҗеҠҹиҝһжҺҘпјҢжңҚеҠЎе°Ҷз»§з»ӯиҝҗиЎҢ
  • Bridge еҙ©жәғ вҖ”вҖ” жҢҮж•°йҖҖйҒҝпјҲжңҖеӨҡйҮҚиҜ• 3 ж¬ЎпјүпјҢеңЁжүҖжңү Channel дёҠи°ғз”Ё setBridge()пјҢжҒўеӨҚдјҡиҜқ
  • дјҡиҜқеәҸеҲ—еҢ– вҖ”вҖ” еҹәдәҺжҜҸдёӘдјҡиҜқзҡ„ Promise й“ҫпјҢйҳІжӯўе№¶еҸ‘ Prompt еҶІзӘҒ

жҸ’д»¶зі»з»ҹ

иҜҘжһ¶жһ„е…·еӨҮй«ҳжү©еұ•жҖ§ вҖ”вҖ” ж— йңҖдҝ®ж”№ж ёеҝғд»Јз ҒеҚіеҸҜжҺҘе…Ҙж–°йҖӮй…ҚеҷЁпјҲеҗ«з¬¬дёүж–№пјүгҖӮеҶ…зҪ® Channel еҗҢж ·еӨҚз”ЁжӯӨжҸ’д»¶жҺҘеҸЈпјҲеҶ…йғЁиҮӘз ”иҮӘз”ЁпјүгҖӮ

жҸ’д»¶еҘ‘зәҰ

дёҖдёӘ ChannelPlugin йңҖеЈ°жҳҺ channelTypeгҖҒdisplayNameгҖҒrequiredConfigFields д»ҘеҸҠ createChannel() е·ҘеҺӮж–№жі•гҖӮжҸ’д»¶йңҖе®һзҺ°д»ҘдёӢдёүдёӘж–№жі•пјҡ

ж–№жі•иҒҢиҙЈ
connect()иҝһжҺҘе№іеҸ°е№¶жіЁеҶҢж¶ҲжҒҜеӨ„зҗҶеҷЁ
sendMessage(chatId, text)ж јејҸеҢ–е№¶еҸ‘йҖҒ Agent е“Қеә”
disconnect()е…ій—ӯж—¶жү§иЎҢжё…зҗҶе·ҘдҪң

еӨ„зҗҶе…Ҙз«ҷж¶ҲжҒҜж—¶пјҢжҸ’д»¶йңҖжһ„е»ә Envelope е№¶и°ғз”Ё this.handleInbound(envelope) вҖ”вҖ” еҹәзұ»е°ҶеӨ„зҗҶеү©дҪҷйҖ»иҫ‘пјҡи®ҝй—®жҺ§еҲ¶гҖҒзҫӨиҒҠжӢҰжҲӘгҖҒй…ҚеҜ№гҖҒдјҡиҜқи·Ҝз”ұгҖҒPrompt еәҸеҲ—еҢ–гҖҒж–ңжқ е‘Ҫд»ӨгҖҒжҢҮд»ӨжіЁе…ҘгҖҒеӣһеӨҚдёҠдёӢж–Үд»ҘеҸҠеҙ©жәғжҒўеӨҚгҖӮ

жү©еұ•зӮ№

  • йҖҡиҝҮ registerCommand() жіЁеҶҢиҮӘе®ҡд№үж–ңжқ е‘Ҫд»Ө
  • йҖҡиҝҮеҢ…иЈ… handleInbound() е®һзҺ°иҫ“е…Ҙдёӯ/еҸҚеә”зҠ¶жҖҒжҢҮзӨәеҷЁ
  • йҖҡиҝҮ onToolCall() е®һзҺ°е·Ҙе…·и°ғз”Ёй’©еӯҗ
  • еңЁи°ғз”Ё handleInbound() еүҚе°ҶеӘ’дҪ“ж•°жҚ®йҷ„еҠ еҲ° Envelope д»ҘеӨ„зҗҶеӘ’дҪ“ж–Үд»¶
