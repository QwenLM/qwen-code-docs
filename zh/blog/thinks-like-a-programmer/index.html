Qwen Code: 像程序员一样思考的 AI 编程助手

Qwen Team
2026年1月30日

如果 AI 能够改变世界，那么第一步，它首先改变的是数字世界。Qwen Code 不仅仅是一个命令行工具，更是一个你随时可以从命令行中唤出的程序员伙伴。

在过去的一年里，AI 编程范式发生了巨大的转变。从最早的 Chat 模式，到 Copilot 的补全模式，再到如今的 Agent 自主模式。今天我们来介绍阿里通义实验室推出的 AI 编程工具：Qwen Code。

它是开源的、免费的，由强大的 Qwen3-Coder 模型驱动。不仅能在终端像聊天一样写代码，让灵感即刻落地；也能深度集成开发流，修复 BUG，重构代码，从容应对。它更是一个全能伙伴，写文章、画图表，丝滑融入你的工作流。

什么是 Qwen Code？

我们通过聊天对话，就完成了 Qwen Code 的产品介绍站点。整个过程 Qwen Code像程序员一样思考和行动：

  • 拆解任务：列出 To-do List。
  • 编写代码：自动读写文件、执行脚本。
  • 自我修正：遇到报错自动 Debug。
  • 交付结果：不仅是代码，甚至是完整的 Web 应用或文档。

你不再需要一步步指导它“写这个函数”、“改那个变量”。它不再是一个需要很多操作手册的工具，而是一个真正的编码伙伴，自主且高效。

为什么选择 Qwen Code？

1. 强劲内核：Qwen3-Coder 驱动

Qwen Code 内置强大的 Qwen3-Coder 模型，并为其深度优化。我们在训练阶段就让模型认识了工具，框架适配了模型。在代码生成、推理和长上下文理解上展现出不俗的能力。

2. 真正的 Agentic Workflow（代理工作流）

Qwen Code 内置了 Skills（技能）、SubAgents（子智能体）、Plan Mode（计划模式） 等高级功能，让 Agent 更加自主可靠。审批模式也让工作更加安全，既可以安全地一步步确认操作，也可以开启 YOLO 模式，让 AI 自主放手去干。

3. 全场景覆盖：无处不在

虽然 Qwen Code 生于终端，但它并不受限于终端。它足够 small，也足够 strong，可以存在于以下五种环境中：

  • 终端环境：极客首选，一键启动终端（Terminal），在简洁的交互式界面中完成编码与创意。
  • IDE 环境：绝大多数程序员的首选，支持 VS Code、Qoder、Zed、JetBrains 等主流编辑器，既能享受 IDE 带来的 GUI 交互优势，又能享受 Qwen Code 带来的 CodeAgent 便利。
  • CI/CD 环境：使用 qwen -p 启动 Headless Mode（无头模式），你可以在 GitHub Actions 中 @QwenCode，让它帮你自动回复 ISSUE、Review 代码或提交 PR。
  • 浏览器环境：正在尝试的实验性质功能，可将 Qwen Code 集成在浏览器插件中，感知网页状态，分析接口参数，是页面调试和数据采集的好帮手。
  • 编程环境：提供 Qwen Code SDK，将智能编码能力带入编程环境中，目前支持了 Node.js 和 Java 语言的集成。

4. 开源与免费的诚意

我们坚持开源和免费的策略。在当前 AI Coding 快速迭代的时刻，我们相信开源会让用户用得更放心，免费会让用户敞开了用，探索更多的可能性。

  • 免费调用：通过 Qwen OAuth 登录，即可享受每日 2000 次的免费请求额度。
  • 协议兼容：作为开源项目，我们兼容了 Anthropic、Google Gemini、OpenAI 等主流协议。

立即开始

Qwen Code 现已在 GitHub 全面开源，拥有 17.4k Star。仅需三步，快速开始：

  1. 安装（需要 Node.js 20+）： 
    npm install -g @qwen-code/qwen-code@latest
  2. 启动： 
    qwen
  3. 认证： 在会话中输入 /auth，选择 Qwen OAuth 即可开启你的 AI 编程之旅。
