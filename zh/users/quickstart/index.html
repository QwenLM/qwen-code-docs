用户指南快速开始

快速入门

👏 欢迎使用 Qwen Code！

本快速入门指南将帮助你在几分钟内开始使用 AI 驱动的编码辅助功能。完成后，你将了解如何使用 Qwen Code 来完成常见的开发任务。

开始之前

请确保你已准备好：

  • 打开一个 终端 或命令行提示符
  • 准备好一个代码项目用于操作
  • 注册一个 Qwen Code  账户

步骤 1：安装 Qwen Code

你可以通过以下任一方式安装 Qwen Code：

NPM（推荐）

需要 Node.js 20+ ，可使用 node -v 检查版本。如果尚未安装，请使用以下命令进行安装。

如果你已经安装了 Node.js 或更高版本 ：

npm install -g @qwen-code/qwen-code@latest

Homebrew（macOS、Linux）

brew install qwen-code

步骤 2：登录你的账户

Qwen Code 需要一个账户才能使用。当你通过 qwen 命令启动交互式会话时，你需要先登录：

# 首次使用时系统将提示你登录
qwen
# 按照提示使用你的账户登录
/auth

选择 Qwen OAuth，登录你的账户并按照提示确认。登录后，你的凭证会被保存，无需再次登录。

Note

当你首次使用 Qwen 账户对 Qwen Code 进行身份验证时，系统会自动为你创建一个名为“.qwen”的工作区。该工作区为组织内所有 Qwen Code 使用情况提供集中的成本跟踪和管理功能。

Tip

如果你需要重新登录或切换账户，请在 Qwen Code 内部使用 /auth 命令。

步骤 3：开始你的第一个会话

在任意项目目录中打开终端并启动 Qwen Code：

# 可选
cd /path/to/your/project
 
# 启动通义千问
qwen

你会看到 Qwen Code 的欢迎界面，其中包含会话信息、最近对话和最新更新。输入 /help 可查看可用命令。

与 Qwen Code 聊天

提出你的第一个问题

Qwen Code 将分析你的文件并提供摘要。你也可以提出更具体的问题：

解释文件夹结构

你还可以询问 Qwen Code 自身的能力：

Qwen Code 能做什么？
Note

Qwen Code 会根据需要读取你的文件——你无需手动添加上下文。Qwen Code 还可以访问其自身的文档，并能回答有关其功能和能力的问题。

进行你的第一次代码修改

现在让我们让 Qwen Code 进行一些实际的编码工作。尝试一个简单的任务：

在主文件中添加一个 hello world 函数

Qwen Code 将会：

  1. 找到合适的文件
  2. 向你展示建议的更改
  3. 请求你的批准
  4. 进行编辑
Note

Qwen Code 在修改文件前总是会请求权限。你可以逐个批准更改，或为整个会话启用“全部接受”模式。

使用 Git 与 Qwen Code 协作

Qwen Code 使 Git 操作变得像对话一样简单：

我修改了哪些文件？
用一条描述性信息提交我的更改

你也可以提示进行更复杂的 Git 操作：

创建一个名为 feature/quickstart 的新分支
显示最近的 5 次提交
帮助我解决合并冲突

修复 bug 或添加功能

Qwen Code 擅长调试和功能实现。

用自然语言描述你的需求：

为用户注册表单添加输入验证

或修复现有问题：

存在一个用户可以提交空表单的 bug — 请修复它

Qwen Code 将会：

  • 定位相关代码
  • 理解上下文
  • 实现解决方案
  • 如有测试，将运行测试

重构代码

为计算器函数编写单元测试

Tip

记住：Qwen Code 是你的 AI 结对程序员。像与乐于助人的同事交谈一样与它交流——描述你想要实现的目标，它会帮助你达成目标。

基本命令

以下是日常使用中最重要的命令：

命令功能说明示例
qwen启动 Qwen Codeqwen
/auth更改认证方式/auth
/help显示可用命令的帮助信息/help 或 /?
/compress用摘要替换聊天历史以节省 Token/compress
/clear清除终端屏幕内容/clear（快捷键：Ctrl+L）
/theme更改 Qwen Code 的视觉主题/theme
/language查看或更改语言设置/language
→ ui [language]设置用户界面语言/language ui zh-CN
→ output [language]设置大语言模型输出语言/language output Chinese
/quit立即退出 Qwen Code/quit 或 /exit

有关完整命令列表，请参阅 CLI 参考。

初学者专业技巧

请求要具体

  • ❌ 不要这样问：“修复这个 bug”
  • ✅ 而是这样问：“修复登录 bug，用户输入错误凭证后看到空白屏幕的问题”

使用逐步指导

  • 将复杂任务分解为步骤：
1. 为用户资料创建新的数据库表
2. 创建用于获取和更新用户资料的 API 端点
3. 构建允许用户查看和编辑其信息的网页

让 Qwen Code 先探索

  • 在进行更改之前，让 Qwen Code 了解你的代码：
分析数据库模式
构建一个仪表板，显示我们英国客户最常退货的产品

使用快捷键节省时间

  • 按 ? 查看所有可用的键盘快捷键
  • 使用 Tab 进行命令补全
  • 按 ↑ 查看命令历史
  • 输入 / 查看所有斜杠命令

获取帮助

  • 在 Qwen Code 中：输入 /help 或询问 “如何…”

  • 社区：加入我们的 GitHub 讨论区  获取技巧和支持
