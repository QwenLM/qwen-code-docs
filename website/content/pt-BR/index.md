# Documentação do Qwen Code

Bem-vindo à documentação do Qwen Code. O Qwen Code é uma ferramenta de programação baseada em agentes que reside no seu terminal e ajuda você a transformar ideias em código mais rapidamente do que nunca.

Aprenda a usar o Qwen Code como usuário final. Esta seção aborda:

- Padrões comuns de uso
- Recursos e funcionalidades
- Opções de configuração
- Solução de problemas
