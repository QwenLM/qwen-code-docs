# Documentação do Qwen Code

Bem-vindo à documentação do Qwen Code. O Qwen Code é uma ferramenta de codificação baseada em agentes que funciona no seu terminal e ajuda você a transformar ideias em código mais rápido do que nunca.

## Seções da Documentação

Aprenda como usar o Qwen Code como usuário final. Esta seção aborda:

- Padrões comuns de uso
- Recursos e funcionalidades
- Opções de configuração
- Solução de problemas
