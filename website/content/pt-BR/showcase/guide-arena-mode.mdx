## Visão Geral

Para código importante, você pediria a um colega para revisar. Agora você pode fazer múltiplos modelos de IA processarem a mesma tarefa simultaneamente. O Agent Arena permite comparar as capacidades de diferentes modelos — basta chamar `/arena`, selecionar modelos, inserir sua tarefa e escolher o melhor resultado.

## O que você pode fazer com isso

- **Refatorar módulos principais**: Fazer múltiplos modelos proporem soluções simultaneamente e escolher a melhor
- **Comparar abordagens**: Comparar soluções de diferentes modelos ao escrever algoritmos, aprender diferentes perspectivas
- **Avaliar novos modelos**: Usar tarefas reais para avaliar o desempenho de novos modelos no seu projeto

## Passos

<Steps>

### Inserir o comando /arena

No chat do Qwen Code, digite `/arena` e selecione `start` para entrar no modo arena.

```bash
/arena start
```

### Selecionar os modelos participantes

Na lista de modelos disponíveis, selecione os modelos que deseja comparar. Pressione `space` para selecionar 2 ou mais modelos.

### Inserir a descrição da tarefa

Insira a tarefa que deseja que cada modelo processe, por exemplo:

```
Refatore esta função para melhorar a legibilidade e o desempenho
```

### Comparar resultados e escolher o melhor

Todos os modelos processarão a tarefa simultaneamente. Após a conclusão, você pode comparar suas soluções e aplicar o melhor resultado ao seu código.

```bash
/arena select
```

</Steps>

<Callout type="info">
O modo arena é especialmente adequado para decisões importantes de código, permitindo que múltiplos modelos de IA forneçam diferentes perspectivas de soluções.
</Callout>
