## O que é o Qwen Code Cowork?

Em resumo, é um assistente de IA de desktop **gratuito e open source**, construído com base no SDK do Qwen Code.

Mas não é apenas uma caixa de chat, é uma IA que realmente **trabalha para você**:

| Capacidade | Descrição |
|------------|-----------|
| 📝 Escrever e editar código | Suporta qualquer linguagem de programação |
| 📁 Gerenciar arquivos | Criar, mover, renomear, organizar |
| ⚡ Executar comandos | Build, teste, deploy |
| 💬 Responder perguntas | Sobre sua base de código ou qualquer assunto |
| 🔧 Automatizar tarefas | Desde que você possa descrever em linguagem natural |

O mais importante: **totalmente gratuito, open source e controlável, dados armazenados localmente**.

Em comparação, o Claude Code está disponível apenas para assinantes Max, com mensalidade de 100-200 dólares, enquanto o Qwen Code Cowork oferece capacidades equivalentes sem custo algum.

## Guia completo de casos de uso

A seguir estão alguns cenários super úteis que descobri no uso real, cada um com demonstrações operacionais específicas.

### Cenário 1: Organizar a área de trabalho bagunçada / pasta de downloads

A área de trabalho e a pasta de downloads são basicamente aterros digitais, misturando capturas de tela, documentos, pacotes de instalação e arquivos temporários. Toda vez que você quer encontrar algo, leva uma eternidade, e parece estressante, mas organizar manualmente leva muito tempo.

**Como o Cowork pode ajudar**:

Diga a ele diretamente: "Me ajude a organizar a área de trabalho, classifique por tipo de arquivo, coloque capturas de tela na pasta de capturas de tela, documentos na pasta de documentos, pacotes de instalação na pasta de pacotes de instalação"

Ele vai:

1. Primeiro escanear sua área de trabalho e listar todos os arquivos
2. Dar uma prévia do plano de organização
3. Após sua confirmação, executar automaticamente a classificação e movimentação

 
<Callout>
    **Dica**:
    Use o modo Plan para que a IA planeje primeiro, confirme o plano e depois execute, assim é mais seguro e controlável.
</Callout>

### Cenário 2: Renomear arquivos em lote (fotos renomeadas por data)

As fotos exportadas do celular têm nomes confusos, IMG_0001, IMG_0002... completamente impossível de saber quando foram tiradas. Você quer renomear no formato "2024-01-15_Pôr do sol na praia", mas renomear manualmente centenas de fotos levaria uma eternidade.

**Como o Cowork pode ajudar**:

Diga a ele: "Renomeie as fotos desta pasta por data de captura, no formato AAAA-MM-DD_Número"

Ele vai:

1. Ler as informações EXIF de cada foto (data de captura)
2. Gerar novos nomes de arquivo no formato especificado por você
3. Pré-visualizar todas as operações de renomeação
4. Executar em lote após confirmação

### Cenário 3: Verificar espaço de armazenamento local e encontrar arquivos grandes

O computador está avisando novamente sobre espaço em disco insuficiente, mas você não sabe o que está ocupando o espaço. Quer limpar, mas teme excluir arquivos importantes por engano, e ao abrir vários softwares de limpeza, preocupa-se com a segurança da privacidade.

**Como o Cowork pode ajudar**:

Diga a ele: "Me ajude a verificar o uso do espaço de armazenamento do computador, encontre as 10 pastas e arquivos que ocupam mais espaço, explique brevemente o conteúdo, e se podem ser excluídos e o impacto potencial da exclusão"

Ele vai:

1. Escanear o uso do disco
2. Listar os arquivos e pastas que ocupam mais espaço
3. Analisar quais podem ser excluídos com segurança (como cache, arquivos temporários)
4. Dar recomendações de limpeza

<Callout type="error">
    **Atenção**:
    Confirme sempre antes de excluir arquivos! Recomenda-se mover primeiro para a lixeira e, após confirmar que está tudo bem, excluir permanentemente.
</Callout>

### Cenário 4: Integração e formatação de notas

As notas estão espalhadas por vários lugares: favoritos do WeChat, notas, Notion, Obsidian... Você quer integrá-las, mas os formatos não são uniformes, e copiar e colar manualmente ainda requer ajuste de formatação, o que é muito trabalhoso.

**Como o Cowork pode ajudar**:

Diga a ele: "Integre todos os arquivos docx desta pasta em um único arquivo Markdown, organize por data de criação, unifique o formato, exiba o nome do arquivo como subtítulo de nível 2"

Ele vai:

1. Ler o conteúdo de todos os arquivos
2. Identificar e unificar o formato
3. Integrar e ordenar de acordo com as regras especificadas por você
4. Gerar um arquivo Markdown com formatação padronizada

### Cenário 5: Processamento em lote de documentos (extração de conteúdo PDF)

Você baixou vários materiais em PDF e quer extrair o texto para fazer anotações, mas abrir e copiar um por um é muito demorado. Alguns PDFs são versões digitalizadas, e ao copiar, saem todos embaralhados.

**Como o Cowork pode ajudar**:

Diga a ele: "Extraia o conteúdo de todos os arquivos PDF desta pasta e salve como arquivos markdown correspondentes"

Ele vai:

1. Percorrer todos os PDFs na pasta
2. Extrair o conteúdo de texto
3. Gerar arquivos txt correspondentes para cada PDF
4. Relatar os resultados do processamento

### Instalação a partir do código fonte (avançado)

Se você deseja construir a partir do código fonte, precisa instalar o Bun primeiro:

```bash
# Instalar Bun
curl -fsSL https://bun.sh/install | bash
exec /bin/zsh

# Clonar o repositório
git clone https://github.com/QwenLM/qwen-code-examples

# Entrar no diretório e instalar
cd qwen-code-examples/apps/qwen-cowork
bun install

# Iniciar
bun run dev
```

## Dicas de uso

### 1. Use bem o modo Plan

Para tarefas de manipulação de arquivos que são irreversíveis, recomenda-se primeiro pedir à IA para planejar a solução:

```
"Primeiro me ajude a analisar a situação desta pasta e me dê um plano de organização, não execute diretamente"
```

Após confirmar que o plano está correto, então peça para ele executar.

### 2. Especifique o diretório de trabalho

Ao criar uma sessão, você pode especificar o diretório de trabalho, assim a IA só operará neste diretório, o que é mais seguro.

### 3. Operações sensíveis requerem confirmação

O Cowork solicitará sua confirmação para operações sensíveis como exclusão e movimentação, então não se preocupe com operações acidentais.
