# Visual Studio Code

> Die VS Code-Erweiterung (Beta) ermöglicht es Ihnen, Änderungen von Qwen in Echtzeit über eine native grafische Oberfläche zu sehen, die direkt in Ihre IDE integriert ist, wodurch der Zugriff auf und die Interaktion mit Qwen Code einfacher wird.

### Funktionen

- **Native IDE-Erfahrung**: Dediziertes Qwen Code-Seitenpanel, auf das über das Qwen-Symbol zugegriffen werden kann
- **Automatischer Annahme-Modus für Bearbeitungen**: Automatische Anwendung von Qwens Änderungen, sobald sie vorgenommen wurden
- **Dateiverwaltung**: Dateien per @-Erwähnung erwähnen oder Dateien und Bilder mit dem System-Dateiauswahltool anhängen
- **Konversationsverlauf**: Zugriff auf vergangene Konversationen
- **Mehrere Sitzungen**: Gleichzeitiges Ausführen mehrerer Qwen Code-Sitzungen

### Voraussetzungen

- VS Code 1.85.0 oder höher

### Installation

Laden Sie die Erweiterung aus dem [Visual Studio Code Extension Marketplace](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion) herunter und installieren Sie sie.

## Problembehandlung

### Erweiterung wird nicht installiert

- Stellen Sie sicher, dass Sie über VS Code 1.85.0 oder höher verfügen
- Überprüfen Sie, ob VS Code die Berechtigung zum Installieren von Erweiterungen hat
- Versuchen Sie eine direkte Installation über die Marketplace-Website

### Qwen Code reagiert nicht

- Überprüfen Sie Ihre Internetverbindung
- Starten Sie eine neue Konversation, um zu sehen, ob das Problem weiterhin besteht
- [Melden Sie ein Problem auf GitHub](https://github.com/qwenlm/qwen-code/issues), wenn das Problem weiterhin besteht
