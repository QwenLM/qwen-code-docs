Diese Woche wurden die Hauptversions v0.10.0, v0.11.0 und 5 Bugfix-Versionen veröffentlicht.

- Veröffentlichung von zwei Hauptversions: v0.10.0 und v0.11.0, sowie 5 Bugfix-Versionen
- OAuth-Authentifizierung mit Upgrade auf qwen3.5-plus, 1000 kostenlose Aufrufe pro Tag
- Neue Funktion: Insight-Analytik
- Zwischenablage-Bildunterstützung
- Terminal Capture automatisierte Validierung und Screenshot-Generierung

## ✨ Neue Funktionen

### Insight-Analytik

Offizieller integrierter Befehl zur tiefen Analyse lokaler Session-Datensätze für personalisierte Programmier-Einblicke und Nutzungsanalysen.

![](https://gw.alicdn.com/imgextra/i1/O1CN01fU1ONc1cQ6aODsN3J_!!6000000003594-2-tps-1698-954.png)

Verwendung: `/insight` eingeben, um parallele LLM-Analyse zu starten

Generiert einen detaillierten HTML-Bericht in 7 Abschnitten:
- **What You Work On** - Ihre Arbeitsschwerpunkte
- **How You Use Qwen Code** - Wie Sie Qwen Code nutzen
- **Impressive Things** - Bemerkenswerte Erfolge
- **Where Things Go Wrong** - Problembereiche
- **Features to Try** - Ausprobierenswerte Funktionen
- **New Usage Patterns** - Neue Nutzungsmuster
- **On the Horizon** - Zukunftsausblick

Abschließend kann ein Bild für Social-Media-Sharing exportiert werden.

![](https://gw.alicdn.com/imgextra/i4/O1CN01xJKhpq1WNSbVJyveF_!!6000000002776-2-tps-2400-1332.png)

### Zwischenablage-Bildunterstützung

Plattformübergreifende Unterstützung: macOS / Windows / Linux. Siehe PR [#1612](https://github.com/QwenLM/qwen-code/pull/1612)

Funktionen:
- **Bilderfassung**: Bilder aus der Systemzwischenablage erfassen
- **Temporäre Speicherung**: Im Zwischenablage-Temp-Verzeichnis speichern
- **LRU-Bereinigung**: Automatisch die ältesten 50 Bilder löschen bei Überschreitung von 100
- **Anhang-UI**: `↑` für Anhang-Modus, `←/→` zum Navigieren, `Delete` zum Entfernen

![](https://gw.alicdn.com/imgextra/i2/O1CN01OsEDov1z4nJto1CfQ_!!6000000006661-2-tps-1694-956.png)

### Terminal-Capture

Zweck: Automatisiertes Terminal-Screenshot-Tool für CLI-Visuelle-Verifizierung, validiert automatisch Funktionen und erzeugt Demo-Screenshots.

Architektur: node-pty (Pseudo-Terminal) → xterm.js (Rendering) → Playwright (Screenshot), siehe PR [#1840](https://github.com/QwenLM/qwen-code/pull/1840)

### MCP-Tool Fortschrittsupdates

Funktion: MCP-Tools unterstützen Echtzeit-Fortschrittsbenachrichtigungen

- **TUI-Modus**: Anzeige von `⏳ [50/100] Processing...`
- **SDK-Modus**: Neuer `tool_progress` Stream-Ereignistyp

Siehe PR [#1756](https://github.com/QwenLM/qwen-code/pull/1756)
