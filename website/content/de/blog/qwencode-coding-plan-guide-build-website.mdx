## Persönliche Portfolio-Website in 5 Minuten

Benötigen Sie eine gute Lebenslaufseite, um Ihr Portfolio bei der Jobsuche zu präsentieren? Anstatt endlos Templates anzupassen, lassen Sie KI Ihnen helfen, eines zu generieren.

Ich nutzte einfach den `@`-Befehl in Qwen Code, um auf meinen Lebenslauf zu verweisen, und ließ mir eine persönliche Website mit schönen Animationen erstellen (simuliert, PS sie sieht wirklich toll aus). Nur ein Prompt:

```
/skills react-native-design build a personal resume website based on @resume-website/resume.md
```

## Teilnehmen Sie an Open-Source-Projekten, beginnend mit Ihrem ersten PR

Möchten Sie sich an Open Source beteiligen, wissen aber nicht wo anzufangen? Qwen Code kann den gesamten Prozess automatisieren.

Werfen Sie ihm einen Issue-Link hin, dann wird Folgendes passieren:

1. Automatisches Verstehen der Problembeschreibung
2. Erforschung der Repository-Struktur
3. Lokalisierung der Fehlerposition
4. Bereitstellung von repariertem Code
5. Eigenständiges Einreichen eines PR

<video class="mt-5" src="https://cloud.video.taobao.com/vod/JV-7WNvq-GF9ZRHhrr1igplZDtHaFeNNoJ412hgkKYI.mp4" controls></video>

Keinerlei manuelles Eingreifen erforderlich. Natürlich empfehle ich beim ersten Mal, den Code manuell zu überprüfen, bevor Sie ihn komplett automatisieren lassen. (Fragen Sie nicht, wie ich weiß...)

Besonders nützlich für Studenten, die Open-Source-Erfahrung sammeln möchten. Sie können mit einfachen Bugfixes beginnen und sich allmählich an komplexere Funktionen beteiligen.

(Ich empfehle ernsthaft allen Informatikstudenten, es auszuprobieren – Open-Source-Erfahrung zählt wirklich bei der Jobsuche)

## Generieren Sie Demovideos per Code, sagen Sie goodbye zu Bearbeitungssoftware

Kurspräsentationen, Projektverteidigungen, Wettbewerbsroadshows – inklusive früherer Open-Source-Projekte – braucht man immer wieder Demovideos. Aber Premiere zu lernen und Videos zu schneiden kostet echt viel Zeit. (Wer schonmal geschnitten hat, weiß Bescheid!)

Basierend auf Remotion (einer Bibliothek zum Erstellen von Videos mit React) kann Qwen Code professionelle Demovideos per Code generieren.

Beispiel: Ein Produkteinleitungsvideo erstellen:

1. Installieren relevanter Remotion-Fähigkeiten
2. Skripte und Storyboard-Beschreibungen schreiben
3. Videocode generieren lassen
4. Rendering-Ausgabe

**Passende Szenarien:**

- Kursprojekt-Verteidigungsvideos
- Innovations-Unternehmenswettbewerb Roadshows
- Vereinswerbung
- Automatisierte persönliche Vlogs

Vorteil von Remotion: Man kann Videos mittels React-Komponenten erstellen, unterstützt Animationen, Übergänge, Untertitel, Audio-Bearbeitung sowie Versionskontrolle (ja, Videos lassen sich sogar mit Git verwalten – ziemlich cool!).

## Wie fange ich an?

1. Holen Sie sich einen API-Schlüssel vom BaiLian-Konsolen
2. Laden Sie Qwen Code herunter und starten Sie ihn mit `qwen`

```bash
# Linux/macOS
curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh | bash

# Windows
curl -fsSL -o %TEMP%\install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%\install-qwen.bat
```

3. Geben Sie `/auth` ein, wählen Sie «Alibaba Cloud Coding Plan» und fügen Sie den API-Schlüssel ein
4. Nutzen Sie `/model`, um das Modell auszuwählen und loszulegen!

Qwen Code verfügt über einen Skill-Marktplatz (ähnlich wie der VS Code Plugin-Marktplatz) mit verschiedenen fertigen Skill-Paketen. Mit dem `/extensions`-Befehl können Sie Erweiterungen auswählen und installieren oder weitere Plugins und Skills erkunden.

Versuchen Sie nicht gleich große Projekte anzugehen. Beginnen Sie mit einfachen Szenarien wie:

- Bitten Sie KI, eine Hello World Webseite zu schreiben
- Lassen Sie es unverständlichen Code erklären
- Generieren Sie eine Vorlage für Kursnotizen

Wenn Sie sich daran gewöhnt haben, probieren Sie nach und nach komplexere Aufgaben aus.

## Empfohlene Skill-Pakete für Universitätsstudenten

Auf Basis meiner Erfahrung sind diese Skill-Pakete besonders nützlich für Uni-Studenten:

| Skill-Paket | Zweck | Geeignete Studiengänge |
| --- | --- | --- |
| `skill-creator` | Erstellen von Skills nach Standards, eigene Skills erstellen | Alle Studiengänge |
| `find-skills` | Neue Skills entdecken und installieren | Alle Studiengänge |
| `react-native-design` | Schnelles Generieren von React-Webseiten | Informatik, Design, Neue Medien |
| `web-component-design` | Webseiten aus Screenshots reproduzieren | Alle Studiengänge |
| `remotion-best-practices` | Videos per Code generieren | Journalismus, Kommunikation, Informatik |

Installationsmethode:

```bash
npx skills add <SkillPaketName> -g
```

## Ein paar Tipps zur Nutzung

1. **Verlassen Sie sich nicht komplett auf KI.** Es ist ein Werkzeug, kein Ersatz. Verstehen Sie den Code selbst, begreifen Sie Ihre Hausaufgaben.
2. **Beginnen Sie mit Nachahmung.** Wenn jemand etwas Tolles mit bestimmten Skills erreicht hat, ahmen Sie dessen Prompt zuerst nach, danach passen Sie ihn nach und nach Ihrem eigenen Stil an.
3. **Nutzen Sie den Skill-Marktplatz gut.** Die meisten Bedürfnisse, an die Sie denken, haben wahrscheinlich bereits entsprechende Skill-Pakete. Suchen Sie zuerst, bevor Sie etwas Neues kreieren, um Radladungen zu vermeiden.
4. **Protokollieren Sie Ihre Prompts.** Notieren Sie effektive Prompts für direkte Wiederverwendung oder Feinjustierung nächstes Mal.
5. **Teilen Sie Ihre Arbeit.** Sobald Sie etwas Gutes erschaffen haben, veröffentlichen Sie es auf GitHub, Zhihu, Xiaohongshu – sowohl um Werke anzuhäufen als auch Gleichgesinnte kennenzulernen.
