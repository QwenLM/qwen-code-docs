## Was ist Qwen Code Cowork?

Einfach gesagt, ist es ein **kostenloser, Open-Source** Desktop-KI-Assistent, der auf dem Qwen Code SDK basiert.

Aber es ist nicht nur ein Chat-Fenster, sondern eine KI, die **tatsächlich die Hände schmutzig macht** und dir bei der Arbeit hilft:

| Fähigkeit | Beschreibung |
|------|------|
| 📝 Code schreiben und bearbeiten | Unterstützt jede Programmiersprache |
| 📁 Dateien verwalten | Erstellen, verschieben, umbenennen, organisieren |
| ⚡ Befehle ausführen | Build, Test, Deployment |
| 💬 Fragen beantworten | Zu deiner Codebasis oder zu jedem Thema |
| 🔧 Aufgaben automatisieren | Alles, was du in natürlicher Sprache beschreiben kannst |

Das Wichtigste dabei: **Völlig kostenlos, Open-Source und kontrollierbar, Daten werden lokal gespeichert**.

Im Vergleich dazu ist Claude Code nur für Max-Abonnenten verfügbar, mit monatlichen Kosten von 100-200 Dollar, während Qwen Code Cowork dieselben Fähigkeiten ohne Kosten bietet.

## Die ultimativen Anwendungsszenarien

Hier sind einige extrem nützliche Szenarien, die ich bei der tatsächlichen Nutzung entdeckt habe, jeweils mit konkreten Demonstrationsvideos.

### Szenario 1: Den chaotischen Desktop / Download-Ordner aufräumen

Desktop und Download-Ordner sind wahre digitale Müllhalden – Screenshots, Dokumente, Installationspakete und temporäre Dateien vermischen sich. Jedes Mal, wenn du etwas suchst, musst du ewig wühlen, es macht dich wahnsinnig, aber das manuelle Aufräumen dauert zu lange.

**Wie Cowork dir hilft**:

Sag es einfach: "Hilf mir, den Desktop aufzuräumen, sortiere Dateien nach Typ, Screenshots in den Screenshot-Ordner, Dokumente in den Dokumenten-Ordner, Installationspakete in den Installationspaket-Ordner"

Es wird:

1. Zuerst deinen Desktop scannen und alle Dateien auflisten
2. Dir eine Vorschau des Aufräumplans geben
3. Nach deiner Bestätigung die Klassifizierung und Verschiebung automatisch durchführen

 
<Callout>
    **Kleiner Tipp**:
    Verwende den Plan-Modus, damit die KI zuerst plant, und führe den Plan nach Bestätigung aus – so ist es sicherer und kontrollierbarer.
</Callout>

### Szenario 2: Massenumbenennung von Dateien (Fotos nach Datum umbenennen)

Die von Handys exportierten Fotos haben chaotische Namen wie IMG_0001, IMG_0002... Man kann überhaupt nicht erkennen, wann sie aufgenommen wurden. Du möchtest sie im Format „2024-01-15_Sonnenuntergang am Strand" umbenennen, aber Hunderte von Fotos manuell umzubenennen würde ewig dauern.

**Wie Cowork dir hilft**:

Sag es: "Benenne die Fotos in diesem Ordner nach dem Aufnahmedatum um, im Format YYYY-MM-DD_Nummer"

Es wird:

1. Die EXIF-Informationen (Aufnahmedatum) jedes Fotos lesen
2. Neue Dateinamen nach deinem angegebenen Format generieren
3. Alle Umbenennungsaktionen in einer Vorschau anzeigen
4. Nach Bestätigung die Massenumbenennung durchführen

### Szenario 3: Lokale Speichernutzung prüfen und große Dateien finden

Der Computer meldet wieder, dass der Speicherplatz nicht ausreicht, aber du weißt nicht, wovon der Platz belegt wird. Du möchtest aufräumen, fürchtest aber, wichtige Dateien zu löschen, und verschiedene Bereinigungssoftware öffnet dir Sorgen um die Datensicherheit.

**Wie Cowork dir hilft**:

Sag es: "Hilf mir, die Speichernutzung des Computers zu überprüfen, finde die 10 größten Ordner und Dateien, gib eine kurze Beschreibung des Inhalts sowie ob sie gelöscht werden können und welche Auswirkungen das Löschen haben könnte"

Es wird:

1. Die Datenträgernutzung scannen
2. Die Dateien und Ordner mit dem größten Speicherbedarf auflisten
3. Analysieren, welche sicher gelöscht werden können (z.B. Cache, temporäre Dateien)
4. Bereinigungsempfehlungen geben

<Callout type="error">
    **Achtung**:
    Bestätige immer vor dem Löschen von Dateien! Es wird empfohlen, sie zuerst in den Papierkorb zu verschieben und erst nach Bestätigung, dass alles in Ordnung ist, endgültig zu löschen.
</Callout>

### Szenario 4: Notizen integrieren und formatieren

Notizen sind überall verstreut: WeChat-Favoriten, Memo, Notion, Obsidian... Du möchtest sie an einem Ort zusammenführen, aber die Formate sind nicht einheitlich, und das manuelle Kopieren und Einfügen erfordert Formatierungsanpassungen – das ist zu mühsam.

**Wie Cowork dir hilft**:

Sag es: "Integriere alle docx-Dateien in diesem Ordner in eine Markdown-Datei, sortiere sie nach Erstellungsdatum, vereinheitliche das Format, zeige Dateinamen als Überschrift zweiter Ordnung an"

Es wird:

1. Den Inhalt aller Dateien lesen
2. Das Format erkennen und vereinheitlichen
3. Nach deinen angegebenen Regeln sortieren und integrieren
4. Eine formatgerechte Markdown-Datei generieren

### Szenario 5: Massenverarbeitung von Dokumenten (PDF-Inhalte extrahieren)

Du hast eine Menge PDF-Materialien heruntergeladen und möchtest den Text extrahieren, um Notizen zu machen, aber das einzelne Öffnen und Kopieren ist zu langsam. Einige PDFs sind gescannte Versionen, und das Kopieren ergibt nur wirre Zeichen.

**Wie Cowork dir hilft**:

Sag es: "Extrahiere den Inhalt aller PDF-Dateien in diesem Ordner und speichere ihn als entsprechende Markdown-Dateien"

Es wird:

1. Alle PDFs im Ordner durchlaufen
2. Den Textinhalt extrahieren
3. Für jedes PDF eine entsprechende txt-Datei generieren
4. Die Verarbeitungsergebnisse melden

## 🚀 Schnellstart


### Direkte Installation mit Qwen Code (empfohlen)

Sag Qwen Code einfach, und es wird in 3 Minuten erledigt.

```
Hilf mir, das Cowork-Projekt zu installieren und zu starten, die Repository-Adresse lautet: https://github.com/QwenLM/qwen-code-examples
```

### Installation aus dem Quellcode (fortgeschritten)

Wenn du aus dem Quellcode bauen möchtest, musst du zuerst Bun installieren:

```bash
# Bun installieren
curl -fsSL https://bun.sh/install | bash
exec /bin/zsh

# Repository klonen
git clone https://github.com/QwenLM/qwen-code-examples

# In das Verzeichnis wechseln und installieren
cd qwen-code-examples/apps/qwen-cowork
bun install

# Starten
bun run dev
```

## Nützliche Tipps

### 1. Den Plan-Modus nutzen

Für Dateioperationen, die nicht rückgängig gemacht werden können, wird empfohlen, die KI zuerst einen Plan erstellen zu lassen:

```
„Analysiere zuerst die Situation in diesem Ordner und gib mir einen Aufräumplan, führe ihn aber noch nicht aus"
```

Nach Bestätigung, dass der Plan in Ordnung ist, kannst du es ausführen lassen.

### 2. Arbeitsverzeichnis angeben

Beim Erstellen einer Sitzung kannst du ein Arbeitsverzeichnis angeben, sodass die KI nur in diesem Verzeichnis arbeitet – das ist sicherer.

### 3. Bestätigung bei sensiblen Operationen

Cowork fragt bei sensiblen Operationen wie Löschen oder Verschieben zuerst nach deiner Bestätigung, sodass du dir keine Sorgen um versehentliche Operationen machen musst.
