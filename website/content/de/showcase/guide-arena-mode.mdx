## Überblick

Bei wichtigem Code bitten Sie Kollegen um ein Review. Jetzt können Sie auch mehrere KI-Modelle gleichzeitig dieselbe Aufgabe bearbeiten lassen. Die Agent Arena ermöglicht es Ihnen, die Fähigkeiten verschiedener Modelle zu vergleichen — rufen Sie einfach `/arena` auf, wählen Sie Modelle aus, geben Sie Ihre Aufgabe ein und wählen Sie das beste Ergebnis.

## Was Sie damit tun können

- **Kernmodule refaktorisieren**: Mehrere Modelle gleichzeitig Lösungen vorschlagen lassen und die beste auswählen
- **Ansätze vergleichen**: Beim Schreiben von Algorithmen verschiedene Lösungsansätze der Modelle vergleichen und unterschiedliche Perspektiven kennenlernen
- **Neue Modelle evaluieren**: Mit realen Aufgaben bewerten, wie neue Modelle in Ihrem Projekt abschneiden

## Schritte

<Steps>

### /arena-Befehl eingeben

Geben Sie im Qwen Code Chat `/arena` ein und wählen Sie `start`, um den Arena-Modus zu betreten.

```bash
/arena start
```

### Teilnehmende Modelle auswählen

Wählen Sie aus der verfügbaren Modellliste die Modelle aus, die Sie vergleichen möchten. Drücken Sie `space`, um 2 oder mehr Modelle auszuwählen.

### Aufgabenbeschreibung eingeben

Geben Sie die Aufgabe ein, die jedes Modell bearbeiten soll, zum Beispiel:

```
Refaktorisiere diese Funktion, um Lesbarkeit und Leistung zu verbessern
```

### Ergebnisse vergleichen und das beste auswählen

Alle Modelle bearbeiten die Aufgabe gleichzeitig. Nach Abschluss können Sie ihre Lösungen vergleichen und das beste Ergebnis auf Ihren Code anwenden.

```bash
/arena select
```

</Steps>

<Callout type="info">
Der Arena-Modus eignet sich besonders für wichtige Code-Entscheidungen und lässt mehrere KI-Modelle verschiedene Lösungsperspektiven bieten.
</Callout>
