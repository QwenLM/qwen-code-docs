## 概要

重要なコードは同僚にレビューを頼みますよね。今では複数のAIモデルに同じタスクを同時に処理させることができます。Agent Arenaでは異なるモデルの能力を比較できます — `/arena` を呼び出し、モデルを選択し、タスクを入力して、最良の結果を選びましょう。

## これでできること

- **コアモジュールのリファクタリング**：複数のモデルに同時にソリューションを提案させ、最良のものを選ぶ
- **アプローチの比較**：アルゴリズムを書く際に異なるモデルの解法を比較し、異なる視点を学ぶ
- **新しいモデルの評価**：実際のタスクで新しいモデルがプロジェクトでどう機能するか評価する

## 手順

<Steps>

### /arenaコマンドを入力

Qwen Codeのチャットで `/arena` と入力し、`start` を選択してアリーナモードに入ります。

```bash
/arena start
```

### 参加するモデルを選択

利用可能なモデルリストから比較したいモデルを選択します。`space` を押して2つ以上のモデルを選択できます。

### タスクの説明を入力

各モデルに処理させたいタスクを入力します。例えば：

```
この関数をリファクタリングして、可読性とパフォーマンスを向上させてください
```

### 結果を比較して最良のものを選ぶ

すべてのモデルが同時にタスクを処理します。完了後、それぞれのソリューションを比較し、最良の結果をコードに適用できます。

```bash
/arena select
```

</Steps>

<Callout type="info">
アリーナモードは重要なコード決定に特に適しており、複数のAIモデルが異なる視点のソリューションを提供します。
</Callout>
