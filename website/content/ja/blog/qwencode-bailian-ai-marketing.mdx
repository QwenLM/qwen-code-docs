## まだ素材作りで徹夜していますか？

こんな経験はありませんか：

新商品の発売が決まり、マーケティングチームからモバイルスプラッシュ画面1枚、バナー広告3枚、商品詳細ページの画像一式、15秒の商品紹介動画、さらにSNS用コピーとWeChat記事が必要だと言われる——**締切は明後日**。

Photoshopを開き、30秒待ってロードが完了したら、デザイナーが休暇中だと気づく。動画エディタを開き、30分かけてクリップを並べ替え、間違った解像度でエクスポートしてしまう。ドキュメントを開いてコピーを書き始め、2行書いたところで画面を見つめるだけ。

**深夜2時まで残業して、ようやく素材が揃う。次の新商品発売で、また同じことの繰り返し。**

もしこう変わるとしたら：

- ターミナルで一文タイプするだけで、**10秒でモバイルスプラッシュ画面を生成**
- スマホで商品写真を撮って、Feishuグループで@ボットに送るだけで、**1分でプロ品質の商品画像が返ってくる**
- AIに「このモバイルバッテリーのレビュー記事を書いて」と言うだけで、**30秒で下書き完成**

これこそ2026年のあるべき働き方だと思いませんか？

この記事では、3つの実際のシナリオを使って、**Qwen Code + 百炼 CLI**で画像生成から写真のレタッチ、コピーライティングまでの全工程をどう処理するかをステップバイステップで紹介します。各シナリオには操作手順付きで、すぐに試せます。

## まずツールの紹介：Qwen Codeと百炼 CLIとは？

シンプルに言うと：

- **Qwen Code**はAIプログラミングアシスタントですが、コードを書くだけではありません——自然言語の指示を理解し、様々なツールを呼び出してタスクを完了する汎用AIエージェントです
- **百炼 CLI**（`bl`コマンド）はAlibaba Cloud Model Studioプラットフォームのコマンドラインツールで、テキストから画像、画像から画像、テキストから動画、画像から動画などのAI機能を統合しています

これらを組み合わせると、**自然言語を理解するマーケティング素材の生産ライン**が手に入ります。

### 必要なもの

- Qwen Code CLIのインストール：[Qwen Code 概要 \| Qwen Code Docs](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)
- 百炼 CLIのインストール：[Alibaba Cloud 百炼 CLI \| ワンコマンドで全AI機能を利用](https://bailian.console.aliyun.com/cli?source_channel=cn-beijing-cli&&utm_content=se_1023346373)
- Alibaba Cloud Model Studio APIキー（[Model Studioコンソール](https://bailian.console.aliyun.com)で申請）

準備できましたか？始めましょう。

## シナリオ1：一言でマーケティング画像を生成 🎨

セールや新商品発売のたびに、デザイナーにメイン画像、詳細ページ画像、プロモーションバナーの作成を依頼する必要があります。デザイナーのスケジュールはいっぱいで、小さな修正でも半日待ち。**急ぎの時には、とても待てません。**

今は欲しい画像を一文で説明するだけで、AIが生成します。白背景メイン画像、シーン画像、プロモーション雰囲気画像、テック感のある画像——すべて対応可能。

商品があれば、まず商品画像を生成してみましょう：

```
帮我生成 3 张充电宝的电商白底主图，要求：淡米色背景、专业棚拍风格、柔和灯光、底部轻微阴影、构图居中干净，3:4 尺寸
```

商品画像に満足したら、テキストでセールスポイントを追加したい場合は、もう一行追加するだけ：

```
已经生成了商品图，现在给这些图片加上宣传语，以及卖点内容。
```

<video class="mt-5" src="https://cloud.video.taobao.com/vod/jepzY16ry-ep9h6Arp_iKjY1_zqMGQR8Pp_xNSJhXMc.mp4" controls></video>

満足できない？微調整しましょう

画像が80点で、もう少し調整したい？Qwen Codeに直接言うだけ：

```
把第一张图的背景换成紫红渐变，加金色点缀，更适合双 11 氛围
```

テキストを変更したい？最初からやり直す必要はありません。既存の画像上で反復改善——調整するほど精度が上がります。HTMLデザインでテキストを個別に管理しているため、コピーの変更では画像を再生成する必要がなく、Tokenの節約になります。Qwen Codeに直接言うだけ：

```
把 20000mAh，修改为10000mAh
```

### 対応する画像タイプ

| タイプ | サイズ | 用途 |
| --- | --- | --- |
| ECメイン画像 | 1024×1024（正方形 1:1） | Taobao/JD商品メイン画像、プロモーション画像 |
| 商品詳細ページ | 1024×1366（縦型 3:4） | 詳細ページのシングルスクリーン表示 |
| モバイルスプラッシュ | 1080×1920（縦型 9:16） | アプリ起動広告 |
| バナー広告 | 1344×756（横型 16:9） | PCバナー |

すべてのサイズで**マーケティングテキスト付き商品画像**に対応——AIがまず純粋なビジュアルの商品ベース画像を生成し、次にHTMLでセールスポイント、スローガン、価格などのテキストを正確にオーバーレイし、最終的に完成したPNGとしてエクスポートします。テキストは歪みなく鮮明で、コピーの変更は画像を再生成せずに2秒で再レンダリングするだけです。

## シナリオ2：Feishuで写真を撮って、自動的に商品画像と動画を生成 📸

EC商品撮影はコストが高い——スタジオのレンタル、カメラマンの手配、後処理と、パッケージ全体で数千元からで、期間は1〜2週間。小さなチームには到底負担できません。さらに今はどのプラットフォームもショート動画を推進しているので、商品動画は追加の出費です。

今最も効率的なのは：**スマホで写真を撮って、Feishuグループで@ボットに送ると、AIが自動的に商品を識別し、白背景商品画像、テキスト付きマーケティング画像、5秒のショーケース動画を一括生成します。**

全プロセスでパソコン不要。

**ステップ1：写真を撮ってグループに送る**

スマホで商品のカジュアルな写真を撮り（背景は何でもOK——デスクの上、ソファの上、どこでも）、Feishuグループに送って@Qwen Code：

```
使用 open1v-product-visual 技能帮我生成2张商品图
```

ボットが自動的に3つのことを実行：

1. **商品を識別** — 撮影したものを自動判定
2. **標準商品画像を生成** — 識別した商品に基づいて画像を作成し、プロジェクトフォルダに保存
3. **結果を返す** — クラウドストレージの画像リンクを直接返す

**ステップ2：画像を選んで、マーケティングテキストを追加**

満足のいく白背景画像を選んで、次に言う：

```
现在我需要加上营销文字，小红书3:4尺寸，布局你可以自选，我要白底的。
```

ボットが識別した商品情報に基づいてマーケティングコピーを抽出し、画像にオーバーレイ：メインタイトル、サブタイトル、機能タグ、スペックデータ——完成した画像はそのまま出品可能。

テキストやレイアウトを変更したい？言うだけ：

```
把图片整体向右侧移动一点，产品主体居中，文字和排版不变
```

テキストの変更は画像の再生成不要——**2秒**で再レンダリングして完了。

**ステップ3：完成したアセットのリンクを取得**

```
现在我需要你生成图片的链接，上传到云空间并返回给我链接，只需要终稿版本
```

全工程の動画 👇

### 手に入るのは完全なプロジェクトフォルダ

```
product-charger-0610/
├── raw/          ← 撮影した原画像
├── product/      ← 生成された標準商品画像（白背景、シーン）
├── index.html    ← テキストレイアウトのソースファイル（テキストを手動で編集、2秒で再エクスポート）
├── final/        ← テキスト付き完成画像（すぐに出品可能）
└── video/        ← 5秒ショーケース動画
```

撮影から完全なアセットセットまで、**10分以内**。以前はカメラマン＋デザイナーで1週間かかっていました。

> Feishuを使わない？Qwen Codeのターミナルでも全く同じ操作ができます——写真のパスを送るだけです。

## シナリオ3：AIがマーケティングコピーを書く ✍️

コピーライティングは最も時間のかかるタスクの1つ。WeChat記事は長文ストーリーテリング、SNSは短くパンチのあるポスト、詳細ページは機能紹介コピー——**同じ商品、3チャネル、3つの異なるライティングスタイル**。毎回ゼロから書き始め、最終的にネタが尽きて同じフレーズの繰り返し。

今はAIに商品情報と配信チャネルを伝えるだけ。WeChatにはストーリーテリング、SNSにはライフスタイルポスト、詳細ページには機能ハイライト——一度の入力でマルチチャネル出力。

**例：ポータブルモバイルバッテリーの3チャネル向けコピーを生成**

Qwen Codeに入力：

```
帮我写一篇充电宝的推广文案。
产品名：PowerMax 迷你充电宝
卖点：1. 仅重 120g，比一颗鸡蛋还轻  2. 10000mAh 大容量  3. 支持 65W 快充，30 分钟充满 iPhone
目标人群：经常出差的商务人士
渠道：公众号、小红书、详情页
```

AIが3つの個別バージョンを出力します。デモ動画をご覧ください：

### 満足できない？会話を続けて、WeChatに直接公開も可能

内容に満足できない？Qwen Codeに何を調整するか伝えるだけ、例えば：

```
公众号版本内容再轻松一些，直接帮我发不到公众号后台
```

Qwen CodeにWeChat公式アカウントのAppIDとAppSecretを伝え、WeChat開発者プラットフォームでIPをホワイトリストに追加すれば、一度設定するだけで下書きボックスに直接公開できます——フォーマットの悩みから完全に解放：

## 3つのシナリオをつなげれば、完全な商品発売パイプライン

冒頭のシナリオに戻りましょう——新商品が発売され、マーケティング素材のフルセットが必要。

ワークフローはこうなります：

```
商品のカジュアルな写真を撮る
    ↓
シナリオ1：スプラッシュ画面、バナー、ポスターを生成  → 10分で全マーケティング画像完了
    ↓
シナリオ2：商品写真をレタッチ + 動画を生成         → 20分で商品画像と動画完了
    ↓
シナリオ3：WeChat/SNS/詳細ページのコピーを生成    → 10分で全チャネルコピー完了
```

**40分、1人、素材フルセット完了。** 以前はデザイナー＋カメラマン＋コピーライターが1週間かけて協力していた作業量です。

## 始め方

### 環境準備

1. **Qwen Codeのインストール**：[公式ドキュメント](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)に従う
2. **百炼 CLIのインストール**：[百炼 CLIドキュメント](https://bailian.console.aliyun.com/cli)に従う
3. **Qwen Codeの起動**：Qwen CodeにAPIキーと要件を伝える

### スキルパックのインストール

Qwen Codeを開いて言う：

```
帮我安装 open1v-skills 技能包：git clone https://github.com/joeytoday/open1v-skills.git ~/.agents/skills/open1v-skills，然后安装依赖
```

インストール後、一文で全ワークフローが起動：

- **商品画像を生成**：`帮我生成一张充电宝的白底主图`
- **マーケティングコピーを書く**：`帮我写一篇充电宝的推广文案，渠道：公众号、小红书、详情页`
- **WeChatに公開**：`把公众号文案发布到草稿箱`

これらのスキルをカスタマイズして、自社製品に合わせることもできます。

### Feishuとの連携（オプション）

Feishuグループで@ボットを使って上記すべてを行いたい？Qwen Codeに聞くだけ：

```
帮我配置飞书频道，我想在飞书群里 @机器人 生成商品图和文案，并一步步告诉我步骤
```

Qwen Codeがセットアップをガイドします——5分で完了。
