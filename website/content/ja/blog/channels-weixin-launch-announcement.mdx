## 🚀 クイックスタートガイド

### 準備

以下をご確認ください：

- ✅ Qwen Codeのプレビュー版をインストール `npm i @qwen-code/qwen-code@0.14.0-preview.0 -g`
- ✅ メッセージを受信できるWeChatアカウントを準備

> AI時代、誰がまだ手動でパラメータを設定しますか？`npx skills add https://github.com/QwenLM/qwen-code-examples --skill weixin-channel-setup` でこのweixin-channel-setupスキルをインストールし、Qwenを開いて `{'/'}skills weixin-channel-setup 設定を手伝ってください` と聞くだけ

昔ながらの設定方法がお好きなら、以下の3ステップをご参照ください：

![3ステップ設定フローチャート](https://img.alicdn.com/imgextra/i4/O1CN01aWJWAF1YwqRJ6ryra_!!6000000003124-0-tps-1376-768.jpg)

### ステップ1：スキャンして接続

ターミナルで実行：

```bash
qwen channel configure-weixin
```

ターミナルに「QR code URL」があります。リンクをコピーしてブラウザで開き、WeChatでスキャンしてください。

- WeChatのインストールが必要な場合があります
  スキャンが成功すると、以下が表示されます：

```
Connected to WeChat successfully!
Credentials saved. You can now start a weixin channel with:
  qwen channel start <name>
```

これで、WeChatの認証情報がローカルに安全に保存されました。

### ステップ2：チャンネル設定

`~/.qwen/settings.json`を編集し、以下の設定を追加：

```json
"channels": {
    "my-weixin": {
      "type": "weixin",
      "senderPolicy": "open",
      "sessionScope": "user",
      "model": "qwen3.5-plus",
      "instructions": "You are a concise coding assistant responding via WeChat. Keep responses under 500 characters. Use plain text only."
    }
  }
```

### ステップ3：起動とテスト

```bash
qwen channel start my-weixin
```

`[Channel] "my-weixin" is running.` というプロンプトが表示されたら、WeChatを開いて：

1. ボットアシスタントを見つける（先ほどスキャンしたもの）
2. メッセージを送信：「こんにちは、PythonのHello Worldを書いてください」
3. 返信を待つ...

🎉 **おめでとうございます！これでWeChatでコードが書けるようになりました！**

## 🔧 高度な使い方

### 1. AIのペルソナをカスタマイズ

`instructions`フィールドで、専用のAIアシスタントを作成できます：

```json
{
  "instructions": "あなたはシニアTypeScriptエンジニアです。質問に答える際：\n1. 実行可能なコード例を優先\n2. 重要な設計判断を説明\n3. 潜在的なエッジケースを指摘\n4. プロフェッショナルだがフレンドリーな口調を維持"
}
```

### 2. 起動時自動起動

正直なところ、この問題はQwen Codeに聞いたほうがいい結果が得られるかもしれません。設定を手伝ってもらいましょう～

さらなる使い方は、あなたの貢献をお待ちしています～
