# Qwen Code ドキュメント

Qwen Code ドキュメントへようこそ。Qwen Code は、ターミナル内で動作するエージェント型コーディングツールであり、これまで以上に素早くアイデアをコードに変換するお手伝いをします。

## ドキュメントのセクション

### [ユーザーガイド](./users/overview)

エンドユーザーとして Qwen Code を使用する方法を学びます。このセクションでは以下の内容を扱います：

- 基本的なインストールとセットアップ
- 一般的な使用パターン
- 機能と能力
- 設定オプション
- トラブルシューティング

### [開発者ガイド](./developers/architecture)

Qwen Code への貢献や開発方法を学びます。このセクションでは以下の内容を扱います：

- アーキテクチャの概要
- 貢献ガイドライン
- コア概念および実装の詳細
- 開発に使用するツールとワークフロー
- 拡張機能およびプラグインの開発
