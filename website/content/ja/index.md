# Qwen Code ドキュメント

Qwen Code ドキュメントへようこそ。Qwen Code はターミナル内で動作するエージェント型のコーディングツールであり、これまで以上に高速でアイデアをコードに変換するのを支援します。

## ドキュメントセクション

### [ユーザーガイド](./users/overview)

Qwen Code をエンドユーザーとして使用する方法を学びます。このセクションでは以下を扱います：

- 基本的なインストールとセットアップ
- 一般的な使用パターン
- 機能と能力
- 設定オプション
- トラブルシューティング

### [開発者ガイド](./developers/architecture)

Qwen Code への貢献方法と開発方法を学びます。このセクションでは以下を扱います：

- アーキテクチャ概要
- 貢献ガイドライン
- 中核概念と実装の詳細
- ツールと開発ワークフロー
- 拡張機能とプラグイン開発
