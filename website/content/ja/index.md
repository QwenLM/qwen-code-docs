# Qwen Code ドキュメント

Qwen Code ドキュメントへようこそ。Qwen Code は、ターミナル上で動作するエージェント型のコーディングツールで、アイデアをこれまでより迅速にコードに変換するお手伝いをします。

## ドキュメントセクション

### [ユーザーガイド](./users/overview)

Qwen Code をエンドユーザーとして使用する方法について学びます。このセクションでは以下の内容をカバーしています：

- 基本的なインストールとセットアップ
- よく使う使い方のパターン
- 機能と能力
- 設定オプション
- トラブルシューティング

### [開発者ガイド](./developers/contributing)

Qwen Code への貢献および開発方法について学びます。このセクションでは以下の内容をカバーしています：

- アーキテクチャ概要
- 貢献ガイドライン
- コアコンセプトと実装の詳細
- ツールと開発ワークフロー
- 拡張機能およびプラグイン開発
