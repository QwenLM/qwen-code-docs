# ファイルの除外

このドキュメントでは、Qwen Code の Qwen Ignore（`.qwenignore`）機能の概要を説明します。Qwen Code は `context.fileFiltering.customIgnoreFiles` で設定されたカスタム無視ファイルも認識します。デフォルトでは互換性ファイルである `.agentignore` と `.aiignore` が対象です。

Qwen Code には `.gitignore`（Git が使用）と同様に、ファイルを自動的に除外する機能があります。`.qwenignore` または設定済みのカスタム無視ファイルにパスを追加すると、この機能をサポートするツールからそれらが除外されます。ただし、Git などの他のサービスからは引き続き参照可能です。

## 動作の仕組み

これらの無視ファイルにパスを追加すると、Qwen の無視ルールに対応したツールは、マッチするファイルやディレクトリを処理対象から除外します。たとえば [`read_many_files`](../../developers/tools/multi-file) コマンドを使用した場合、`.qwenignore` や設定済みカスタム無視ファイルに記載されたパスは自動的に除外されます。

基本的に、これらの無視ファイルは `.gitignore` の規則に従います。

- 空行および `#` で始まる行は無視されます。
- 標準のグロブパターン（`*`、`?`、`[]` など）が使用できます。
- 末尾に `/` を付けるとディレクトリのみにマッチします。
- 先頭に `/` を付けると、無視ファイルからの相対パスとして固定されます。
- `!` でパターンを否定できます。

無視ファイルはいつでも更新できます。変更を反映するには、Qwen Code セッションを再起動する必要があります。

## 無視ファイルの使い方

| 手順                        | 説明                                                                                                                                          |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **無視ルールを有効にする** | プロジェクトのルートディレクトリに `.qwenignore`、デフォルトのカスタムファイル（`.agentignore` / `.aiignore`）、または設定済みのカスタム無視ファイルを作成する |
| **無視ルールを追加する**   | 無視ファイルを開き、除外するパスを追加する（例: `/archive/` や `apikeys.txt`）                                                                |

デフォルトでは、Qwen Code は `.qwenignore`、`.agentignore`、`.aiignore` を読み込みます。
別のカスタム無視ファイルを使用する場合は、以下を設定してください。

```json
{
  "context": {
    "fileFiltering": {
      "customIgnoreFiles": [".cursorignore"]
    }
  }
}
```

`context.fileFiltering.respectQwenIgnore` が有効な場合、`.qwenignore` は常に読み込まれます。カスタム無視ファイルのパスはプロジェクトルートからの相対パスで指定します。

### 無視ファイルの例

サポートされている任意の無視ファイルを使用して、ディレクトリやファイルを除外できます。

```
# Exclude your /packages/ directory and all subdirectories
/packages/

# Exclude your apikeys.txt file
apikeys.txt
```

`*` を使ったワイルドカードも利用できます。

```
# Exclude all .md files
*.md
```

`!` を使って、除外対象から特定のファイルやディレクトリを除外することもできます。

```
# Exclude all .md files except README.md
*.md
!README.md
```

無視ファイルからパスを削除するには、該当する行を削除してください。
