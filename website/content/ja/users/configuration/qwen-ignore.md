# ファイルの無視

このドキュメントでは、Qwen Code の Qwen Ignore（`.qwenignore`）機能の概要を説明します。Qwen Code は、`context.fileFiltering.customIgnoreFiles` で設定されたカスタム無視ファイル（デフォルトでは互換性のある `.agentignore` および `.aiignore`）も認識します。

Qwen Code には、`.gitignore`（Git で使用）と同様に、ファイルを自動的に無視する機能があります。`.qwenignore` または設定されたカスタム無視ファイルにパスを追加すると、この機能をサポートするツールからそれらが除外されますが、他のサービス（Git など）からは引き続き参照できます。

## 仕組み

無視ファイルのいずれかにパスを追加すると、Qwen の無視ルールに従うツールは、マッチするファイルやディレクトリを操作から除外します。たとえば、[`read_many_files`](../../developers/tools/multi-file) コマンドを使用すると、`.qwenignore` または設定されたカスタム無視ファイル内のパスが自動的に除外されます。

これらの無視ファイルは、大部分において `.gitignore` ファイルの規則に従います。

- 空行と行頭の `#` は無視されます。
- 標準のグロブパターン（`*`、`?`、`[]` など）がサポートされています。
- 末尾に `/` を付けると、ディレクトリのみにマッチします。
- 先頭に `/` を付けると、パスが無視ファイルからの相対パスとして固定されます。
- `!` でパターンを否定します。

これらの無視ファイルはいつでも更新できます。変更を適用するには、Qwen Code セッションを再起動する必要があります。

## 無視ファイルの使い方

| 手順                        | 説明                                                                                                                             |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **無視ルールを有効にする** | プロジェクトルートディレクトリに `.qwenignore`、デフォルトのカスタムファイル（`.agentignore` / `.aiignore`）、または設定したカスタム無視ファイルを作成します |
| **無視ルールを追加する**    | 無視ファイルを開き、無視したいパスを追加します（例：`/archive/` や `apikeys.txt`）                                                |

デフォルトでは、Qwen Code は `.qwenignore`、`.agentignore`、`.aiignore` を読み取ります。
別のカスタム無視ファイルを使用するには、次のように設定します。

```json
{
  "context": {
    "fileFiltering": {
      "customIgnoreFiles": [".cursorignore"]
    }
  }
}
```

`.qwenignore` は、`context.fileFiltering.respectQwenIgnore` が有効な場合、常に含まれます。カスタム無視ファイルのパスは、プロジェクトルートからの相対パスです。

### 無視ファイルの例

サポートされている任意の無視ファイルを使用して、ディレクトリやファイルを無視できます。

```
# /packages/ ディレクトリとそのサブディレクトリをすべて除外
/packages/

# apikeys.txt ファイルを除外
apikeys.txt
```

無視ファイルでは、`*` を使用してワイルドカードを利用できます。

```
# すべての .md ファイルを除外
*.md
```

最後に、`!` を使用して、除外からファイルやディレクトリを除外（復活）できます。

```
# README.md を除くすべての .md ファイルを除外
*.md
!README.md
```

無視ファイルからパスを削除するには、該当する行を削除します。
