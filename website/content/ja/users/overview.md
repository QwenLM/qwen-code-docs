# Qwen Code 概要

> Qwen Code について学びましょう。これは、ターミナルで動作し、アイデアをこれまでより速くコードに変えることを支援する Qwen のエージェント型コーディングツールです。

## 30秒で始める

前提条件：

- [Qwen Code](https://chat.qwen.ai/auth?mode=register) アカウント
- [Node.js 20+](https://nodejs.org/zh-cn/download) が必要です。`node -v` でバージョンを確認できます。インストールされていない場合は、以下のコマンドでインストールしてください。

### Qwen Code をインストール：

**NPM**（推奨）

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew**（macOS、Linux）

```bash
brew install qwen-code
```

### Qwen Code の使い始め:

```bash
cd your-project
qwen
```

**Qwen OAuth (無料)** 認証を選択し、指示に従ってログインしてください。その後、コードベースの理解から始めましょう。以下のコマンドを試してみてください:

```
what does this project do?
```

## Qwen Code がやってくれること

- **説明から機能を構築**: 自然言語で構築したいものを Qwen Code に伝えると、計画を作成し、コードを書き、正しく動作することを保証します。
- **デバッグと問題の修正**: バグの内容を説明するか、エラーメッセージを貼り付けてください。Qwen Code はコードベースを分析し、問題を特定して修正を実装します。
- **あらゆるコードベースをナビゲート**: チームのコードベースについて何でも質問でき、丁寧な回答を得られます。Qwen Code はプロジェクト全体の構造を把握し続け、最新の情報をウェブから取得できます。また、[MCP](./features/mcp) を使用すれば、Google Drive、Figma、Slack などの外部データソースから情報を取り込むことも可能です。
- **面倒な作業を自動化**: 細かい lint の問題を修正したり、マージコンフリクトを解決したり、リリースノートを作成したりできます。これらすべてを、開発者のマシンから単一のコマンドで実行するか、CI で自動的に処理できます。

## 開発者がQwen Codeを愛する理由

- **ターミナルで動作する**: 新しいチャットウィンドウでも、新しいIDEでもありません。Qwen Codeはあなたがすでに作業している場所、そしてすでに使っているツールを使って、あなたの作業に合わせてきます。
- **アクションを起こせる**: Qwen Codeはファイルの直接編集、コマンドの実行、コミットの作成が可能です。もっと必要ですか？[MCP](./features/mcp)により、Qwen CodeはGoogle Drive内の設計文書を読み込み、Jiraのチケットを更新したり、_あなたの_カスタム開発者用ツールを使用することさえできます。
- **Unix哲学**: Qwen Codeは組み合わせ可能でスクリプト化可能です。`tail -f app.log | qwen -p "このログストリームに異常が見られたらSlackで通知してください"` というような使い方が _実際に可能です_ 。CIでは `qwen -p "新しいテキスト文字列があれば、フランス語に翻訳して@lang-fr-teamのレビューのためにPRを作成してください"` のように使うことができます。
