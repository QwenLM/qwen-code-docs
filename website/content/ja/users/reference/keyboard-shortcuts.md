# Qwen Code キーボードショートカット

このドキュメントでは、Qwen Code で利用可能なキーボードショートカットを一覧します。

## 全般

| ショートカット                       | 説明                                                                                                                                                                                                                                                                                               |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | ダイアログや候補を閉じる。                                                                                                                                                                                                                                                                            |
| `Ctrl+C`                       | 実行中のリクエストをキャンセルし、入力をクリアする。2回押すとアプリケーションを終了する。                                                                                                                                                                                                                                      |
| `Ctrl+D`                       | 入力が空の場合にアプリケーションを終了する。2回押して確認する。                                                                                                                                                                                                                                       |
| `Ctrl+L`                       | 画面をクリアする。                                                                                                                                                                                                                                                                                         |
| `Ctrl+O`                       | コンパクトモードの切り替え（ツール出力とthinkingの表示/非表示）。                                                                                                                                                                                                                                                 |
| `Ctrl+S`                       | 長いレスポンスを切り捨てずに全て表示する。ターミナルのスクロールバックで全体を確認できる。                                                                                                                                                                                                                                     |
| `Ctrl+T`                       | ツールの説明表示を切り替える。                                                                                                                                                                                                                                                                          |
| `Ctrl+B`                       | フォアグラウンドのシェルコマンド実行中：バックグラウンドタスクに昇格する。子プロセスは実行を継続し、エージェントのターンがアンブロックされ、シェルは `/tasks` とバックグラウンドタスクダイアログに表示される。シェルが実行されていない場合は無効（`Ctrl+B` はプロンプトエリアのバインディング（カーソル左移動）にフォールスルーする）。 |
| `Alt/Option+M`                 | Markdown出力をリッチレンダリングプレビューと生ソースモードで切り替える。macOSではターミナルがOptionをMetaとして送信するよう設定が必要。                                                                                                                                                               |
| `Shift+Tab`（Windowsでは `Tab`） | 承認モードを循環する（`plan` → `default` → `auto-edit` → `auto` → `yolo`）                                                                                                                                                                                                                                |

## 入力プロンプト

| ショートカット                                              | 説明                                                                                                                         |
| ----------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                   | 入力が空のときにシェルモードを切り替える。                                                                                          |
| `?`                                                   | 入力が空のときにキーボードショートカット表示を切り替える。                                                                          |
| `Ctrl+Enter` / `Cmd+Enter` / `Shift+Enter` / `Ctrl+J` | 改行を挿入する。                                                                                                                   |
| `Down Arrow`                                          | 1行下に移動し、末尾にスナップし、次の履歴に移動する。                                                                      |
| `Enter`                                               | 現在のプロンプトを送信する。                                                                                                          |
| `Meta+Delete` / `Ctrl+Delete`                         | カーソル右側の単語を削除する。                                                                                                         |
| `Tab`                                                 | 候補がある場合、現在の候補をオートコンプリートする。                                                                                  |
| `Up Arrow`                                            | 1行上に移動し、先頭にスナップし、前の履歴に移動する。                                                                      |
| `Ctrl+A` / `Home`                                     | カーソルを行の先頭に移動する。                                                                                                       |
| `Ctrl+B` / `Left Arrow`                               | カーソルを1文字左に移動する。                                                                                                          |
| `Ctrl+C`                                              | 入力プロンプトをクリアする。                                                                                                              |
| `Esc`（2回押し）                                  | 入力プロンプトをクリアする。                                                                                                             |
| `Ctrl+D` / `Delete`                                   | カーソル右側の文字を削除する。                                                                                                    |
| `Ctrl+E` / `End`                                      | カーソルを行の末尾に移動する。                                                                                                       |
| `Ctrl+F` / `Right Arrow`                              | カーソルを1文字右に移動する。                                                                                                         |
| `Ctrl+H` / `Backspace`                                | カーソル左側の文字を削除する。                                                                                                     |
| `Ctrl+K`                                              | カーソルから行末まで削除する。                                                                                                      |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`      | カーソルを1単語左に移動する。                                                                                                               |
| `Ctrl+N`                                              | 1行下に移動し、末尾にスナップし、次の履歴に移動する。                                                                      |
| `Ctrl+P`                                              | 1行上に移動し、先頭にスナップし、前の履歴に移動する。                                                                      |
| `Ctrl+R`                                              | 入力/シェル履歴を逆方向検索する。                                                                                                         |
| `Ctrl+Y`                                              | 最後に失敗したリクエストを再試行する。                                                                                                      |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F`    | カーソルを1単語右に移動する。                                                                                                              |
| `Ctrl+U`                                              | カーソルから行頭まで削除する。                                                                                                      |
| `Ctrl+V`（Windows: `Alt+V`）                           | クリップボードの内容を貼り付ける。クリップボードに画像が含まれる場合、保存されてプロンプトに参照が挿入される。 |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`        | カーソル左側の単語を削除する。                                                                                                          |
| `Ctrl+X` / `Meta+Enter`                               | 現在の入力を外部エディタで開く。                                                                                                       |

## 候補

| ショートカット                | 説明                            |
| ----------------------- | -------------------------------------- |
| `Down Arrow` / `Ctrl+N` | 候補を下に移動する。 |
| `Tab` / `Enter`         | 選択した候補を確定する。        |
| `Up Arrow` / `Ctrl+P`   | 候補を上に移動する。   |

## ラジオボタン選択

| ショートカット                      | 説明                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` / `Ctrl+N` | 選択を下に移動する。                                                                                          |
| `Enter`                       | 選択を確定する。                                                                                            |
| `Up Arrow` / `k` / `Ctrl+P`   | 選択を上に移動する。                                                                                            |
| `1-9`                         | 番号で項目を選択する。                                                                                 |
| （複数桁）                 | 9より大きい番号の項目は、数字を素早く続けて押すことで対応する項目を選択できる。 |

## 履歴スクロールバック

`ui.useTerminalBuffer` が有効な場合のみ動作する（設定 → UI → 仮想化履歴）。このモードでは会話履歴がホストターミナルのスクロールバックではなくアプリ内ビューポートでレンダリングされるため、以下のキーがターミナルのネイティブスクロールを置き換える。

| ショートカット        | 説明                                          |
| --------------- | ---------------------------------------------------- |
| `Shift+Up`      | 履歴を1行上にスクロールする。                          |
| `Shift+Down`    | 履歴を1行下にスクロールする。                        |
| `PgUp`          | 履歴を1ページ上にスクロールする（ビューポートの高さ）。        |
| `PgDn`          | 履歴を1ページ下にスクロールする（ビューポートの高さ）。      |
| `Ctrl+Home`     | 会話の先頭にジャンプする。                 |
| `Ctrl+End`      | 末尾にジャンプする（ライブ自動追従を再有効化）。 |
| **マウスホイール** | 履歴をスクロールする（1ティックで3行）。                   |

`ui.useTerminalBuffer` が有効な場合、ターミナルはマウスイベントを qwen-code に転送するため、ホイールでアプリ内ビューポートを操作できる。副作用として、**ネイティブのクリック＆ドラッグによるテキスト選択がプログラムに消費される** — マウスキャプチャを回避してテキストを通常の方法で選択するには、ドラッグ中に `Shift`（macOS Terminal / iTerm では `Option`）を押し続ける。

### tmux トラックパッドスクロール

tmux 内では、一部のターミナルがトラックパッドやホイール操作を qwen-code が受け取る前に通常の `Up Arrow` / `Down Arrow` シーケンスに変換する。これらのバイト列は実際の矢印キー入力と同一のため、qwen-code はビューポートのスクロールを意図しているのかプロンプト履歴の操作を意図しているのかを判別できない。

tmux でトラックパッドスクロールがプロンプト履歴を変更してしまう場合は、`ui.useTerminalBuffer` を有効にし、`Shift+Up` / `Shift+Down` か、tmux がホイールイベントをアプリに転送する場合はマウスホイールを使用する。ホストのスクロールバックを使いたい場合は、ホイールイベント用の tmux マウスバインディングを調整する。
