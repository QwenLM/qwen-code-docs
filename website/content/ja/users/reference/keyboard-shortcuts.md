# Qwen Code キーボードショートカット

このドキュメントでは、Qwen Code で利用可能なキーボードショートカットを一覧で示します。

## 一般

| ショートカット                       | 説明                                                                                                           |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | ダイアログと候補を閉じます。                                                                                        |
| `Ctrl+C`                       | 進行中のリクエストをキャンセルし、入力をクリアします。2回押すとアプリケーションを終了します。                                  |
| `Ctrl+D`                       | 入力が空の場合、アプリケーションを終了します。2回押して確定します。                                                   |
| `Ctrl+L`                       | 画面をクリアします。                                                                                                     |
| `Ctrl+O`                       | コンパクトモードの切り替え（ツール出力と思考プロセスの表示/非表示）。                                                             |
| `Ctrl+S`                       | 長い応答の切り捨てを無効化し、完全に表示します。ターミナルのスクロールバック機能を使用して出力全体を確認してください。 |
| `Ctrl+T`                       | ツールの説明の表示を切り替えます。                                                                              |
| `Shift+Tab`（Windows では `Tab`） | 承認モードを切り替えます（`plan` → `default` → `auto-edit` → `yolo`）                                                      |

## 入力プロンプト

| ショートカット                                           | 説明                                                                                                                         |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                | 入力が空の場合、シェルモードを切り替えます。                                                                                          |
| `?`                                                | 入力が空の場合、キーボードショートカットの表示を切り替えます。                                                                          |
| `\`（行末） + `Enter`                     | 改行を挿入します。                                                                                                                   |
| `Down Arrow`                                       | 入力履歴を下に移動します。                                                                                            |
| `Enter`                                            | 現在のプロンプトを送信します。                                                                                                          |
| `Meta+Delete` / `Ctrl+Delete`                      | カーソル右側の単語を削除します。                                                                                         |
| `Tab`                                              | 候補が存在する場合、現在の候補を自動補完します。                                                                                  |
| `Up Arrow`                                         | 入力履歴を上に移動します。                                                                                              |
| `Ctrl+A` / `Home`                                  | カーソルを行頭に移動します。                                                                                       |
| `Ctrl+B` / `Left Arrow`                            | カーソルを1文字左に移動します。                                                                                          |
| `Ctrl+C`                                           | 入力プロンプトをクリアします。                                                                                                              |
| `Esc`（2回押し）                               | 入力プロンプトをクリアします。                                                                                                             |
| `Ctrl+D` / `Delete`                                | カーソル右側の文字を削除します。                                                                                    |
| `Ctrl+E` / `End`                                   | カーソルを行末に移動します。                                                                                             |
| `Ctrl+F` / `Right Arrow`                           | カーソルを1文字右に移動します。                                                                                         |
| `Ctrl+H` / `Backspace`                             | カーソル左側の文字を削除します。                                                                                     |
| `Ctrl+K`                                           | カーソル位置から行末までを削除します。                                                                                      |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`   | カーソルを1単語左に移動します。                                                                                               |
| `Ctrl+N`                                           | 入力履歴を下に移動します。                                                                                            |
| `Ctrl+P`                                           | 入力履歴を上に移動します。                                                                                              |
| `Ctrl+R`                                           | 入力/シェル履歴を逆順に検索します。                                                                                         |
| `Ctrl+Y`                                           | 最後に失敗したリクエストを再試行します。                                                                                                      |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F` | カーソルを1単語右に移動します。                                                                                              |
| `Ctrl+U`                                           | カーソル位置から行頭までを削除します。                                                                                |
| `Ctrl+V`（Windows: `Alt+V`）                        | クリップボードの内容を貼り付けます。クリップボードに画像が含まれている場合、画像が保存され、プロンプトにその参照が挿入されます。 |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | カーソル左側の単語を削除します。                                                                                          |
| `Ctrl+X` / `Meta+Enter`                            | 現在の入力を外部エディタで開きます。                                                                                       |

## 候補

| ショートカット        | 説明                            |
| --------------- | -------------------------------------- |
| `Down Arrow`    | 候補リストを下に移動します。 |
| `Tab` / `Enter` | 選択した候補を確定します。        |
| `Up Arrow`      | 候補リストを上に移動します。   |

## ラジオボタンの選択

| ショートカット           | 説明                                                                                                   |
| ------------------ | ------------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` | 選択項目を下に移動します。                                                                                          |
| `Enter`            | 選択を確定します。                                                                                            |
| `Up Arrow` / `k`   | 選択項目を上に移動します。                                                                                            |
| `1-9`              | 番号で項目を選択します。                                                                                 |
| `（複数桁）`      | 9より大きい番号の項目を選択するには、対応する数字を素早く連続して押します。 |

## IDE 統合

| ショートカット | 説明                       |
| -------- | --------------------------------- |
| `Ctrl+G` | IDE から受信したコンテキスト CLI を表示します。 |
