# Qwen Code キーボードショートカット

このドキュメントでは、Qwen Code で利用可能なキーボードショートカットを一覧しています。

## 一般

| ショートカット | 説明                                                                                                               |
| -------------- | ------------------------------------------------------------------------------------------------------------------ |
| `Esc`          | ダイアログと候補を閉じます。                                                                                        |
| `Ctrl+C`       | 進行中のリクエストをキャンセルし、入力をクリアします。2回押すとアプリケーションを終了します。                      |
| `Ctrl+D`       | 入力が空の場合、アプリケーションを終了します。2回押して確認してください。                                          |
| `Ctrl+L`       | 画面をクリアします。                                                                                                |
| `Ctrl+O`       | デバッグコンソールの表示を切り替えます。                                                                            |
| `Ctrl+S`       | 長いレスポンスを完全に表示できるようにし、切り捨てを無効にします。端末のスクロールバックを使用して全体を表示します。 |
| `Ctrl+T`       | ツール説明の表示を切り替えます。                                                                                    |
| `Shift+Tab`    | 承認モードを循環切り替えします（`plan` → `default` → `auto-edit` → `yolo`）。                                      |

## 入力プロンプト

| ショートカット                                     | 説明                                                                                                                               |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                | 入力が空のときにシェルモードを切り替える。                                                                                            |
| `\`（行末）+ `Enter`                               | 改行を挿入する。                                                                                                                     |
| `下矢印キー`                                       | 入力履歴を下に移動する。                                                                                                             |
| `Enter`                                            | 現在のプロンプトを送信する。                                                                                                         |
| `Meta+Delete` / `Ctrl+Delete`                      | カーソルの右側の単語を削除する。                                                                                                     |
| `Tab`                                              | サジェストがある場合は、現在のサジェストを自動補完する。                                                                              |
| `上矢印キー`                                       | 入力履歴を上に移動する。                                                                                                             |
| `Ctrl+A` / `Home`                                  | 行の先頭にカーソルを移動する。                                                                                                       |
| `Ctrl+B` / `左矢印キー`                            | カーソルを1文字左に移動する。                                                                                                        |
| `Ctrl+C`                                           | 入力プロンプトをクリアする。                                                                                                         |
| `Esc`（ダブルプレス）                              | 入力プロンプトをクリアする。                                                                                                         |
| `Ctrl+D` / `Delete`                                | カーソルの右側の文字を削除する。                                                                                                     |
| `Ctrl+E` / `End`                                   | 行の末尾にカーソルを移動する。                                                                                                       |
| `Ctrl+F` / `右矢印キー`                            | カーソルを1文字右に移動する。                                                                                                        |
| `Ctrl+H` / `Backspace`                             | カーソルの左側の文字を削除する。                                                                                                     |
| `Ctrl+K`                                           | カーソル位置から行末までを削除する。                                                                                                 |
| `Ctrl+左矢印キー` / `Meta+左矢印キー` / `Meta+B`   | カーソルを1単語左に移動する。                                                                                                        |
| `Ctrl+N`                                           | 入力履歴を下に移動する。                                                                                                             |
| `Ctrl+P`                                           | 入力履歴を上に移動する。                                                                                                             |
| `Ctrl+右矢印キー` / `Meta+右矢印キー` / `Meta+F`   | カーソルを1単語右に移動する。                                                                                                        |
| `Ctrl+U`                                           | カーソル位置から行頭までを削除する。                                                                                                 |
| `Ctrl+V`                                           | クリップボードの内容を貼り付ける。クリップボードに画像が含まれている場合、保存され、その参照がプロンプトに挿入される。               |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | カーソルの左側の単語を削除する。                                                                                                     |
| `Ctrl+X` / `Meta+Enter`                            | 現在の入力を外部エディタで開く。                                                                                                     |

## サジェスト

| ショートカット  | 説明                                   |
| --------------- | -------------------------------------- |
| `Down Arrow`    | サジェストを下に移動します。           |
| `Tab` / `Enter` | 選択されたサジェストを受け入れます。   |
| `Up Arrow`      | サジェストを上に移動します。           |

## ラジオボタン選択

| ショートカット     | 説明                                                                                                         |
| ------------------ | ----------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` | 選択を下に移動します。                                                                                      |
| `Enter`            | 選択を確定します。                                                                                          |
| `Up Arrow` / `k`   | 選択を上に移動します。                                                                                      |
| `1-9`              | 番号でアイテムを選択します。                                                                                |
| （複数桁）         | 9より大きい番号のアイテムについては、対応するアイテムを選択するために素早く連続して数字キーを押してください。 |

## IDE 連携

| ショートカット | 説明                             |
| -------------- | -------------------------------- |
| `Ctrl+G`       | IDE から受信したコンテキスト CLI を表示 |
