# Qwen Code のキーボードショートカット

このドキュメントでは、Qwen Code で利用可能なキーボードショートカットを一覧表示します。

## 一般

| ショートカット                       | 説明                                                                                                                   |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| `Esc`                                | ダイアログおよび候補を閉じます。                                                                                         |
| `Ctrl+C`                             | 実行中のリクエストをキャンセルし、入力をクリアします。アプリケーションを終了するには、2 回押します。                      |
| `Ctrl+D`                             | 入力が空の場合はアプリケーションを終了します。確認のためには、2 回押します。                                            |
| `Ctrl+L`                             | 画面をクリアします。                                                                                                     |
| `Ctrl+O`                             | デバッグコンソールの表示を切り替えます。                                                                                 |
| `Ctrl+S`                             | 長い応答を完全に表示し、切り捨てを無効化します。ターミナルのスクロールバック機能を使って、出力全体を表示できます。         |
| `Ctrl+T`                             | ツールの説明の表示を切り替えます。                                                                                       |
| `Shift+Tab` (`Windows` では `Tab`) | 承認モードを切り替えます（`plan` → `default` → `auto-edit` → `yolo`）                                                    |

## 入力プロンプト

| ショートカット                                         | 説明                                                                                                                                 |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| `!`                                                    | 入力が空のときにシェルモードを切り替えます。                                                                                         |
| `?`                                                    | 入力が空のときにキーボードショートカット一覧を表示・非表示に切り替えます。                                                           |
| `\`（行末）＋ `Enter`                                  | 改行を挿入します。                                                                                                                   |
| `↓`（下矢印キー）                                      | 入力履歴を下方向に移動します。                                                                                                       |
| `Enter`                                                | 現在のプロンプトを送信します。                                                                                                       |
| `Meta+Delete` / `Ctrl+Delete`                          | カーソルの右側にある単語を削除します。                                                                                               |
| `Tab`                                                  | 一致する候補が存在する場合、現在の候補を自動補完します。                                                                             |
| `↑`（上矢印キー）                                      | 入力履歴を上方向に移動します。                                                                                                       |
| `Ctrl+A` / `Home`                                      | カーソルを行頭に移動します。                                                                                                         |
| `Ctrl+B` / `←`（左矢印キー）                           | カーソルを1文字左に移動します。                                                                                                      |
| `Ctrl+C`                                               | 入力プロンプトをクリアします。                                                                                                       |
| `Esc`（2回連打）                                       | 入力プロンプトをクリアします。                                                                                                       |
| `Ctrl+D` / `Delete`                                    | カーソルの右側にある文字を削除します。                                                                                               |
| `Ctrl+E` / `End`                                       | カーソルを行末に移動します。                                                                                                         |
| `Ctrl+F` / `→`（右矢印キー）                           | カーソルを1文字右に移動します。                                                                                                      |
| `Ctrl+H` / `Backspace`                                 | カーソルの左側にある文字を削除します。                                                                                               |
| `Ctrl+K`                                               | カーソルから行末までを削除します。                                                                                                   |
| `Ctrl+←`（左矢印キー） / `Meta+←`（左矢印キー） / `Meta+B` | カーソルを1語分左に移動します。                                                                                                      |
| `Ctrl+N`                                               | 入力履歴を下方向に移動します。                                                                                                       |
| `Ctrl+P`                                               | 入力履歴を上方向に移動します。                                                                                                       |
| `Ctrl+R`                                               | 入力履歴／シェル履歴を逆順で検索します。                                                                                             |
| `Ctrl+Y`                                               | 最後に失敗したリクエストを再実行します。                                                                                             |
| `Ctrl+→`（右矢印キー） / `Meta+→`（右矢印キー） / `Meta+F` | カーソルを1語分右に移動します。                                                                                                      |
| `Ctrl+U`                                               | カーソルから行頭までを削除します。                                                                                                   |
| `Ctrl+V`（Windows: `Alt+V`）                           | クリップボードの内容を貼り付けます。クリップボードに画像が含まれている場合は、その画像が保存され、プロンプト内に参照が挿入されます。 |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`         | カーソルの左側にある単語を削除します。                                                                                               |
| `Ctrl+X` / `Meta+Enter`                                | 現在の入力を外部エディタで開きます。                                                                                                 |

## サジェスト

| ショートカット     | 説明                                   |
| ------------------ | -------------------------------------- |
| `↓（下矢印）`      | サジェストを下方向に移動します。         |
| `Tab` / `Enter`    | 選択中のサジェストを適用します。         |
| `↑（上矢印）`      | サジェストを上方向に移動します。         |

## ラジオボタン選択

| ショートカット         | 説明                                                                                           |
| ---------------------- | ------------------------------------------------------------------------------------------------ |
| `↓` / `j`              | 選択を下に移動します。                                                                           |
| `Enter`                | 選択を確定します。                                                                               |
| `↑` / `k`              | 選択を上に移動します。                                                                           |
| `1–9`                  | 番号で項目を選択します。                                                                         |
| （複数桁）             | 番号が 9 より大きい項目の場合は、対応する項目を選択するために、数字を連続して素早く入力します。 |

## IDE 統合

| ショートカット | 説明                                 |
| ------------ | ------------------------------------ |
| `Ctrl+G`     | IDE から受信したコンテキスト CLI を表示 |
