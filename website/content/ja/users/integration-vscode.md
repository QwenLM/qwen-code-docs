# Visual Studio Code

> VS Code 拡張機能（ベータ版）では、IDE に直接統合されたネイティブなグラフィカルインターフェースを通じて、Qwen の変更をリアルタイムで確認できます。これにより、Qwen Code へのアクセスと対話がより簡単になります。

### 主な機能

- **ネイティブな IDE 体験**: Qwen アイコンからアクセス可能な専用 Qwen Code サイドバーパネル
- **編集の自動承認モード**: Qwen による変更を即座に自動適用
- **ファイル管理**: `@` を使ってファイルをメンションしたり、システムのファイルピッカーを使ってファイルや画像を添付したりできます
- **会話履歴**: 過去の会話にアクセス可能
- **複数セッション対応**: 複数の Qwen Code セッションを同時に実行可能

### 必要条件

- VS Code 1.85.0 以降

### インストール

[Visual Studio Code 拡張機能マーケットプレイス](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion) から拡張機能をダウンロードしてインストールしてください。

## トラブルシューティング

### 拡張機能がインストールされない

- VS Code のバージョンが 1.85.0 以降であることを確認してください。
- VS Code が拡張機能のインストールを許可しているか確認してください。
- マーケットプレイスのウェブサイトから直接インストールしてみてください。

### Qwen Code が応答しない

- インターネット接続を確認してください。
- 新しい会話を開始して、問題が継続するかどうかを確認してください。
- 問題が解決しない場合は、[GitHub で Issue を報告してください](https://github.com/qwenlm/qwen-code/issues)。
