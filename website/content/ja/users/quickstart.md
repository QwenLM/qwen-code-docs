# クイックスタート

> 👏 Qwen Code へようこそ！

このクイックスタートガイドでは、わずか数分で AI を活用したコーディング支援を開始できます。完了後には、Qwen Code を使用して一般的な開発タスクを実行する方法が理解できるようになります。

## はじめに

以下の準備が整っていることを確認してください。

- **ターミナル**またはコマンドプロンプトが起動している
- 作業対象のコードプロジェクトがある
- [Qwen Code](https://chat.qwen.ai/auth?mode=register) のアカウントがある

## ステップ 1：Qwen Code のインストール

Qwen Code をインストールするには、以下のいずれかの方法を使用します。

### クイックインストール（推奨）

**Linux / macOS**

```sh
curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh | bash
```

**Windows（管理者として実行した CMD）**

```sh
curl -fsSL -o %TEMP%\install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%\install-qwen.bat
```

> [!note]
>
> 環境変数が正しく適用されるよう、インストール後にターミナルを再起動することを推奨します。

### 手動インストール

**前提条件**

Node.js 20 以降がインストールされていることを確認してください。[nodejs.org](https://nodejs.org/ja/download/) からダウンロードできます。

**NPM**

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew（macOS、Linux）**

```bash
brew install qwen-code
```

## ステップ 2：アカウントにログインする

Qwen Code を使用するにはアカウントが必要です。`qwen` コマンドでインタラクティブなセッションを開始すると、ログインを求められます。

```bash

# 初回使用時にログインを促されます
qwen
```

# アカウントでログインするためのプロンプトに従ってください
/auth
```

`Qwen OAuth` を選択し、アカウントにログインして、確認のためのプロンプトに従ってください。ログインが完了すると、認証情報が保存され、以降は再度ログインする必要はありません。

> [!note]
>
> Qwen Code を初めて Qwen アカウントで認証すると、「.qwen」というワークスペースが自動的に作成されます。このワークスペースでは、組織内のすべての Qwen Code 利用にかかるコストを一元管理・追跡できます。

> [!tip]
>
> 再度ログインする必要がある場合や、アカウントを切り替える場合は、Qwen Code 内で `/auth` コマンドを使用してください。

## ステップ 3：最初のセッションを開始する

任意のプロジェクトディレクトリでターミナルを開き、Qwen Code を起動します：

```bash

# （任意）プロジェクトディレクトリに移動
cd /path/to/your/project

# Qwen を起動
qwen
```

Qwen Code のウェルカム画面が表示され、セッション情報、最近の会話、最新の更新情報などが確認できます。利用可能なコマンドを確認するには、`/help` と入力してください。

## Qwen Code とチャットする

### 最初の質問をしてみましょう

Qwen Code は、あなたのファイルを分析して要約を提供します。また、より具体的な質問も可能です。

```
フォルダー構造を説明してください
```

Qwen Code 自身の機能についても質問できます。

```
Qwen Code は何ができますか？
```

> [!note]
>
> Qwen Code は必要に応じてファイルを読み込みます — 手動でコンテキストを追加する必要はありません。また、Qwen Code は自身のドキュメントにもアクセス可能であり、その機能や能力に関する質問にも答えられます。

### 最初のコード変更を行いましょう

次に、Qwen Code に実際にコードを書かせてみましょう。簡単なタスクから始めてみましょう。

```
メインファイルに「Hello, World」関数を追加してください
```

Qwen Code は以下の手順で実行します。

1. 適切なファイルを特定する  
2. 提案される変更内容を表示する  
3. 変更の承認を依頼する  
4. 編集を適用する  

> [!note]
>
> Qwen Code は、ファイルを変更する前に必ず許可を求めるようになっています。個別の変更を承認することも、セッション単位で「すべて承認」モードを有効化することもできます。

### Qwen Code で Git を使用する

Qwen Code では、Git の操作を会話形式で行えます。

```
変更したファイルはどれですか？
```

```
説明的なメッセージで変更をコミットしてください
```

さらに複雑な Git 操作についても指示できます。

```
feature/quickstart という名前の新しいブランチを作成してください
```

```
直近の 5 件のコミットを表示してください
```

```
マージコンフリクトの解決を手伝ってください
```

### バグの修正または機能の追加

Qwen Code はデバッグおよび機能実装に優れています。

自然言語で希望する内容を記述してください。

```
ユーザー登録フォームに入力バリデーションを追加してください
```

あるいは既存の問題を修正してください。

```
ユーザーが空のフォームを送信できてしまうバグがあります — これを修正してください
```

Qwen Code は以下の処理を行います。

- 関連するコードを特定します
- コンテキストを理解します
- 解決策を実装します
- 利用可能な場合はテストを実行します

### その他の一般的なワークフローを試す

Qwen Code を使用する方法はいくつかあります。

**コードのリファクタリング**

```
認証モジュールをコールバックから async/await にリファクタリングする
```

**テストの作成**

```
電卓関数のユニットテストを作成する
```

**ドキュメントの更新**

```
README をインストール手順で更新する
```

**コードレビュー**

```
私の変更をレビューし、改善点を提案する
```

> [!tip]
>
> **覚えておいてください**: Qwen Code はあなたの AI ペアプログラマーです。役立つ同僚と話すように、Qwen Code にも話しかけてください。実現したいことを説明すれば、それに向かってサポートしてくれます。

## 基本コマンド

日常的に使用する最も重要なコマンドを以下に示します：

| コマンド               | 機能                                             | 例                             |
| --------------------- | ------------------------------------------------ | ----------------------------- |
| `qwen`                | Qwen Code を起動                                 | `qwen`                        |
| `/auth`               | 認証方法を変更                                   | `/auth`                       |
| `/help`               | 利用可能なコマンドのヘルプ情報を表示             | `/help` または `/?`           |
| `/compress`           | チャット履歴を要約に置き換えてトークンを節約     | `/compress`                   |
| `/clear`              | ターミナル画面の内容をクリア                     | `/clear`（ショートカット: `Ctrl+L`） |
| `/theme`              | Qwen Code のビジュアルテーマを変更               | `/theme`                      |
| `/language`           | 言語設定を表示または変更                         | `/language`                   |
| → `ui [言語]`         | UI インターフェースの言語を設定                  | `/language ui zh-CN`          |
| → `output [言語]`     | LLM の出力言語を設定                             | `/language output Chinese`    |
| `/quit`               | Qwen Code を即座に終了                           | `/quit` または `/exit`        |

コマンドの完全な一覧については、[CLI リファレンス](./features/commands) を参照してください。

## 初心者向けのプロのヒント

**リクエストは具体的に記述しましょう**

- 次のように書かないでください。「バグを修正してください」
- 代わりに、次のように具体的に記述してください。「ユーザーが誤った認証情報を入力した後に空白画面が表示される、ログインに関するバグを修正してください」

**ステップバイステップの指示を使用しましょう**

- 複雑なタスクは、ステップに分割して記述します：

```
1. ユーザープロフィール用の新しいデータベーステーブルを作成する
2. ユーザープロフィールを取得・更新するための API エンドポイントを作成する
3. ユーザーが自身の情報を閲覧・編集できるウェブページを作成する
```

**Qwen Code にまず探索させましょう**

- 変更を加える前に、Qwen Code にコードを理解させます：

```
データベーススキーマを分析する
```

```
英国の顧客から最も多く返品されている商品を表示するダッシュボードを作成する
```

**ショートカットで時間を節約しましょう**

- `?` を押すと、利用可能なすべてのキーボードショートカットが表示されます
- Tab キーを使ってコマンドの自動補完を行います
- ↑ キーを押すと、実行履歴を参照できます
- `/` を入力すると、すべてのスラッシュコマンドが表示されます
