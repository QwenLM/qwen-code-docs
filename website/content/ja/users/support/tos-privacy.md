# Qwen Code：利用規約およびプライバシーに関する通知

Qwen Code は、Qwen Code チームがメンテナンスするオープンソースの AI コーディングアシスタントツールです。本ドキュメントでは、Qwen Code の認証方法および AI モデルサービスを利用する際に適用される利用規約およびプライバシーポリシーについて説明します。

## 認証方法の確認方法

Qwen Code では、AI モデルにアクセスするための認証方法として、以下の 3 種類をサポートしています。使用する認証方法によって、適用される利用規約およびプライバシーポリシーが異なります。

1. **Qwen OAuth** — qwen.ai アカウントでログイン（無料の日次クォータあり）
2. **Alibaba Cloud Coding Plan** — Alibaba Cloud から取得した API キーを使用
3. **API キー** — 自分で用意した API キーを使用

各認証方法について、基盤となるサービスプロバイダーに応じて、異なる利用規約およびプライバシーに関する通知が適用される場合があります。

| 認証方法                  | プロバイダー        | 利用規約                                                                 | プライバシーに関する通知                                                     |
| :------------------------ | :---------------- | :----------------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI           | [Qwen 利用規約](https://qwen.ai/termsservice)                            | [Qwen プライバシーポリシー](https://qwen.ai/privacypolicy)                   |
| Alibaba Cloud Coding Plan | Alibaba Cloud     | [以下で詳しく説明](#2-if-you-are-using-alibaba-cloud-coding-plan)      | [以下で詳しく説明](#2-if-you-are-using-alibaba-cloud-coding-plan)          |
| API キー                  | さまざまなプロバイダー | 使用する API プロバイダー（OpenAI、Anthropic など）に依存                 | 使用する API プロバイダーに依存                                              |

## 1. Qwen の OAuth 認証を使用している場合

qwen.ai アカウントで認証を行う場合、以下の利用規約およびプライバシーに関する通知が適用されます。

- **利用規約**: お客様の利用は、[Qwen 利用規約](https://qwen.ai/termsservice)によって規定されます。
- **プライバシーに関する通知**: お客様のデータの収集および利用方法については、[Qwen プライバシーポリシー](https://qwen.ai/privacypolicy)をご覧ください。

認証の設定方法、クォータ、およびサポートされている機能の詳細については、「[認証の設定](../configuration/settings)」を参照してください。

## 2. Alibaba Cloud コーディングプランをご利用の場合

Alibaba Cloud の API キーを使用して認証を行う場合、Alibaba Cloud の利用規約およびプライバシーノーティスが適用されます。

Alibaba Cloud コーディングプランは、以下の 2 つのリージョンでご利用いただけます。

- **阿里云百炼 (aliyun.com)** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)  
- **Alibaba Cloud (alibabacloud.com)** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> Alibaba Cloud コーディングプランをご利用の場合は、Alibaba Cloud の利用規約およびプライバシーポリシーが適用されます。データの使用方法、保存期間、およびプライバシーに関する実践内容などの詳細については、Alibaba Cloud のドキュメントをご確認ください。

## 3. 自分で取得した API キーを使用する場合

他のプロバイダーから取得した API キーで認証を行う場合、適用される利用規約およびプライバシー通知は、選択したプロバイダーによって異なります。

> [!important]
>
> 自分で取得した API キーを使用する場合、Qwen Code の利用規約ではなく、選択した API プロバイダーの利用規約およびプライバシーポリシーが適用されます。データの使用方法、保存期間、およびプライバシーに関する具体的な詳細については、プロバイダーのドキュメントをご確認ください。

Qwen Code は、OpenAI 互換のさまざまなプロバイダーをサポートしています。詳細な情報については、ご利用のプロバイダーの利用規約およびプライバシーポリシーをご確認ください。

## 使用状況統計およびテレメトリ

Qwen Code では、ユーザー体験および製品品質の向上を目的として、匿名の使用状況統計および[テレメトリ](../../developers/development/telemetry)データを収集することがあります。このデータ収集は任意であり、設定を通じて制御できます。

### 収集されるデータ

有効化されている場合、Qwen Code は以下のデータを収集する可能性があります。

- 匿名の使用状況統計（実行されたコマンド、パフォーマンス指標）
- エラー報告およびクラッシュデータ
- 機能の使用パターン

### 認証方法別のデータ収集

- **Qwen OAuth:** 使用状況統計は Qwen のプライバシーポリシーに準拠します。Qwen Code の設定からオプトアウトできます。
- **Alibaba Cloud Coding プラン:** 使用状況統計は Alibaba Cloud のプライバシーポリシーに準拠します。Qwen Code の設定からオプトアウトできます。
- **API キー:** Qwen Code が収集する追加のデータはなく、選択した API プロバイダーが収集するデータのみとなります。

## よくある質問（FAQ）

### 1. 私のコード（プロンプトおよび回答を含む）は、AI モデルの学習に使用されますか？

プロンプトおよび回答を含むあなたのコードが AI モデルの学習に使用されるかどうかは、認証方法および利用中の特定の AI サービスプロバイダーによって異なります。

- **Qwen OAuth**: データ利用に関する規定は、[Qwen のプライバシーポリシー](https://qwen.ai/privacy)に基づきます。データ収集およびモデル学習に関する詳細については、当該ポリシーをご確認ください。

- **Alibaba Cloud コーディングプラン**: データ利用に関する規定は、Alibaba Cloud のプライバシーポリシーに基づきます。データ収集およびモデル学習に関する詳細については、当該ポリシーをご確認ください。

- **API キー**: データ利用は、選択した API プロバイダーに完全に依存します。各プロバイダーには独自のデータ利用ポリシーがあります。ご利用中のプロバイダーのプライバシーポリシーおよび利用規約を必ずご確認ください。

**重要**: Qwen Code 自体は、ユーザーのプロンプト、コード、または応答をモデル学習目的で使用しません。学習目的でのデータ利用が発生する場合、それはすべて、ユーザーが認証を行った AI サービスプロバイダーのポリシーに従います。

### 2. 使用状況統計情報とは何か、およびオプトアウト制御とは？

**使用状況統計情報** の設定は、ユーザー体験および製品品質の向上を目的として、Qwen Code が実施する任意のデータ収集を制御します。

この設定が有効な場合、Qwen Code は以下の情報を収集することがあります：

- 匿名のテレメトリ（実行されたコマンド、パフォーマンス指標、機能の使用状況）
- エラー報告およびクラッシュデータ
- 一般的な使用パターン

**Qwen Code が収集しない情報：**

- お客様のコードの内容
- AI モデルに送信したプロンプト
- AI モデルからの応答
- 個人情報

使用状況統計情報の設定は、Qwen Code 自体によるデータ収集のみを制御します。お客様が選択した AI サービスプロバイダー（Qwen、OpenAI など）が、それぞれのプライバシーポリシーに基づいて収集するデータについては、この設定は一切影響しません。

### 3. 認証方法を切り替えるにはどうすればよいですか？

Qwen OAuth、Alibaba Cloud Coding Plan、および独自の API キーの間で、いつでも認証方法を切り替えることができます。

1. **起動時**: プロンプトが表示された際に、希望する認証方法を選択します
2. **CLI 内で**: `/auth` コマンドを使用して、認証方法を再設定します
3. **環境変数**: `.env` ファイルを設定して、API キーによる自動認証を有効にします

詳細な手順については、[認証のセットアップ](../configuration/settings#environment-variables-for-api-access) のドキュメントをご覧ください。
