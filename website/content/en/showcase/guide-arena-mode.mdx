## Overview

For important code, you'd ask a colleague to review it. Now you can have multiple AI models process the same task simultaneously. The Agent Arena lets you compare different models' capabilities — just call `/arena`, select models, input your task, and pick the best result.

## What You Can Do With It

- **Refactor core modules**: Have multiple models propose solutions simultaneously, pick the best one
- **Compare approaches**: Compare different models' solutions when writing algorithms, learn different perspectives
- **Evaluate new models**: Use real tasks to evaluate how new models perform on your project

## Steps

<Steps>

### Enter the /arena command

In the Qwen Code chat, type `/arena` and select `start` to enter arena mode.

```bash
/arena start
```

### Select participating models

From the available model list, select the models you want to compare. Press `space` to select 2 or more models to participate.

### Enter the task description

Enter the task you want each model to handle, for example:

```
Refactor this function to improve readability and performance
```

### Compare results and pick the best

All models will process the task simultaneously. Once done, you can compare their solutions and apply the best result to your code.

```bash
/arena select
```

</Steps>

<Callout type="info">
Arena mode is especially suited for important code decisions, letting multiple AI models provide different perspectives on solutions.
</Callout>
