## First, the Tools: What Are Qwen Code and Bailian CLI?

In simple terms:

- **Qwen Code** is an AI coding assistant, but it does more than write code — it's a general-purpose AI Agent that understands natural language instructions and helps you invoke various tools to complete tasks
- **Bailian CLI** (the `bl` command) is a command-line tool for Alibaba Cloud's Model Studio platform, integrating text-to-image, image-to-image, text-to-video, image-to-video, and more

Put them together, and you have a **marketing asset production line that understands plain language**.

### What You'll Need

- Install Qwen Code CLI: [Qwen Code Overview \| Qwen Code Docs](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)
- Install Bailian CLI: [Alibaba Cloud Bailian CLI \| One Command for All AI Capabilities](https://bailian.console.aliyun.com/cli?source_channel=cn-beijing-cli&&utm_content=se_1023346373)
- An Alibaba Cloud Model Studio API Key (apply at the [Model Studio Console](https://bailian.console.aliyun.com))

## Scenario 2: Snap a Photo in Feishu, Auto-Generate Product Images and Videos 📸

E-commerce product photography is expensive — renting a studio, hiring a photographer, post-processing, the whole package starts at several thousand yuan with a one-to-two-week turnaround. Small teams simply can't afford it. And now every platform is pushing short videos, so product videos are another expense.

The best approach now: **Take a photo with your phone, send it to a Feishu group @bot, and the AI automatically identifies your product, then generates white-background product images, marketing images with text, and a 5-second showcase video — all in one go.**

No computer needed for the entire process.

**Step 1: Take a Photo and Send It to the Group**

Snap a casual photo of the product with your phone (any background — on your desk, on the couch, anywhere), send it to the Feishu group, and @Qwen Code:

```
使用 open1v-product-visual 技能帮我生成2张商品图
```

The bot automatically does 3 things:

1. **Identifies the product** — automatically determines what you photographed
2. **Generates standard product images** — creates images based on the identified product, saved to the project folder
3. **Returns results** — can return cloud storage image links directly

**Step 2: Pick an Image, Add Marketing Text**

Choose a satisfactory white-background image, then say:

```
现在我需要加上营销文字，小红书3:4尺寸，布局你可以自选，我要白底的。
```

The bot extracts marketing copy based on the identified product info and overlays it on the image: main title, subtitle, feature tags, spec data — the finished image is ready to list.

Want to change the text or layout? Just say:

```
把图片整体向右侧移动一点，产品主体居中，文字和排版不变
```

Changing text doesn't require regenerating the image — **2 seconds** to re-render and it's done.

**Step 3: Get the Final Asset Links**

```
现在我需要你生成图片的链接，上传到云空间并返回给我链接，只需要终稿版本
```

## Scenario 3: AI Writes Your Marketing Copy ✍️

Writing copy is one of the most time-consuming tasks. WeChat articles need long-form storytelling, social media needs short punchy posts, detail pages need feature copy — **same product, three channels, three different writing styles**. Starting from scratch every time, eventually running out of things to say, recycling the same phrases over and over.

Now just tell the AI the product info and distribution channels. WeChat gets storytelling, social media gets lifestyle posts, detail pages get feature highlights — one input, multi-channel output.

**Example: Generate copy for a portable power bank across 3 channels**

In Qwen Code, enter:

```
帮我写一篇充电宝的推广文案。
产品名：PowerMax 迷你充电宝
卖点：1. 仅重 120g，比一颗鸡蛋还轻  2. 10000mAh 大容量  3. 支持 65W 快充，30 分钟充满 iPhone
目标人群：经常出差的商务人士
渠道：公众号、小红书、详情页
```
