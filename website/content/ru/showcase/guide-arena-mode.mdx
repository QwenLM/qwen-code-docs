## Обзор

Для важного кода вы бы попросили коллегу сделать ревью. Теперь вы можете заставить несколько моделей ИИ обрабатывать одну и ту же задачу одновременно. Agent Arena позволяет сравнивать возможности разных моделей — просто вызовите `/arena`, выберите модели, введите задачу и выберите лучший результат.

## Что вы можете с этим делать

- **Рефакторинг основных модулей**: Заставить несколько моделей одновременно предложить решения и выбрать лучшее
- **Сравнение подходов**: Сравнить решения разных моделей при написании алгоритмов, изучить разные перспективы
- **Оценка новых моделей**: Использовать реальные задачи для оценки производительности новых моделей в вашем проекте

## Шаги

<Steps>

### Введите команду /arena

В чате Qwen Code введите `/arena` и выберите `start` для входа в режим арены.

```bash
/arena start
```

### Выберите участвующие модели

Из списка доступных моделей выберите те, которые хотите сравнить. Нажмите `space` для выбора 2 или более моделей.

### Введите описание задачи

Введите задачу, которую хотите поручить каждой модели, например:

```
Рефакторинг этой функции для улучшения читаемости и производительности
```

### Сравните результаты и выберите лучший

Все модели обработают задачу одновременно. После завершения вы можете сравнить их решения и применить лучший результат к вашему коду.

```bash
/arena select
```

</Steps>

<Callout type="info">
Режим арены особенно подходит для важных решений по коду, позволяя нескольким моделям ИИ предоставить разные перспективы решений.
</Callout>
