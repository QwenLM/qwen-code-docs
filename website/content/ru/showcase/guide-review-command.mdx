## Обзор

Хотите, чтобы кто-то проверил ваш код перед коммитом, но коллеги слишком заняты? Просто используйте `/review` — ИИ проверит качество кода, найдёт потенциальные проблемы и предложит улучшения. Это не простая проверка lint, а тщательный обзор вашей логики, именования и обработки граничных случаев, как опытный коллега.

## Что вы можете с этим делать

- **Ревью перед коммитом**: Пусть ИИ просмотрит ваш код, обнаружит проблемы заранее — больше никаких сюрпризов на Code Review
- **Второй взгляд при самостоятельной разработке**: Уменьшить простые ошибки, повысить качество кода
- **Изучение лучших практик**: ИИ объясняет, почему определённый подход лучше, а не просто «здесь есть проблема»

## Шаги

<Steps>

### Откройте файл для ревью

В Qwen Code откройте файл кода или директорию проекта, которую хотите проверить. Убедитесь, что код сохранён.

### Введите команду /review

Введите `/review` в чате. Qwen Code автоматически проанализирует текущий файл или недавние изменения кода.

```bash
/review
```

### Просмотрите результаты ревью

ИИ предоставит обратную связь по нескольким измерениям: качество кода, логическая корректность, конвенции именования и обработка граничных случаев, а также конкретные предложения по улучшению.

</Steps>

<Callout type="info">
`/review` отличается от простых проверок lint — он глубоко понимает логику кода, выявляя потенциальные баги, проблемы производительности и недостатки дизайна.
</Callout>
