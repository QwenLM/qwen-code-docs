# Документация Qwen Code

Добро пожаловать в документацию Qwen Code. Qwen Code — это агентное средство разработки, которое работает в вашем терминале и помогает быстрее, чем раньше, превращать идеи в код.

## Разделы документации

### [Руководство пользователя](./users/overview)

Узнайте, как использовать Qwen Code в качестве конечного пользователя. В этом разделе рассматриваются следующие темы:

- Базовая установка и настройка
- Распространенные сценарии использования
- Функции и возможности
- Параметры конфигурации
- Устранение неполадок

### [Руководство разработчика](./developers/contributing)

Узнайте, как внести свой вклад в развитие Qwen Code и участвовать в его разработке. Этот раздел охватывает:

- Обзор архитектуры
- Руководящие принципы участия в проекте
- Основные концепции и детали реализации
- Инструменты и рабочий процесс разработки
- Разработка расширений и плагинов
