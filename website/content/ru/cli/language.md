# Команда /language

Команда `/language` позволяет настроить языковые параметры как для пользовательского интерфейса (UI) Qwen Code, так и для языка вывода языковой модели. Эта команда поддерживает две отдельные функции:

1. Установка языка UI для интерфейса Qwen Code
2. Установка языка вывода для языковой модели (LLM)

## Настройки языка UI

Чтобы изменить язык пользовательского интерфейса Qwen Code, используйте подкоманду `ui`:

```
/language ui [zh-CN|en-US]
```

### Доступные языки UI

- **zh-CN**: Упрощенный китайский (简体中文)
- **en-US**: Английский

### Примеры

```
/language ui zh-CN    # Установить язык UI на упрощенный китайский
/language ui en-US    # Установить язык UI на английский
```

### Подкоманды языка UI

Для удобства вы также можете использовать прямые подкоманды:

- `/language ui zh-CN` или `/language ui zh` или `/language ui 中文`
- `/language ui en-US` или `/language ui en` или `/language ui english`

## Настройки языка вывода LLM

Чтобы задать язык ответов языковой модели, используйте подкоманду `output`:

```
/language output <language>
```

Эта команда генерирует файл правила языка, который указывает LLM отвечать на заданном языке. Файл правила сохраняется в `~/.qwen/output-language.md`.

### Примеры

```
/language output 中文      # Установить китайский язык вывода LLM
/language output English   # Установить английский язык вывода LLM
/language output 日本語    # Установить японский язык вывода LLM
```

## Просмотр текущих настроек

При использовании без аргументов команда `/language` отображает текущие языковые настройки:

```
/language
```

Будет показано:

- Текущий язык интерфейса
- Текущий язык вывода LLM (если установлен)
- Доступные подкоманды

## Примечания

- Изменения языка интерфейса применяются сразу же и перезагружают все описания команд
- Настройки языка вывода LLM сохраняются в файле правил, который автоматически добавляется в контекст модели
- Чтобы запросить дополнительные языковые пакеты для интерфейса, откройте issue на GitHub
