# Инструмент Web Fetch (`web_fetch`)

В этом документе описывается инструмент `web_fetch` для Qwen Code.

## Описание

Используйте `web_fetch`, чтобы получить содержимое по указанному URL и обработать его с помощью AI-модели. На вход инструмент принимает URL и промпт, загружает содержимое страницы, конвертирует HTML в Markdown и обрабатывает данные с помощью промпта, используя легковесную и быструю модель.

### Аргументы

`web_fetch` принимает два аргумента:

- `url` (string, required): URL, с которого нужно получить содержимое. Должен быть корректным полным URL, начинающимся с `http://` или `https://`.
- `prompt` (string, required): Промпт, описывающий, какую информацию необходимо извлечь из содержимого страницы.

## Как использовать `web_fetch` с Qwen Code

Чтобы использовать `web_fetch` с Qwen Code, укажите URL и промпт, описывающий, какую информацию нужно извлечь. Перед загрузкой инструмент запросит подтверждение. После подтверждения он напрямую загрузит содержимое и обработает его с помощью AI-модели.

Инструмент автоматически конвертирует HTML в текст, обрабатывает blob-URL GitHub (преобразуя их в raw-URL) и для безопасности автоматически заменяет HTTP на HTTPS.

Использование:

```
web_fetch(url="https://example.com", prompt="Summarize the main points of this article")
```

## Примеры использования `web_fetch`

Краткое содержание одной статьи:

```
web_fetch(url="https://example.com/news/latest", prompt="Can you summarize the main points of this article?")
```

Извлечение конкретной информации:

```
web_fetch(url="https://arxiv.org/abs/2401.0001", prompt="What are the key findings and methodology described in this paper?")
```

Анализ документации GitHub:

```
web_fetch(url="https://github.com/QwenLM/Qwen/blob/main/README.md", prompt="What are the installation steps and main features?")
```

## Важные примечания

- **Обработка одного URL:** `web_fetch` обрабатывает только один URL за раз. Для анализа нескольких URL выполняйте отдельные вызовы инструмента.
- **Формат URL:** Инструмент автоматически заменяет HTTP на HTTPS и преобразует blob-URL GitHub в raw-формат для более удобного доступа к содержимому.
- **Обработка содержимого:** Инструмент напрямую загружает содержимое и обрабатывает его с помощью AI-модели, конвертируя HTML в читаемый текстовый формат.
- **Качество результата:** Качество вывода зависит от четкости инструкций в промпте.
- **Инструменты MCP:** Если доступен инструмент для загрузки веб-страниц от MCP (начинается с `mcp__`), рекомендуется использовать его, так как он может иметь меньше ограничений.
