# Статус реализации подсказок (Prompt Suggestion)

> Отслеживает статус реализации функции prompt suggestion (NES) во всех пакетах.

## Основной модуль (`packages/core/src/followup/`)

| Компонент                  | Статус      | Строки | Описание                                                              |
| -------------------------- | ----------- | ------ | --------------------------------------------------------------------- |
| `followupState.ts`         | ✅ Готово   | ~230   | Фреймворк-независимый контроллер с таймером/debounce                  |
| `suggestionGenerator.ts`   | ✅ Готово   | ~260   | Генерация LLM + 12 правил фильтрации + поддержка разветвлённых запросов |
| `forkedQuery.ts`           | ✅ Готово   | ~240   | CacheSafeParams + createForkedChat + runForkedQuery                   |
| `overlayFs.ts`             | ✅ Готово   | ~140   | Copy-on-write оверлейная файловая система                             |
| `speculationToolGate.ts`   | ✅ Готово   | ~150   | Контроль границ инструментов с AST-парсером shell                     |
| `speculation.ts`           | ✅ Готово   | ~540   | Движок спекуляции с конвейерной подсказкой + переопределение модели   |
