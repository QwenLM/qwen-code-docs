# Qwen Code: Условия использования и Политика конфиденциальности

Qwen Code — это инструмент для программирования с поддержкой ИИ с открытым исходным кодом, разрабатываемый командой Qwen Code. В этом документе описаны условия использования и политики конфиденциальности, которые применяются при использовании методов аутентификации и сервисов ИИ-моделей Qwen Code.

## Как определить ваш метод аутентификации

Qwen Code поддерживает три метода аутентификации для доступа к ИИ-моделям. Выбранный метод определяет, какие условия использования и политики конфиденциальности применяются к вашему аккаунту:

1. **Qwen OAuth** — Вход через аккаунт qwen.ai (бесплатный тариф отключен 15.04.2026)
2. **Alibaba Cloud Coding Plan** — Использование API-ключа от Alibaba Cloud
3. **API Key** — Использование собственного API-ключа

Для каждого метода аутентификации могут применяться разные Условия использования и Политики конфиденциальности в зависимости от провайдера сервиса.

| Метод аутентификации      | Провайдер         | Условия использования                                              | Политика конфиденциальности                                        |
| :------------------------ | :---------------- | :----------------------------------------------------------------- | :----------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI           | [Условия использования Qwen](https://qwen.ai/termsservice)         | [Политика конфиденциальности Qwen](https://qwen.ai/privacypolicy)  |
| Alibaba Cloud Coding Plan | Alibaba Cloud     | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) |
| API Key                   | Различные провайдеры | Зависит от выбранного вами API-провайдера (OpenAI, Anthropic и др.) | Зависит от выбранного вами API-провайдера                          |

## 1. Если вы используете аутентификацию Qwen OAuth

При аутентификации через аккаунт qwen.ai применяются следующие документы:

- **Условия использования:** Ваше использование регулируется [Условиями использования Qwen](https://qwen.ai/termsservice).
- **Политика конфиденциальности:** Сбор и использование ваших данных описаны в [Политике конфиденциальности Qwen](https://qwen.ai/privacypolicy).

Подробности о настройке аутентификации, квотах и поддерживаемых функциях см. в разделе [Настройка аутентификации](../configuration/settings).

## 2. Если вы используете Alibaba Cloud Coding Plan

При аутентификации с помощью API-ключа от Alibaba Cloud применяются соответствующие Условия использования и Политика конфиденциальности Alibaba Cloud.

Alibaba Cloud Coding Plan доступен в двух регионах:

- **阿里云百炼 (aliyun.com)** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)
- **Alibaba Cloud (alibabacloud.com)** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> При использовании Alibaba Cloud Coding Plan вы соглашаетесь с условиями и политикой конфиденциальности Alibaba Cloud. Пожалуйста, ознакомьтесь с их документацией для получения конкретных сведений об использовании данных, их хранении и практиках обеспечения конфиденциальности.

## 3. Если вы используете собственный API-ключ

При аутентификации с помощью API-ключей от других провайдеров применимые Условия использования и Политика конфиденциальности зависят от выбранного вами провайдера.

> [!important]
>
> При использовании собственного API-ключа вы подчиняетесь условиям и политике конфиденциальности выбранного вами API-провайдера, а не условиям Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего провайдера для получения конкретных сведений об использовании данных, их хранении и практиках обеспечения конфиденциальности.

Qwen Code поддерживает различных провайдеров, совместимых с OpenAI. Для получения подробной информации обратитесь к условиям использования и политике конфиденциальности вашего конкретного провайдера.

## Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные [телеметрии](../../developers/development/telemetry) для улучшения пользовательского опыта и качества продукта. Сбор этих данных является опциональным и может управляться через настройки конфигурации.

### Какие данные собираются

При включении этой функции Qwen Code может собирать:

- Анонимную статистику использования (выполненные команды, метрики производительности)
- Отчеты об ошибках и данные о сбоях
- Паттерны использования функций

### Сбор данных в зависимости от метода аутентификации

- **Qwen OAuth:** Статистика использования регулируется политикой конфиденциальности Qwen. Вы можете отказаться от сбора данных в настройках конфигурации Qwen Code.
- **Alibaba Cloud Coding Plan:** Статистика использования регулируется политикой конфиденциальности Alibaba Cloud. Вы можете отказаться от сбора данных в настройках конфигурации Qwen Code.
- **API Key:** Qwen Code не собирает дополнительных данных сверх тех, что собирает выбранный вами API-провайдер.

## Часто задаваемые вопросы (FAQ)

### 1. Используется ли мой код, включая промпты и ответы, для обучения ИИ-моделей?

Использование вашего кода, включая промпты и ответы, для обучения ИИ-моделей зависит от вашего метода аутентификации и конкретного провайдера ИИ-сервиса:

- **Qwen OAuth**: Использование данных регулируется [Политикой конфиденциальности Qwen](https://qwen.ai/privacy). Пожалуйста, ознакомьтесь с их политикой для получения конкретных сведений о сборе данных и практиках обучения моделей.

- **Alibaba Cloud Coding Plan**: Использование данных регулируется политикой конфиденциальности Alibaba Cloud. Пожалуйста, ознакомьтесь с их политикой для получения конкретных сведений о сборе данных и практиках обучения моделей.

- **API Key**: Использование данных полностью зависит от выбранного вами API-провайдера. У каждого провайдера есть собственные политики использования данных. Пожалуйста, ознакомьтесь с политикой конфиденциальности и условиями использования вашего конкретного провайдера.

**Важно**: Сам Qwen Code не использует ваши промпты, код или ответы для обучения моделей. Любое использование данных в целях обучения регулируется политиками провайдера ИИ-сервиса, через который вы проходите аутентификацию.

### 2. Что такое статистика использования и что контролирует отключение сбора данных?

Настройка **Usage Statistics** управляет опциональным сбором данных Qwen Code для улучшения пользовательского опыта и качества продукта.

При включении Qwen Code может собирать:

- Анонимную телеметрию (выполненные команды, метрики производительности, использование функций)
- Отчеты об ошибках и данные о сбоях
- Общие паттерны использования

**Что НЕ собирает Qwen Code:**

- Содержимое вашего кода
- Промпты, отправляемые ИИ-моделям
- Ответы от ИИ-моделей
- Персональную информацию

Настройка Usage Statistics управляет только сбором данных самим Qwen Code. Она не влияет на то, какие данные может собирать выбранный вами провайдер ИИ-сервиса (Qwen, OpenAI и др.) в соответствии с их собственными политиками конфиденциальности.

### 3. Как переключаться между методами аутентификации?

Вы можете в любой момент переключаться между Qwen OAuth, Alibaba Cloud Coding Plan и собственным API-ключом:

1. **При запуске**: Выберите предпочтительный метод аутентификации при появлении запроса
2. **В CLI**: Используйте команду `/auth` для повторной настройки метода аутентификации
3. **Переменные окружения**: Настройте файлы `.env` для автоматической аутентификации по API-ключу

Подробные инструкции см. в документации по [Настройке аутентификации](../configuration/settings#environment-variables-for-api-access).
