# Qwen Code: Условия использования и политика конфиденциальности

Qwen Code — это инструмент с открытым исходным кодом для помощи в программировании с использованием ИИ, который поддерживается командой Qwen Code. В этом документе изложены условия использования и политики конфиденциальности, применимые при использовании методов аутентификации и сервисов ИИ-модели Qwen Code.

## Как определить ваш метод аутентификации

Qwen Code поддерживает два основных метода аутентификации для доступа к моделям ИИ. Ваш метод аутентификации определяет, какие условия использования и политики конфиденциальности применяются к вашему использованию:

1. **Qwen OAuth** — Вход через учетную запись qwen.ai
2. **Совместимый с OpenAI API** — Использование ключей API от различных поставщиков моделей ИИ

Для каждого метода аутентификации могут применяться различные Условия использования и Уведомления о конфиденциальности в зависимости от базового поставщика услуг.

| Метод аутентификации   | Поставщик         | Условия использования                                                         | Уведомление о конфиденциальности                      |
| :--------------------- | :---------------- | :--------------------------------------------------------------------------- | :-------------------------------------------------- |
| Qwen OAuth             | Qwen AI           | [Условия использования Qwen](https://qwen.ai/termsservice)                   | [Политика конфиденциальности Qwen](https://qwen.ai/privacypolicy) |
| Совместимый с OpenAI API | Разные поставщики | Зависит от выбранного вами поставщика API (OpenAI, Alibaba Cloud, ModelScope и др.) | Зависит от выбранного вами поставщика API           |

## 1. Если вы используете аутентификацию через OAuth Qwen

При аутентификации с помощью вашей учетной записи qwen.ai применяются следующие документы: Условия использования и Уведомление о конфиденциальности:

- **Условия использования:** Ваше использование регулируется [Условиями использования Qwen](https://qwen.ai/termsservice).
- **Уведомление о конфиденциальности:** Сбор и использование ваших данных описаны в [Политике конфиденциальности Qwen](https://qwen.ai/privacypolicy).

Подробности о настройке аутентификации, квотах и поддерживаемых функциях см. в разделе [Настройка аутентификации](../configuration/settings).

## 2. Если вы используете аутентификацию через OpenAI-совместимый API

При аутентификации с использованием ключей API от провайдеров, совместимых с OpenAI, применимые Условия использования и Политика конфиденциальности зависят от выбранного вами провайдера.

> [!important]
>
> При использовании аутентификации через OpenAI-совместимый API вы подчиняетесь условиям и политикам конфиденциальности вашего выбранного провайдера API, а не условиям Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего провайдера для получения конкретной информации об использовании данных, их хранении и практике обеспечения конфиденциальности.

Qwen Code поддерживает различные OpenAI-совместимые провайдеры. Пожалуйста, обратитесь к условиям использования и политике конфиденциальности вашего конкретного провайдера для получения подробной информации.

## Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные [телеметрии](../../developers/development/telemetry) для улучшения пользовательского опыта и качества продукта. Сбор этих данных является необязательным и может контролироваться через настройки конфигурации.

### Какие данные собираются

Если функция включена, Qwen Code может собирать:

- Анонимную статистику использования (выполняемые команды, метрики производительности)
- Отчеты об ошибках и данные о сбоях
- Паттерны использования функций

### Сбор данных в зависимости от метода аутентификации

- **Qwen OAuth:** Статистика использования регулируется политикой конфиденциальности Qwen. Вы можете отказаться от сбора данных через настройки конфигурации Qwen Code.
- **Совместимый с OpenAI API:** Qwen Code не собирает дополнительных данных, кроме тех, что собираются выбранным вами провайдером API.

## Часто задаваемые вопросы (FAQ)

### 1. Используется ли мой код, включая запросы и ответы, для обучения моделей искусственного интеллекта?

Используется ли ваш код, включая запросы и ответы, для обучения моделей искусственного интеллекта, зависит от метода аутентификации и конкретного поставщика услуг искусственного интеллекта, которого вы используете:

- **Qwen OAuth**: Использование данных регулируется [Политикой конфиденциальности Qwen](https://qwen.ai/privacy). Пожалуйста, ознакомьтесь с их политикой для получения конкретной информации о сборе данных и практике обучения моделей.

- **Совместимый с OpenAI API**: Использование данных полностью зависит от выбранного вами поставщика API. У каждого поставщика есть свои собственные политики использования данных. Пожалуйста, ознакомьтесь с политикой конфиденциальности и условиями предоставления услуг вашего конкретного поставщика.

**Важно**: Сам Qwen Code не использует ваши запросы, код или ответы для обучения моделей. Любое использование данных в целях обучения будет регулироваться политиками поставщика услуг искусственного интеллекта, через которого вы проходите аутентификацию.

### 2. Что такое статистика использования и за что отвечает переключатель отказа от сбора данных?

Параметр **Статистика использования** управляет дополнительным сбором данных в Qwen Code с целью улучшения пользовательского опыта и качества продукта.

Если этот параметр включен, Qwen Code может собирать следующие данные:

- Анонимную телеметрию (запускаемые команды, показатели производительности, использование функций)
- Отчеты об ошибках и данные о сбоях
- Общие шаблоны использования

**Что НЕ собирается Qwen Code:**

- Содержимое вашего кода
- Запросы, отправленные моделям ИИ
- Ответы от моделей ИИ
- Персональные данные

Параметр статистики использования управляет только сбором данных самим Qwen Code. Он не влияет на то, какие данные может собирать выбранный вами поставщик услуг ИИ (Qwen, OpenAI и др.) в соответствии со своей политикой конфиденциальности.

### 3. Как переключаться между методами аутентификации?

Вы можете переключаться между аутентификацией через Qwen OAuth и аутентификацией, совместимой с OpenAI API, в любое время:

1. **При запуске**: Выберите предпочитаемый метод аутентификации при появлении запроса
2. **Внутри CLI**: Используйте команду `/auth` для изменения метода аутентификации
3. **Переменные окружения**: Настройте файлы `.env` для автоматической аутентификации через API, совместимый с OpenAI

Подробные инструкции см. в документации по [настройке аутентификации](../configuration/settings#environment-variables-for-api-access).
