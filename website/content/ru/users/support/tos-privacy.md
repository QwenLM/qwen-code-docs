# Qwen Code: Условия использования и уведомление о конфиденциальности

Qwen Code — это инструмент с открытым исходным кодом для помощи в программировании на основе ИИ, поддерживаемый командой Qwen Code. В этом документе изложены условия использования и политика конфиденциальности, применимые при использовании методов аутентификации Qwen Code и сервисов ИИ-моделей.

## Как определить используемый метод аутентификации

Qwen Code поддерживает три метода аутентификации для доступа к ИИ-моделям. Выбранный вами метод аутентификации определяет, какие условия использования и политики конфиденциальности применяются к вашему использованию:

1. **Qwen OAuth** — вход с помощью учётной записи qwen.ai (бесплатная ежедневная квота);
2. **План Alibaba Cloud Coding** — использование ключа API от Alibaba Cloud;
3. **Ключ API** — использование собственного ключа API.

Для каждого метода аутентификации могут применяться разные условия использования и уведомления о конфиденциальности в зависимости от поставщика базовой службы.

| Метод аутентификации      | Поставщик         | Условия использования                                              | Уведомление о конфиденциальности                                   |
| :------------------------ | :---------------- | :----------------------------------------------------------------- | :----------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI           | [Условия использования Qwen](https://qwen.ai/termsservice)         | [Политика конфиденциальности Qwen](https://qwen.ai/privacypolicy)  |
| План Alibaba Cloud Coding | Alibaba Cloud     | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) |
| Ключ API                  | Различные поставщики | Зависит от выбранного вами поставщика API (OpenAI, Anthropic и др.) | Зависит от выбранного вами поставщика API                          |

## 1. Если вы используете аутентификацию Qwen через OAuth

При аутентификации с помощью учётной записи qwen.ai применяются следующие документы: «Условия использования» и «Уведомление о конфиденциальности»:

- **Условия использования:** Использование сервиса регулируется [Условиями использования Qwen](https://qwen.ai/termsservice).
- **Уведомление о конфиденциальности:** Сбор и использование ваших данных описаны в [Политике конфиденциальности Qwen](https://qwen.ai/privacypolicy).

Сведения об установке аутентификации, квотах и поддерживаемых функциях см. в разделе [Настройка аутентификации](../configuration/settings).

## 2. Если вы используете план Alibaba Cloud Coding

При аутентификации с помощью API-ключа от Alibaba Cloud применяются соответствующие Условия обслуживания и Уведомление о конфиденциальности от Alibaba Cloud.

План Alibaba Cloud Coding доступен в двух регионах:

- **Алиюнь Байлянь (aliyun.com)** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)  
- **Alibaba Cloud (alibabacloud.com)** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> При использовании плана Alibaba Cloud Coding вы подпадаете под действие условий и политик конфиденциальности Alibaba Cloud. Ознакомьтесь с их документацией, чтобы узнать подробные сведения об использовании данных, их хранении и практике обеспечения конфиденциальности.

## 3. Если вы используете собственный API-ключ

При аутентификации с помощью API-ключей от других поставщиков действующие Условия обслуживания и Уведомление о конфиденциальности зависят от выбранного вами поставщика.

> [!important]
>
> При использовании собственного API-ключа вы подпадаете под действие условий и политик конфиденциальности выбранного вами поставщика API, а не условий Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего поставщика для получения подробной информации о правилах использования данных, их хранения и практиках обеспечения конфиденциальности.

Qwen Code поддерживает различные поставщики, совместимые с OpenAI. Подробную информацию см. в Условиях обслуживания и Политике конфиденциальности вашего конкретного поставщика.

## Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные [телеметрии](../../developers/development/telemetry) для улучшения пользовательского опыта и качества продукта. Сбор этих данных является необязательным и может управляться через настройки конфигурации.

### Какие данные собираются

При включении Qwen Code может собирать следующие данные:

- Анонимную статистику использования (выполняемые команды, метрики производительности);
- Отчёты об ошибках и данные о сбоях;
- Шаблоны использования функций.

### Сбор данных в зависимости от метода аутентификации

- **OAuth Qwen:** Статистика использования регулируется политикой конфиденциальности Qwen. Отслеживание можно отключить в настройках конфигурации Qwen Code.
- **План Alibaba Cloud Coding:** Статистика использования регулируется политикой конфиденциальности Alibaba Cloud. Отслеживание можно отключить в настройках конфигурации Qwen Code.
- **API-ключ:** Qwen Code не собирает дополнительные данные помимо тех, которые собирает выбранный вами поставщик API.

## Часто задаваемые вопросы (FAQ)

### 1. Используется ли мой код, включая промпты и ответы, для обучения моделей ИИ?

То, используется ли ваш код, включая промпты и ответы, для обучения моделей ИИ, зависит от выбранного способа аутентификации и конкретного поставщика ИИ-сервиса:

- **OAuth Qwen**: Использование данных регулируется [Политикой конфиденциальности Qwen](https://qwen.ai/privacy). Подробные сведения о сборе данных и практиках обучения моделей см. в их политике.

- **План «Кодинг» Alibaba Cloud**: Использование данных регулируется политикой конфиденциальности Alibaba Cloud. Подробные сведения о сборе данных и практиках обучения моделей см. в их политике.

- **API-ключ**: Использование данных полностью зависит от выбранного вами поставщика API. У каждого поставщика свои политики использования данных. Ознакомьтесь с политикой конфиденциальности и условиями обслуживания вашего конкретного поставщика.

**Важно**: Сам Qwen Code не использует ваши промпты, код или ответы для обучения моделей. Любое использование данных в целях обучения регулируется политиками поставщика ИИ-сервиса, с которым вы прошли аутентификацию.

### 2. Что такое статистика использования и как работает параметр отказа от участия?

Параметр **«Статистика использования»** управляет необязательным сбором данных Qwen Code для улучшения пользовательского опыта и качества продукта.

При включении Qwen Code может собирать:

- Анонимную телеметрию (выполняемые команды, метрики производительности, использование функций)
- Отчёты об ошибках и данные о сбоях
- Общие шаблоны использования

**Что Qwen Code НЕ собирает:**

- Содержимое вашего кода
- Запросы, отправляемые моделям ИИ
- Ответы от моделей ИИ
- Персональные данные

Параметр «Статистика использования» регулирует только сбор данных самим Qwen Code. Он не влияет на то, какие данные может собирать выбранный вами поставщик ИИ-сервисов (Qwen, OpenAI и др.) в соответствии с их собственной политикой конфиденциальности.

### 3. Как переключаться между методами аутентификации?

Вы можете переключаться между Qwen OAuth, тарифным планом Alibaba Cloud Coding и собственным API-ключом в любое время:

1. **При запуске**: выберите предпочитаемый метод аутентификации при появлении соответствующего запроса  
2. **В CLI**: используйте команду `/auth`, чтобы изменить настройки метода аутентификации  
3. **Переменные окружения**: настройте файлы `.env` для автоматической аутентификации с помощью API-ключа  

Подробные инструкции см. в документации по [настройке аутентификации](../configuration/settings#environment-variables-for-api-access).
