# Qwen Code: Условия использования и уведомление о конфиденциальности

Qwen Code — это открытый инструмент-помощник для написания кода с использованием ИИ, поддерживаемый командой Qwen Code. Этот документ описывает условия использования и политики конфиденциальности, которые применяются при использовании методов аутентификации Qwen Code и сервисов моделей ИИ.

## Как определить ваш метод аутентификации

Qwen Code поддерживает четыре метода аутентификации для доступа к моделям ИИ. Ваш метод аутентификации определяет, какие условия использования и политики конфиденциальности применяются к вашему использованию:

1. **Qwen OAuth** — Вход с учётной записью qwen.ai (бесплатный тариф прекращён 2026-04-15)
2. **Alibaba Cloud Coding Plan** — Использование API-ключа от Alibaba Cloud
3. **API Key** — Использование собственного API-ключа
4. **Vertex AI** — Использование Google Cloud Vertex AI

Для каждого метода аутентификации могут применяться различные условия использования и уведомления о конфиденциальности в зависимости от поставщика услуг.

| Метод аутентификации     | Поставщик          | Условия использования                                                   | Уведомление о конфиденциальности                                                     |
| :------------------------ | :---------------- | :----------------------------------------------------------------- | :----------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI           | [Условия использования Qwen](https://qwen.ai/termsservice)              | [Политика конфиденциальности Qwen](https://qwen.ai/privacypolicy)               |
| Alibaba Cloud Coding Plan | Alibaba Cloud     | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) |
| API Key                   | Различные поставщики | Зависит от выбранного вами поставщика API (OpenAI, Anthropic и др.)      | Зависит от выбранного вами поставщика API                                |
| Vertex AI                 | Google Cloud      | [Условия использования Google Cloud](https://cloud.google.com/terms)               | [Конфиденциальность Google Cloud](https://cloud.google.com/privacy)           |

## 1. Если вы используете аутентификацию Qwen OAuth

При аутентификации с помощью учётной записи qwen.ai применяются следующие документы об условиях использования и конфиденциальности:

- **Условия использования:** Ваше использование регулируется [Условиями использования Qwen](https://qwen.ai/termsservice).
- **Уведомление о конфиденциальности:** Сбор и использование ваших данных описаны в [Политике конфиденциальности Qwen](https://qwen.ai/privacypolicy).

Подробнее о настройке аутентификации, квотах и поддерживаемых функциях см. в разделе [Настройка аутентификации](../configuration/settings).

## 2. Если вы используете Alibaba Cloud Coding Plan

При аутентификации с помощью API-ключа от Alibaba Cloud применяются соответствующие условия использования и уведомление о конфиденциальности от Alibaba Cloud.

Alibaba Cloud Coding Plan доступен в двух регионах:

- **阿里云百炼 (aliyun.com)** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)
- **Alibaba Cloud (alibabacloud.com)** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> При использовании Alibaba Cloud Coding Plan вы соглашаетесь с условиями и политиками конфиденциальности Alibaba Cloud. Пожалуйста, ознакомьтесь с их документацией для получения конкретных сведений об использовании данных, их хранении и методах обеспечения конфиденциальности.

## 3. Если вы используете собственный API-ключ

При аутентификации с помощью API-ключей от других поставщиков применимые условия использования и уведомление о конфиденциальности зависят от выбранного вами поставщика.

> [!important]
>
> При использовании собственного API-ключа вы соглашаетесь с условиями и политиками конфиденциальности выбранного вами поставщика API, а не с условиями Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего поставщика для получения конкретных сведений об использовании данных, их хранении и методах обеспечения конфиденциальности.

Qwen Code поддерживает различных поставщиков, совместимых с OpenAI. Пожалуйста, обратитесь к условиям использования и политике конфиденциальности вашего конкретного поставщика для получения подробной информации.

## 4. Если вы используете Vertex AI

При аутентификации с помощью Google Cloud Vertex AI применяются соответствующие условия использования и уведомление о конфиденциальности Google Cloud.

> [!important]
>
> При использовании Vertex AI вы соглашаетесь с [Условиями использования Google Cloud](https://cloud.google.com/terms) и [Уведомлением о конфиденциальности Google Cloud](https://cloud.google.com/privacy), а не с условиями Qwen Code. Пожалуйста, ознакомьтесь с документацией Google Cloud для получения конкретных сведений об использовании данных, их хранении и методах обеспечения конфиденциальности.

## Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные [телеметрии](../../developers/development/telemetry) для улучшения пользовательского опыта и качества продукта. Этот сбор данных является необязательным и может быть настроен через параметры конфигурации.

### Какие данные собираются

При включении Qwen Code может собирать:

- Анонимную статистику использования (выполненные команды, метрики производительности)
- Отчёты об ошибках и данные о сбоях
- Паттерны использования функций

### Сбор данных по методу аутентификации

- **Qwen OAuth:** Статистика использования регулируется политикой конфиденциальности Qwen. Вы можете отказаться от сбора через настройки конфигурации Qwen Code.
- **Alibaba Cloud Coding Plan:** Статистика использования регулируется политикой конфиденциальности Alibaba Cloud. Вы можете отказаться от сбора через настройки конфигурации Qwen Code.
- **API Key:** Qwen Code не собирает дополнительных данных помимо тех, что собирает выбранный вами поставщик API.
- **Vertex AI:** Статистика использования регулируется политикой конфиденциальности Google Cloud. Qwen Code не собирает дополнительных данных помимо тех, что собирает Google Cloud.

## Часто задаваемые вопросы (FAQ)

### 1. Используется ли мой код, включая запросы и ответы, для обучения моделей ИИ?

Использование вашего кода, включая запросы и ответы, для обучения моделей ИИ зависит от вашего метода аутентификации и конкретного поставщика услуг ИИ:

- **Qwen OAuth:** Использование данных регулируется [Политикой конфиденциальности Qwen](https://qwen.ai/privacypolicy). Пожалуйста, обратитесь к их политике для получения конкретных сведений о сборе данных и практике обучения моделей.

- **Alibaba Cloud Coding Plan:** Использование данных регулируется политикой конфиденциальности Alibaba Cloud. Пожалуйста, обратитесь к их политике для получения конкретных сведений о сборе данных и практике обучения моделей.

- **API Key:** Использование данных полностью зависит от выбранного вами поставщика API. У каждого поставщика свои политики использования данных. Пожалуйста, ознакомьтесь с политикой конфиденциальности и условиями использования вашего конкретного поставщика.

- **Vertex AI:** Использование данных регулируется [Условиями использования Google Cloud](https://cloud.google.com/terms) и [Уведомлением о конфиденциальности](https://cloud.google.com/privacy). Пожалуйста, ознакомьтесь с политиками Google Cloud для получения конкретных сведений о сборе данных и практике обучения моделей.

**Важно:** Сам Qwen Code не использует ваши запросы, код или ответы для обучения моделей. Любое использование данных для целей обучения будет регулироваться политиками поставщика услуг ИИ, с которым вы проходите аутентификацию.

### 2. Что такое статистика использования и что контролирует отказ от сбора?

Настройка **Статистика использования** управляет необязательным сбором данных Qwen Code для улучшения пользовательского опыта и качества продукта.

При включении Qwen Code может собирать:

- Анонимную телеметрию (выполненные команды, метрики производительности, использование функций)
- Отчёты об ошибках и данные о сбоях
- Общие паттерны использования

**Что НЕ собирается Qwen Code:**

- Содержимое вашего кода
- Запросы, отправленные моделям ИИ
- Ответы от моделей ИИ
- Личная информация

Настройка «Статистика использования» управляет только сбором данных самим Qwen Code. Она не влияет на то, какие данные может собирать выбранный вами поставщик услуг ИИ (Qwen, OpenAI и др.) в соответствии с их собственными политиками конфиденциальности.

### 3. Как переключаться между методами аутентификации?

Вы можете переключаться между Qwen OAuth, Alibaba Cloud Coding Plan, собственным API-ключом и Vertex AI в любое время:

1. **При запуске:** Выберите предпочтительный метод аутентификации при появлении запроса
2. **В CLI:** Используйте команду `/auth` для перенастройки метода аутентификации
3. **Переменные окружения:** Настройте файлы `.env` для автоматической аутентификации по API-ключу

Подробные инструкции см. в документации [Настройка аутентификации](../configuration/auth.md).
