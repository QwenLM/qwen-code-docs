# Qwen Code: Условия использования и уведомление о конфиденциальности

Qwen Code — это инструмент AI-ассистента для написания кода с открытым исходным кодом, поддерживаемый командой Qwen Code. Данный документ описывает условия использования и политики конфиденциальности, которые применяются при использовании методов аутентификации и сервисов AI-моделей Qwen Code.

## Как определить ваш метод аутентификации

Qwen Code поддерживает три метода аутентификации для доступа к AI-моделям. Ваш метод аутентификации определяет, какие условия использования и политики конфиденциальности применяются к вашему использованию:

1. **Qwen OAuth** — Вход с помощью учётной записи qwen.ai (бесплатный тариф прекращён 2026-04-15)
2. **Alibaba Cloud Coding Plan** — Использование API-ключа от Alibaba Cloud
3. **API Key** — Использование собственного API-ключа

Для каждого метода аутентификации могут применяться разные условия использования и уведомления о конфиденциальности в зависимости от поставщика услуг.

| Метод аутентификации      | Провайдер          | Условия использования                                             | Уведомление о конфиденциальности                                     |
| :------------------------ | :---------------- | :----------------------------------------------------------------- | :----------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI           | [Qwen Terms of Service](https://qwen.ai/termsservice)              | [Qwen Privacy Policy](https://qwen.ai/privacypolicy)               |
| Alibaba Cloud Coding Plan | Alibaba Cloud     | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) | См. [подробности ниже](#2-if-you-are-using-alibaba-cloud-coding-plan) |
| API Key                   | Различные провайдеры | Зависит от выбранного поставщика API (OpenAI, Anthropic и т.д.)      | Зависит от выбранного поставщика API                                |

## 1. Если вы используете аутентификацию Qwen OAuth

Когда вы проходите аутентификацию с помощью учётной записи qwen.ai, применяются следующие условия использования и уведомление о конфиденциальности:

- **Условия использования:** Ваше использование регулируется [Qwen Terms of Service](https://qwen.ai/termsservice).
- **Уведомление о конфиденциальности:** Сбор и использование ваших данных описаны в [Qwen Privacy Policy](https://qwen.ai/privacypolicy).

Подробнее о настройке аутентификации, квотах и поддерживаемых функциях см. в разделе [Настройка аутентификации](../configuration/settings).

## 2. Если вы используете Alibaba Cloud Coding Plan

Когда вы проходите аутентификацию с помощью API-ключа от Alibaba Cloud, применяются соответствующие условия использования и уведомление о конфиденциальности от Alibaba Cloud.

Alibaba Cloud Coding Plan доступен в двух регионах:

- **阿里云百炼 (aliyun.com)** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)
- **Alibaba Cloud (alibabacloud.com)** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> При использовании Alibaba Cloud Coding Plan на вас распространяются условия и политики конфиденциальности Alibaba Cloud. Пожалуйста, ознакомьтесь с их документацией для получения подробной информации об использовании данных, их хранении и методах обеспечения конфиденциальности.

## 3. Если вы используете собственный API-ключ

Когда вы проходите аутентификацию с помощью API-ключей от других поставщиков, применимые условия использования и уведомление о конфиденциальности зависят от выбранного вами поставщика.

> [!important]
>
> При использовании собственного API-ключа на вас распространяются условия и политики конфиденциальности выбранного поставщика API, а не условия Qwen Code. Пожалуйста, ознакомьтесь с документацией вашего поставщика для получения подробной информации об использовании данных, их хранении и методах обеспечения конфиденциальности.

Qwen Code поддерживает различных поставщиков, совместимых с OpenAI. Пожалуйста, обратитесь к условиям использования и политике конфиденциальности вашего конкретного поставщика для получения подробной информации.

## Статистика использования и телеметрия

Qwen Code может собирать анонимную статистику использования и данные [телеметрии](../../developers/development/telemetry) для улучшения пользовательского опыта и качества продукта. Этот сбор данных является необязательным и может контролироваться через настройки конфигурации.

### Какие данные собираются

При включённой опции Qwen Code может собирать:

- Анонимную статистику использования (выполняемые команды, показатели производительности)
- Отчёты об ошибках и данные о сбоях
- Шаблоны использования функций

### Сбор данных по методу аутентификации

- **Qwen OAuth:** Статистика использования регулируется политикой конфиденциальности Qwen. Вы можете отказаться от сбора через настройки конфигурации Qwen Code.
- **Alibaba Cloud Coding Plan:** Статистика использования регулируется политикой конфиденциальности Alibaba Cloud. Вы можете отказаться от сбора через настройки конфигурации Qwen Code.
- **API Key:** Qwen Code не собирает дополнительных данных, помимо тех, что собирает выбранный вами поставщик API.

## Часто задаваемые вопросы (FAQ)

### 1. Используется ли мой код, включая промпты и ответы, для обучения AI-моделей?

Использование вашего кода, включая промпты и ответы, для обучения AI-моделей зависит от вашего метода аутентификации и конкретного поставщика AI-услуг:

- **Qwen OAuth**: Использование данных регулируется [Политикой конфиденциальности Qwen](https://qwen.ai/privacy). Пожалуйста, обратитесь к их политике для получения подробной информации о сборе данных и практиках обучения моделей.

- **Alibaba Cloud Coding Plan**: Использование данных регулируется политикой конфиденциальности Alibaba Cloud. Пожалуйста, обратитесь к их политике для получения подробной информации о сборе данных и практиках обучения моделей.
- **Ключ API**: Использование данных полностью зависит от выбранного вами поставщика API. У каждого поставщика свои политики использования данных. Пожалуйста, ознакомьтесь с политикой конфиденциальности и условиями предоставления услуг конкретного поставщика.

**Важно**: Qwen Code сам по себе не использует ваши запросы, код или ответы для обучения модели. Любое использование данных для целей обучения регулируется политиками поставщика AI-услуг, у которого вы проходите аутентификацию.

### 2. Что такое статистика использования и что контролирует настройка отказа от её сбора?

Настройка **«Статистика использования»** управляет дополнительным сбором данных со стороны Qwen Code для улучшения пользовательского опыта и качества продукта.

При включении Qwen Code может собирать:

- Анонимную телеметрию (выполненные команды, показатели производительности, использование функций)
- Отчёты об ошибках и данные о сбоях
- Общие шаблоны использования

**Что НЕ собирается Qwen Code:**

- Содержимое вашего кода
- Запросы, отправляемые AI-моделям
- Ответы AI-моделей
- Личную информацию

Настройка «Статистика использования» управляет только сбором данных самим Qwen Code. Она не влияет на то, какие данные может собирать выбранный вами поставщик AI-услуг (Qwen, OpenAI и т. д.) в соответствии с его собственной политикой конфиденциальности.

### 3. Как переключаться между способами аутентификации?

Вы можете в любой момент переключаться между Qwen OAuth, тарифным планом Alibaba Cloud Coding и собственным ключом API:

1. **При запуске**: выберите предпочтительный способ аутентификации при появлении запроса
2. **В CLI**: используйте команду `/auth`, чтобы перенастроить способ аутентификации
3. **Переменные окружения**: настройте файлы `.env` для автоматической аутентификации по ключу API

Подробные инструкции см. в документации [«Настройка аутентификации»](../configuration/auth.md).
