# Горячие клавиши Qwen Code

В этом документе перечислены доступные сочетания клавиш в Qwen Code.

## Общие

| Сочетание клавиш                 | Описание                                                                                                                        |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `Esc`                            | Закрыть диалоговые окна и предложения.                                                                                          |
| `Ctrl+C`                         | Отменить текущий запрос и очистить поле ввода. Нажмите дважды для выхода из приложения.                                         |
| `Ctrl+D`                         | Выйти из приложения, если поле ввода пустое. Нажмите дважды для подтверждения.                                                  |
| `Ctrl+L`                         | Очистить экран.                                                                                                                 |
| `Ctrl+O`                         | Переключить компактный режим (скрыть/показать вывод инструментов и процесс рассуждений).                                        |
| `Ctrl+S`                         | Позволяет выводить длинные ответы полностью, отключая усечение. Используйте прокрутку терминала для просмотра всего вывода.     |
| `Ctrl+T`                         | Переключить отображение описаний инструментов.                                                                                  |
| `Shift+Tab` (`Tab` на Windows)   | Переключать режимы подтверждения (`plan` → `default` → `auto-edit` → `yolo`)                                                    |

## Поле ввода

| Сочетание клавиш                                     | Описание                                                                                                                          |
| ---------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                  | Переключить режим оболочки, когда поле ввода пустое.                                                                              |
| `?`                                                  | Переключить отображение сочетаний клавиш, когда поле ввода пустое.                                                              |
| `\` (в конце строки) + `Enter`                       | Вставить новую строку.                                                                                                            |
| `Down Arrow`                                         | Переместиться вниз по истории ввода.                                                                                              |
| `Enter`                                              | Отправить текущий запрос.                                                                                                         |
| `Meta+Delete` / `Ctrl+Delete`                        | Удалить слово справа от курсора.                                                                                                  |
| `Tab`                                                | Автодополнить текущее предложение, если оно доступно.                                                                             |
| `Up Arrow`                                           | Переместиться вверх по истории ввода.                                                                                             |
| `Ctrl+A` / `Home`                                    | Переместить курсор в начало строки.                                                                                               |
| `Ctrl+B` / `Left Arrow`                              | Переместить курсор на один символ влево.                                                                                          |
| `Ctrl+C`                                             | Очистить поле ввода                                                                                                               |
| `Esc` (двойное нажатие)                              | Очистить поле ввода.                                                                                                              |
| `Ctrl+D` / `Delete`                                  | Удалить символ справа от курсора.                                                                                                 |
| `Ctrl+E` / `End`                                     | Переместить курсор в конец строки.                                                                                                |
| `Ctrl+F` / `Right Arrow`                             | Переместить курсор на один символ вправо.                                                                                         |
| `Ctrl+H` / `Backspace`                               | Удалить символ слева от курсора.                                                                                                  |
| `Ctrl+K`                                             | Удалить текст от курсора до конца строки.                                                                                         |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`     | Переместить курсор на одно слово влево.                                                                                           |
| `Ctrl+N`                                             | Переместиться вниз по истории ввода.                                                                                              |
| `Ctrl+P`                                             | Переместиться вверх по истории ввода.                                                                                             |
| `Ctrl+R`                                             | Обратный поиск по истории ввода/оболочки.                                                                                         |
| `Ctrl+Y`                                             | Повторить последний неудачный запрос.                                                                                             |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F`   | Переместить курсор на одно слово вправо.                                                                                          |
| `Ctrl+U`                                             | Удалить текст от курсора до начала строки.                                                                                        |
| `Ctrl+V` (Windows: `Alt+V`)                          | Вставить содержимое буфера обмена. Если в буфере находится изображение, оно будет сохранено, а в запрос будет вставлена ссылка на него. |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`       | Удалить слово слева от курсора.                                                                                                   |
| `Ctrl+X` / `Meta+Enter`                              | Открыть текущий ввод во внешнем редакторе.                                                                                        |

## Предложения

| Сочетание клавиш  | Описание                               |
| ----------------- | -------------------------------------- |
| `Down Arrow`      | Переместиться вниз по списку предложений. |
| `Tab` / `Enter`   | Принять выбранное предложение.         |
| `Up Arrow`        | Переместиться вверх по списку предложений. |

## Выбор переключателя

| Сочетание клавиш   | Описание                                                                                                  |
| ------------------ | --------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` | Переместить выделение вниз.                                                                               |
| `Enter`            | Подтвердить выбор.                                                                                        |
| `Up Arrow` / `k`   | Переместить выделение вверх.                                                                              |
| `1-9`              | Выбрать элемент по его номеру.                                                                            |
| (многозначное)     | Для элементов с номерами больше 9 быстро введите цифры подряд, чтобы выбрать соответствующий элемент.     |

## Интеграция с IDE

| Сочетание клавиш | Описание                            |
| ---------------- | ----------------------------------- |
| `Ctrl+G`         | Просмотреть контекст CLI, полученный от IDE |
