# Клавиатурные сокращения Qwen Code

Этот документ содержит список доступных клавиатурных сокращений в Qwen Code.

## Общие

| Комбинация клавиш | Описание                                                                                                           |
| ----------------- | ------------------------------------------------------------------------------------------------------------------ |
| `Esc`             | Закрыть диалоги и предложения.                                                                                     |
| `Ctrl+C`          | Отменить текущий запрос и очистить ввод. Нажмите дважды для выхода из приложения.                                  |
| `Ctrl+D`          | Выйти из приложения, если ввод пустой. Нажмите дважды для подтверждения.                                          |
| `Ctrl+L`          | Очистить экран.                                                                                                    |
| `Ctrl+O`          | Переключить отображение консоли отладки.                                                                           |
| `Ctrl+S`          | Позволяет выводу длинных ответов отображаться полностью, отключая усечение. Используйте прокрутку терминала для просмотра всего вывода. |
| `Ctrl+T`          | Переключить отображение описаний инструментов.                                                                     |
| `Shift+Tab`       | Переключение режимов одобрения (`plan` → `default` → `auto-edit` → `yolo`).                                        |

## Входной запрос

| Комбинация клавиш                                  | Описание                                                                                                                            |
| -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                | Переключение режима оболочки при пустом вводе.                                                                                      |
| `?`                                                | Переключение отображения сочетаний клавиш при пустом вводе.                                                                         |
| `\` (в конце строки) + `Enter`                     | Вставить новую строку.                                                                                                              |
| `Стрелка вниз`                                     | Перемещение по истории ввода вниз.                                                                                                 |
| `Enter`                                            | Отправить текущий запрос.                                                                                                           |
| `Meta+Delete` / `Ctrl+Delete`                      | Удалить слово справа от курсора.                                                                                                    |
| `Tab`                                              | Автозаполнение текущего предложения, если оно существует.                                                                           |
| `Стрелка вверх`                                    | Перемещение по истории ввода вверх.                                                                                                |
| `Ctrl+A` / `Home`                                  | Переместить курсор в начало строки.                                                                                                 |
| `Ctrl+B` / `Стрелка влево`                         | Переместить курсор на один символ влево.                                                                                            |
| `Ctrl+C`                                           | Очистить поле ввода                                                                                                                 |
| `Esc` (двойное нажатие)                            | Очистить поле ввода.                                                                                                                |
| `Ctrl+D` / `Delete`                                | Удалить символ справа от курсора.                                                                                                   |
| `Ctrl+E` / `End`                                   | Переместить курсор в конец строки.                                                                                                  |
| `Ctrl+F` / `Стрелка вправо`                        | Переместить курсор на один символ вправо.                                                                                           |
| `Ctrl+H` / `Backspace`                             | Удалить символ слева от курсора.                                                                                                    |
| `Ctrl+K`                                           | Удалить от курсора до конца строки.                                                                                                 |
| `Ctrl+Стрелка влево` / `Meta+Стрелка влево` / `Meta+B` | Переместить курсор на одно слово влево.                                                                                             |
| `Ctrl+N`                                           | Перемещение по истории ввода вниз.                                                                                                 |
| `Ctrl+P`                                           | Перемещение по истории ввода вверх.                                                                                                |
| `Ctrl+R`                                           | Обратный поиск по истории ввода/оболочки.                                                                                          |
| `Ctrl+Стрелка вправо` / `Meta+Стрелка вправо` / `Meta+F` | Переместить курсор на одно слово вправо.                                                                                            |
| `Ctrl+U`                                           | Удалить от курсора до начала строки.                                                                                                |
| `Ctrl+V`                                           | Вставить содержимое буфера обмена. Если буфер содержит изображение, оно будет сохранено, и ссылка на него будет вставлена в запрос. |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | Удалить слово слева от курсора.                                                                                                     |
| `Ctrl+X` / `Meta+Enter`                            | Открыть текущий ввод во внешнем редакторе.                                                                                          |

## Предложения

| Сочетание клавиш | Описание                                        |
| ---------------- | ----------------------------------------------- |
| `Стрелка вниз`   | Перемещение вниз по списку предложений.         |
| `Tab` / `Enter`  | Принять выбранное предложение.                  |
| `Стрелка вверх`  | Перемещение вверх по списку предложений.        |

## Выбор переключателя

| Ярлык              | Описание                                                                                                      |
| ------------------ | ------------------------------------------------------------------------------------------------------------- |
| `Стрелка вниз` / `j` | Переместить выбор вниз.                                                                                       |
| `Ввод`             | Подтвердить выбор.                                                                                            |
| `Стрелка вверх` / `k` | Переместить выбор вверх.                                                                                      |
| `1-9`              | Выбрать элемент по его номеру.                                                                                |
| (многозначный)     | Для элементов с номерами больше 9, нажмите цифры подряд, чтобы выбрать соответствующий элемент.                |

## Интеграция с IDE

| Горячая клавиша | Описание                                 |
| --------------- | ---------------------------------------- |
| `Ctrl+G`        | Просмотреть контекст CLI, полученный от IDE |
