# Горячие клавиши Qwen Code

В этом документе перечислены доступные сочетания клавиш в Qwen Code.

## Общие

| Сочетание клавиш              | Описание                                                                                                                                                                                                                                                                                                   |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Esc`                         | Закрыть диалоги и подсказки.                                                                                                                                                                                                                                                                               |
| `Ctrl+C`                      | Отменить текущий запрос и очистить ввод. Нажмите дважды для выхода из приложения.                                                                                                                                                                                                                          |
| `Ctrl+D`                      | Выйти из приложения, если ввод пуст. Нажмите дважды для подтверждения.                                                                                                                                                                                                                                    |
| `Ctrl+L`                      | Очистить экран.                                                                                                                                                                                                                                                                                            |
| `Ctrl+O`                      | Переключить компактный режим (показать/скрыть вывод инструментов и размышления).                                                                                                                                                                                                                           |
| `Ctrl+S`                      | Разрешить длинным ответам печататься полностью, отключая усечение. Используйте прокрутку терминала для просмотра всего вывода.                                                                                                                                                                             |
| `Ctrl+T`                      | Переключить отображение описаний инструментов.                                                                                                                                                                                                                                                             |
| `Ctrl+B`                      | Если выполняется команда оболочки на переднем плане: повысить её до фоновой задачи. Дочерний процесс продолжает работу, очередь агента разблокируется, и оболочка появляется в `/tasks` + диалоге фоновых задач. Никакого эффекта, если оболочка не выполняется — тогда Ctrl+B срабатывает как привязка к области ввода (курсор влево). |
| `Alt/Option+M`                | Переключить вывод Markdown между богатыми отрендеренными предпросмотрами и сырым/исходным режимом. На macOS терминал должен отправлять Option как Meta.                                                                                                                                                    |
| `Shift+Tab` (`Tab` на Windows) | Циклически переключать режимы одобрения (`plan` → `default` → `auto-edit` → `auto` → `yolo`)                                                                                                                                                                                                              |

## Прокрутка истории

Активна только при включённом `ui.useTerminalBuffer` (Настройки → UI → Virtualized History). В этом режиме история беседы отображается внутри окна приложения, а не в родном буфере прокрутки терминала, поэтому приведённые ниже клавиши заменяют стандартную прокрутку терминала.

| Сочетание клавиш        | Описание                                                      |
| ----------------------- | ------------------------------------------------------------- |
| `Shift+Стрелка вверх`   | Прокрутить историю на одну строку вверх.                     |
| `Shift+Стрелка вниз`    | Прокрутить историю на одну строку вниз.                      |
| `PgUp`                  | Прокрутить историю на одну страницу (высота окна).           |
| `PgDn`                  | Прокрутить историю на одну страницу (высота окна).           |
| `Ctrl+Home`             | Перейти в начало беседы.                                      |
| `Ctrl+End`              | Перейти в конец (и снова включить автоматическое следование). |
| **Колёсико мыши**       | Прокрутка истории (3 строки за щелчок).                      |

Когда `ui.useTerminalBuffer` включён, терминал передаёт события мыши в qwen-code, чтобы колёсико могло управлять окном просмотра. Как побочный эффект, **родное выделение текста перетаскиванием перехватывается программой** — удерживайте `Shift` (или `Option` в macOS Terminal / iTerm) при перетаскивании, чтобы обойти захват мыши и выделить текст обычным способом.

### Прокрутка трекпадом в tmux

Внутри tmux некоторые терминалы преобразуют жесты трекпада или колёсика в обычные последовательности `Стрелка вверх` и `Стрелка вниз` до того, как их увидит qwen-code. Эти байты неотличимы от настоящих нажатий стрелок, поэтому qwen-code не может определить, хотели ли вы прокрутить окно просмотра или перейти по истории команд.

Если прокрутка трекпадом меняет историю команд в tmux, включите `ui.useTerminalBuffer`; затем используйте `Shift+Стрелка вверх` / `Shift+Стрелка вниз` или колёсико мыши, когда tmux передаёт события колёсика в приложение. Если вы предпочитаете родную прокрутку терминала, настройте привязки мыши tmux для событий колёсика.
