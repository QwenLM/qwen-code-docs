# Клавиатурные сокращения Qwen Code

В этом документе перечислены доступные клавиатурные сокращения в Qwen Code.

## Общие

| Сочетание клавиш               | Описание                                                                                                              |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | Закрывает диалоговые окна и подсказки.                                                                                 |
| `Ctrl+C`                       | Отменяет текущий запрос и очищает ввод. Нажмите дважды, чтобы выйти из приложения.                                     |
| `Ctrl+D`                       | Выходит из приложения, если ввод пуст. Нажмите дважды для подтверждения.                                               |
| `Ctrl+L`                       | Очищает экран.                                                                                                        |
| `Ctrl+O`                       | Переключает отображение отладочной консоли.                                                                           |
| `Ctrl+S`                       | Позволяет выводить длинные ответы полностью, отключая их усечение. Используйте буфер прокрутки терминала для просмотра всего вывода. |
| `Ctrl+T`                       | Переключает отображение описаний инструментов.                                                                         |
| `Shift+Tab` (`Tab` в Windows)  | Переключает режимы подтверждения (`plan` → `default` → `auto-edit` → `yolo`)                                         |

## Вводимый запрос

| Сочетание клавиш                                   | Описание                                                                                                                          |
| -------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                | Переключение в режим оболочки при пустом вводе.                                                                                    |
| `?`                                                | Отображение справки по сочетаниям клавиш при пустом вводе.                                                                         |
| `\` (в конце строки) + `Enter`                     | Вставка новой строки.                                                                                                             |
| `Стрелка вниз`                                     | Прокрутка вниз по истории ввода.                                                                                                  |
| `Enter`                                            | Отправка текущего запроса.                                                                                                        |
| `Meta+Delete` / `Ctrl+Delete`                      | Удаление слова справа от курсора.                                                                                                  |
| `Tab`                                              | Автозавершение текущего предложения, если оно доступно.                                                                            |
| `Стрелка вверх`                                    | Прокрутка вверх по истории ввода.                                                                                                  |
| `Ctrl+A` / `Home`                                  | Перемещение курсора в начало строки.                                                                                              |
| `Ctrl+B` / `Стрелка влево`                         | Перемещение курсора на один символ влево.                                                                                          |
| `Ctrl+C`                                           | Очистка поля ввода.                                                                                                               |
| `Esc` (дважды)                                     | Очистка поля ввода.                                                                                                               |
| `Ctrl+D` / `Delete`                                | Удаление символа справа от курсора.                                                                                                |
| `Ctrl+E` / `End`                                   | Перемещение курсора в конец строки.                                                                                               |
| `Ctrl+F` / `Стрелка вправо`                        | Перемещение курсора на один символ вправо.                                                                                       |
| `Ctrl+H` / `Backspace`                             | Удаление символа слева от курсора.                                                                                                |
| `Ctrl+K`                                           | Удаление текста от курсора до конца строки.                                                                                       |
| `Ctrl+Стрелка влево` / `Meta+Стрелка влево` / `Meta+B` | Перемещение курсора на одно слово влево.                                                                                          |
| `Ctrl+N`                                           | Прокрутка вниз по истории ввода.                                                                                                   |
| `Ctrl+P`                                           | Прокрутка вверх по истории ввода.                                                                                                  |
| `Ctrl+R`                                           | Обратный поиск по истории ввода/оболочки.                                                                                         |
| `Ctrl+Y`                                           | Повтор последнего неудачного запроса.                                                                                             |
| `Ctrl+Стрелка вправо` / `Meta+Стрелка вправо` / `Meta+F` | Перемещение курсора на одно слово вправо.                                                                                          |
| `Ctrl+U`                                           | Удаление текста от курсора до начала строки.                                                                                      |
| `Ctrl+V` (Windows: `Alt+V`)                        | Вставка содержимого буфера обмена. Если в буфере обмена находится изображение, оно будет сохранено, а в запрос будет вставлена ссылка на него. |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | Удаление слова слева от курсора.                                                                                                  |
| `Ctrl+X` / `Meta+Enter`                            | Открытие текущего ввода во внешнем редакторе.                                                                                     |

## Предложения

| Сочетание клавиш | Описание                                     |
| ---------------- | -------------------------------------------- |
| `Стрелка вниз`   | Перемещение вниз по списку предложений.      |
| `Tab` / `Enter`  | Принять выбранное предложение.               |
| `Стрелка вверх`  | Перемещение вверх по списку предложений.     |

## Выбор переключателя типа «радиокнопка»

| Сочетание клавиш     | Описание                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------- |
| `Стрелка вниз` / `j` | Переместить выделение вниз.                                                                                   |
| `Enter`              | Подтвердить выбор.                                                                                            |
| `Стрелка вверх` / `k`| Переместить выделение вверх.                                                                                   |
| `1–9`                | Выбрать элемент по его номеру.                                                                                |
| (многозначные числа) | Для элементов с номерами больше 9 нажмите цифры подряд, чтобы выбрать соответствующий элемент.               |

## Интеграция с IDE

| Сочетание клавиш | Описание                                 |
| ---------------- | ---------------------------------------- |
| `Ctrl+G`         | Просмотреть контекст CLI, полученный от IDE |
