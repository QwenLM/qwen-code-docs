# Редактор Zed

> Редактор Zed предоставляет родную поддержку для помощников по написанию кода на основе ИИ через протокол Agent Client Protocol (ACP). Эта интеграция позволяет использовать Qwen Code непосредственно в интерфейсе Zed с предложениями кода в реальном времени.

![Обзор редактора Zed](https://img.alicdn.com/imgextra/i1/O1CN01aAhU311GwEoNh27FP_!!6000000000686-2-tps-3024-1898.png)

### Возможности

- **Родной опыт работы с агентом**: Интегрированная панель ИИ-ассистента в интерфейсе Zed
- **Протокол Agent Client Protocol**: Полная поддержка ACP, обеспечивающая расширенное взаимодействие с IDE
- **Управление файлами**: Упоминание файлов через @ для добавления их в контекст обсуждения
- **История обсуждений**: Доступ к предыдущим обсуждениям внутри Zed

### Требования

- Редактор Zed (рекомендуется последняя версия)
- Установленный Qwen Code CLI

### Установка

1. Установите Qwen Code CLI:

   ```bash
   npm install -g qwen-code
   ```

2. Загрузите и установите [Zed Editor](https://zed.dev/)

3. В Zed нажмите **кнопку настроек** в правом верхнем углу, выберите **"Добавить агента"**, затем **"Создать собственного агента"** и добавьте следующую конфигурацию:

```json
"Qwen Code": {
  "type": "custom",
  "command": "qwen",
  "args": ["--acp"],
  "env": {}
}
```

![Интеграция Qwen Code](https://img.alicdn.com/imgextra/i1/O1CN013s61L91dSE1J7MTgO_!!6000000003734-2-tps-2592-1234.png)

## Устранение неполадок

### Агент не отображается

- Выполните `qwen --version` в терминале, чтобы проверить установку
- Проверьте, что конфигурация JSON действительна
- Перезапустите Zed Editor

### Qwen Code не отвечает

- Проверьте подключение к Интернету
- Убедитесь, что CLI работает, выполнив команду `qwen` в терминале
- [Сообщите о проблеме на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема сохраняется
