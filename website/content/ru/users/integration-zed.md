# Редактор Zed

> Редактор Zed предоставляет встроенную поддержку ИИ-ассистентов для программирования через протокол клиента агента (Agent Client Protocol, ACP). Эта интеграция позволяет использовать Qwen Code непосредственно в интерфейсе Zed с предложениями кода в реальном времени.

![Обзор редактора Zed](https://img.alicdn.com/imgextra/i1/O1CN01aAhU311GwEoNh27FP_!!6000000000686-2-tps-3024-1898.png)

### Возможности

- **Встроенная работа с агентом**: Панель ИИ-ассистента интегрирована в интерфейс Zed  
- **Протокол клиента агента (ACP)**: Полная поддержка ACP для расширенного взаимодействия с IDE  
- **Управление файлами**: Упоминайте файлы с помощью символа `@`, чтобы добавить их в контекст диалога  
- **История диалогов**: Доступ к предыдущим диалогам внутри Zed  

### Требования

- Редактор Zed (рекомендуется последняя версия)  
- Установленный CLI Qwen Code  

### Установка

#### Установка из реестра ACP (рекомендуется)

1. Установите CLI Qwen Code:

```bash
npm install -g @qwen-code/qwen-code
```

2. Скачайте и установите [редактор Zed](https://zed.dev/).

3. В Zed нажмите **кнопку настроек** в правом верхнем углу, выберите **«Добавить агента»**, затем — **«Установить из реестра»**, найдите **Qwen Code** и нажмите **«Установить»**.

   ![Реестр ACP](https://img.alicdn.com/imgextra/i4/O1CN0186ybL61EeG35fHFjy_!!6000000000376-2-tps-3056-1705.png)

   ![Qwen Code установлен из реестра ACP](https://img.alicdn.com/imgextra/i1/O1CN01OXHhoR1J8irAvjs8F_!!6000000000984-2-tps-1247-703.png)

#### Установка вручную

1. Установите CLI Qwen Code:

```bash
npm install -g @qwen-code/qwen-code
```

2. Скачайте и установите [редактор Zed](https://zed.dev/)

3. В Zed нажмите **кнопку настроек** в правом верхнем углу, выберите **«Добавить агента»**, затем **«Создать пользовательского агента»**, и добавьте следующую конфигурацию:

```json
"Qwen Code": {
  "type": "custom",
  "command": "qwen",
  "args": ["--acp"],
  "env": {}
}
```

![Интеграция Qwen Code](https://img.alicdn.com/imgextra/i1/O1CN013s61L91dSE1J7MTgO_!!6000000003734-2-tps-2592-1234.png)

## Устранение неполадок

### Агент не отображается

- Выполните команду `qwen --version` в терминале, чтобы убедиться, что установка прошла успешно.
- Проверьте корректность JSON-конфигурации.
- Перезапустите редактор Zed.

### Qwen Code не отвечает

- Проверьте подключение к интернету.
- Убедитесь, что CLI работает: выполните команду `qwen` в терминале.
- Если проблема сохраняется, [сообщите о ней на GitHub](https://github.com/qwenlm/qwen-code/issues)
