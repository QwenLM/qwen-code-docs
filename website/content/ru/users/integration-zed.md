# Редактор Zed

> Редактор Zed предоставляет нативную поддержку для ИИ-ассистентов в программировании через протокол управления агентами (Agent Control Protocol, ACP). Эта интеграция позволяет использовать Qwen Code напрямую в интерфейсе Zed с предложениями кода в реальном времени.

![Обзор редактора Zed](https://img.alicdn.com/imgextra/i1/O1CN01aAhU311GwEoNh27FP_!!6000000000686-2-tps-3024-1898.png)

### Возможности

- **Нативный опыт взаимодействия с агентом**: Интегрированная панель ИИ-ассистента внутри интерфейса Zed
- **Протокол управления агентами**: Полная поддержка ACP, обеспечивающая расширенное взаимодействие с IDE
- **Управление файлами**: Упоминание файлов с помощью @ для добавления их в контекст разговора
- **История разговоров**: Доступ к предыдущим разговорам внутри Zed

### Требования

- Редактор Zed (рекомендуется последняя версия)
- Установленный CLI Qwen Code

### Установка

1. Установите Qwen Code CLI:

   ```bash
   npm install -g qwen-code
   ```

2. Скачайте и установите [Zed Editor](https://zed.dev/)

3. В Zed нажмите **кнопку настроек** в правом верхнем углу, выберите **"Add agent"**, затем **"Create a custom agent"** и добавьте следующую конфигурацию:

```json
"Qwen Code": {
  "type": "custom",
  "command": "qwen",
  "args": ["--experimental-acp"],
  "env": {}
}
```

## Решение проблем

### Агент не отображается

- Выполните `qwen --version` в терминале, чтобы проверить установку
- Убедитесь, что JSON-конфигурация корректна
- Перезапустите Zed Editor

### Qwen Code не отвечает

- Проверьте подключение к интернету
- Убедитесь, что CLI работает, выполнив `qwen` в терминале
- [Сообщите о проблеме на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема сохраняется
