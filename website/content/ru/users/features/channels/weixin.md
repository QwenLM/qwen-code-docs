# WeChat (Weixin)

В этом руководстве описывается настройка канала Qwen Code в WeChat через официальный API iLink Bot.

## Предварительные требования

- Учётная запись WeChat, способная сканировать QR-коды (мобильное приложение)
- Доступ к платформе iLink Bot (официальный API бота WeChat)

## Настройка

### 1. Вход через QR-код

WeChat использует аутентификацию через QR-код вместо статического токена бота. Выполните команду входа:

```bash
qwen channel configure-weixin
```

Будет отображён URL с QR-кодом. Отсканируйте его при помощи мобильного приложения WeChat для аутентификации. Ваши учётные данные будут сохранены в `~/.qwen/channels/weixin/account.json`.

### 2. Настройка канала

Добавьте канал в `~/.qwen/settings.json`:

```json
{
  "channels": {
    "my-weixin": {
      "type": "weixin",
      "senderPolicy": "pairing",
      "allowedUsers": [],
      "sessionScope": "user",
      "cwd": "/path/to/your/project",
      "model": "qwen3.5-plus",
      "instructions": "You are a concise coding assistant responding via WeChat. Keep responses under 500 characters. Use plain text only."
    }
  }
}
```

Примечание: Каналы WeChat не используют поле `token` — учётные данные поступают из шага входа по QR-коду.

### 3. Запуск канала

```bash
# Запустить только канал WeChat
qwen channel start my-weixin

# Или запустить все настроенные каналы вместе
qwen channel start
```

Откройте WeChat и отправьте сообщение боту. Вы должны увидеть индикатор набора текста («...»), пока агент обрабатывает запрос, а затем — ответ.

## Изображения и файлы

Вы можете отправлять боту не только текст, но и фото и документы.

**Фото:** Отправьте изображение (скриншот, фотографию и т.д.), и агент проанализирует его с помощью функций зрения. Для этого требуется мультимодальная модель — добавьте `"model": "qwen3.5-plus"` (или другую модель со зрением) в конфигурацию канала. Индикатор набора текста показывает, что изображение загружается и обрабатывается.

**Файлы:** Отправьте PDF, файл с кодом или любой документ. Бот загружает и расшифровывает его из CDN WeChat, сохраняет локально, а агент читает его с помощью инструментов для работы с файлами. Это работает с любой моделью.

## Параметры конфигурации

Каналы WeChat поддерживают все стандартные параметры каналов (см. [Обзор каналов](./overview#options)), а также:

| Параметр   | Описание                                                                                      |
| ---------- | --------------------------------------------------------------------------------------------- |
| `baseUrl`  | Переопределить базовый URL API iLink Bot (по умолчанию: `https://ilinkai.weixin.qq.com`)      |

## Ключевые отличия от Telegram

- **Аутентификация:** Вход через QR-код вместо статического токена бота. Сессии могут истекать — канал приостановится и запишет сообщение в лог, если это произойдёт.
- **Форматирование:** WeChat поддерживает только обычный текст. Markdown из ответов агента автоматически удаляется.
- **Индикатор набора текста:** В WeChat используется нативный индикатор «...», а не текстовое сообщение «Работаю...».
- **Группы:** iLink Bot WeChat поддерживает только личные сообщения — групповые чаты не работают.
- **Шифрование медиа:** Изображения и файлы на CDN WeChat зашифрованы с помощью AES-128-ECB. Канал прозрачно выполняет расшифровку.

## Советы

- **Используйте инструкции в обычном тексте** — поскольку WeChat удаляет всю разметку, добавьте инструкции вроде «Используйте только обычный текст», чтобы агент не создавал отформатированные ответы, которые выглядят неаккуратно.
- **Делайте ответы краткими** — пузырьки сообщений WeChat лучше работают с лаконичным текстом. Добавление ограничения на количество символов в инструкциях помогает (например, «Давайте ответы не длиннее 500 символов»).
- **Истечение сессии** — если в логах появилось сообщение «Session expired (errcode -14)», срок действия вашего входа в WeChat истёк. Остановите канал и повторно выполните `qwen channel configure-weixin`, чтобы войти снова.
- **Ограничьте доступ** — используйте `senderPolicy: "pairing"` или `"allowlist"`, чтобы контролировать, кто может общаться с ботом. Подробнее см. в разделе [DM Pairing](./overview#dm-pairing).

## Устранение неполадок

### «WeChat account not configured»

Выполните `qwen channel configure-weixin`, чтобы сначала войти через QR-код.

### «Session expired (errcode -14)»

Срок действия вашей сессии входа в WeChat истёк. Остановите канал и выполните `qwen channel configure-weixin` снова.

### Бот не отвечает

- Проверьте вывод в терминале на наличие ошибок
- Убедитесь, что канал запущен (`qwen channel start my-weixin`)
- Если используется `senderPolicy: "allowlist"`, убедитесь, что ваш ID пользователя WeChat находится в `allowedUsers`

### Изображения не работают

- Убедитесь, что в конфигурации канала указана модель с поддержкой зрения (например, `qwen3.5-plus`)
- Проверьте терминал на наличие ошибок загрузки с CDN — это может указывать на проблему с сетью
