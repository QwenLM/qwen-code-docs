# Контекстные подсказки

Qwen Code включает систему контекстных подсказок, которая помогает узнавать о функциях и отслеживать состояние сессии.

## Подсказки при запуске

При каждом запуске Qwen Code в заголовке отображается подсказка. Подсказки выбираются сначала по приоритету, а затем ротируются между сессиями по алгоритму LRU (наименее недавно использованные) среди подсказок одного приоритета, так что каждый раз вы видите новую подсказку.

Новые пользователи видят подсказки, ориентированные на знакомство с продуктом, в течение первых сессий:

| Сессии | Примеры подсказок                                 |
| ------ | ------------------------------------------------- |
| < 5    | Слэш-команды (`/`), автодополнение по Tab         |
| < 10   | Контекст проекта `QWEN.md`, `--continue` / `--resume` |
| < 15   | Команды оболочки с префиксом `!`                  |

После этого подсказки ротируются по общим функциям, таким как `/compress`, `/approval-mode`, `/insight`, `/btw` и другим.

## Подсказки после ответа

Во время диалога Qwen Code отслеживает использование контекстного окна и показывает подсказки, когда может потребоваться действие:

| Использование контекста | Условие                       | Подсказка                                                   |
| ----------------------- | ----------------------------- | ----------------------------------------------------------- |
| 50–80%                  | После нескольких запросов в сессии | Предлагает `/compress` для освобождения контекста           |
| 80–95%                  | —                             | Предупреждает, что контекст почти заполнен                   |
| >= 95%                  | —                             | Срочно: выполните `/compress` сейчас или `/new` для продолжения |

Для подсказок после ответа установлены индивидуальные таймеры охлаждения, чтобы избежать повторений.

## История подсказок

История отображения подсказок сохраняется в файле `~/.qwen/tip_history.json`. Этот файл отслеживает:

- Количество сессий (используется для выбора подсказок новым пользователям)
- Какие подсказки были показаны и когда (используется для LRU-ротации и таймеров охлаждения)

Вы можете безопасно удалить этот файл, чтобы сбросить историю подсказок.

## Отключение подсказок

Чтобы скрыть все подсказки (как при запуске, так и после ответа), установите `ui.hideTips` в `true` в файле `~/.qwen/settings.json`:

```json
{
  "ui": {
    "hideTips": true
  }
}
```

Вы также можете переключить эту опцию в диалоге настроек с помощью команды `/settings`.

Подсказки также автоматически скрываются, когда включён режим чтения с экрана.
