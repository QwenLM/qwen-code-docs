# Контекстные подсказки

Qwen Code включает систему контекстных подсказок, которая помогает знакомиться с функциями и отслеживать состояние сессии.

## Подсказки при запуске

При каждом запуске Qwen Code в области заголовка отображается подсказка. Подсказки выбираются сначала по приоритету, а затем чередуются между сессиями с использованием алгоритма LRU (least-recently-used) среди подсказок одного приоритета, поэтому каждый раз вы видите новую подсказку.

Новые пользователи в первых сессиях видят подсказки, ориентированные на знакомство с продуктом:

| Сессии | Примеры подсказок                                         |
| -------- | ---------------------------------------------------- |
| < 5      | Слэш-команды (`/`), автодополнение по Tab               |
| < 10     | Контекст проекта `QWEN.md`, `--continue` / `--resume` |
| < 15     | Shell-команды с префиксом `!`                       |

После этого подсказки чередуются, показывая общие функции, такие как `/compress`, `/approval-mode`, `/insight`, `/btw` и другие.

## Подсказки после ответа

Во время диалога Qwen Code отслеживает использование контекстного окна и показывает подсказки, когда могут потребоваться действия:

| Использование контекста | Условие                      | Подсказка                                               |
| ------------- | ------------------------------ | ------------------------------------------------- |
| 50-80%        | После нескольких запросов в сессии | Предлагает `/compress` для освобождения контекста           |
| 80-95%        | —                              | Предупреждает о заполнении контекста                     |
| >= 95%        | —                              | Срочно: выполните `/compress` или `/new` для продолжения |

Для подсказок после ответа настроены индивидуальные задержки показа, чтобы избежать повторений.

## История подсказок

История отображения подсказок сохраняется в `~/.qwen/tip_history.json`. Этот файл отслеживает:

- Количество сессий (используется для выбора подсказок для новых пользователей)
- Какие подсказки были показаны и когда (используется для ротации по LRU и задержек показа)

Вы можете безопасно удалить этот файл, чтобы сбросить историю подсказок.

## Отключение подсказок

Чтобы скрыть все подсказки (как при запуске, так и после ответа), установите для `ui.hideTips` значение `true` в `~/.qwen/settings.json`:

```json
{
  "ui": {
    "hideTips": true
  }
}
```

Вы также можете переключить эту настройку в диалоге настроек с помощью команды `/settings`.

Подсказки также автоматически скрываются при включении режима программы чтения с экрана.
