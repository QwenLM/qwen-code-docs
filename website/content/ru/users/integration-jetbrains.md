# Среды разработки JetBrains

> Среды разработки JetBrains обеспечивают встроенную поддержку помощников по написанию кода с ИИ через протокол Agent Client Protocol (ACP). Эта интеграция позволяет использовать Qwen Code непосредственно в вашей среде разработки JetBrains с предложениями кода в реальном времени.

### Возможности

- **Встроенный агентский интерфейс**: Интегрированная панель помощника с ИИ в вашей среде разработки JetBrains
- **Протокол Agent Client Protocol**: Полная поддержка ACP, обеспечивающая расширенное взаимодействие со средой разработки
- **Управление символами**: Упоминание файлов с помощью #-символа для добавления их в контекст обсуждения
- **История обсуждений**: Доступ к предыдущим обсуждениям внутри среды разработки

### Требования

- Среда разработки JetBrains с поддержкой ACP (IntelliJ IDEA, WebStorm, PyCharm и т. д.)
- Установленный Qwen Code CLI

### Установка

1. Установите Qwen Code CLI:

   ```bash
   npm install -g @qwen-code/qwen-code
   ```

2. Откройте вашу IDE JetBrains и перейдите в окно инструмента AI Chat.

3. Нажмите меню с тремя точками в правом верхнем углу и выберите **Configure ACP Agent**, затем настройте Qwen Code со следующими параметрами:

```json
{
  "agent_servers": {
    "qwen": {
      "command": "/path/to/qwen",
      "args": ["--acp"],
      "env": {}
    }
  }
}
```

4. Агент Qwen Code теперь должен быть доступен на панели AI Assistant

## Устранение неполадок

### Агент не отображается

- Выполните `qwen --version` в терминале, чтобы проверить установку
- Убедитесь, что ваша IDE JetBrains поддерживает ACP
- Перезапустите вашу IDE JetBrains

### Qwen Code не отвечает

- Проверьте подключение к интернету
- Убедитесь, что CLI работает, запустив `qwen` в терминале
- [Сообщите о проблеме на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема сохраняется
