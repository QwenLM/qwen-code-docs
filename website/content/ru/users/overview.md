## Начните за 30 секунд

Предварительные требования:

- Аккаунт [Qwen Code](https://chat.qwen.ai/auth?mode=register)
- Требуется [Node.js 20+](https://nodejs.org/zh-cn/download), вы можете использовать `node -v`, чтобы проверить версию. Если не установлено, используйте следующую команду для установки.

### Установите Qwen Code:

**NPM**(рекомендуется)

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew**(macOS, Linux)

```bash
brew install qwen-code
```

### Начните использовать Qwen Code:

```bash
cd your-project
qwen
```

Выберите аутентификацию **Qwen OAuth (Бесплатно)** и следуйте инструкциям для входа в систему. Затем начните с изучения своей кодовой базы. Попробуйте одну из этих команд:

```
what does this project do?
```

При первом использовании вас попросят войти в систему. Вот и все! [Продолжить с быстрого начала работы (5 мин) →](./quickstart)

## Что делает Qwen Code для вас

- **Создание функций из описаний**: Расскажите Qwen Code, что вы хотите создать, простым языком. Он составит план, напишет код и убедится, что все работает.
- **Отладка и исправление проблем**: Опишите ошибку или вставьте сообщение об ошибке. Qwen Code проанализирует вашу кодовую базу, найдет проблему и реализует исправление.
- **Навигация по любой кодовой базе**: Спросите что угодно о кодовой базе вашей команды и получите содержательный ответ. Qwen Code отслеживает структуру всего вашего проекта, может находить актуальную информацию из интернета и с помощью [MCP](./features/mcp) может извлекать данные из внешних источников, таких как Google Drive, Figma и Slack.
- **Автоматизация рутинных задач**: Исправляйте мелкие проблемы с линтером, разрешайте конфликты слияния и пишите заметки о выпуске. Делайте все это одной командой с ваших разработческих машин или автоматически в CI.

## Почему разработчики любят Qwen Code

- **Работает в вашем терминале**: Не еще одно окно чата. Не еще одна IDE. Qwen Code появляется там, где вы уже работаете, с инструментами, которые вы уже любите.
- **Выполняет действия**: Qwen Code может напрямую редактировать файлы, запускать команды и создавать коммиты. Нужно больше? [MCP](./features/mcp) позволяет Qwen Code читать ваши документы дизайна в Google Drive, обновлять тикеты в Jira или использовать _ваши_ собственные инструменты разработчика.
- **Философия Unix**: Qwen Code можно комбинировать и скриптовать. `tail -f app.log | qwen -p "Сообщи мне в Slack, если в этом потоке лога появятся аномалии"` _работает_. Ваш CI может выполнить `qwen -p "Если есть новые текстовые строки, переведи их на французский и создай PR для проверки @lang-fr-team"`.
