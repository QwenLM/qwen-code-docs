# Обзор Qwen Code

> Узнайте о Qwen Code — агентном инструменте для программирования от Qwen, который работает в вашем терминале и помогает быстрее, чем раньше, превращать идеи в код.

## Начало работы за 30 секунд

Предварительные требования:

- Аккаунт [Qwen Code](https://chat.qwen.ai/auth?mode=register)
- Требуется [Node.js 20+](https://nodejs.org/zh-cn/download). Вы можете проверить версию с помощью команды `node -v`. Если Node.js не установлен, воспользуйтесь следующей командой для его установки.

### Установка Qwen Code:

**NPM** (рекомендуется)

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew** (macOS, Linux)

```bash
brew install qwen-code
```

### Начало работы с Qwen Code:

```bash
cd your-project
qwen
```

Выберите аутентификацию **Qwen OAuth (Бесплатно)** и следуйте инструкциям для входа. Затем давайте начнем с изучения вашей кодовой базы. Попробуйте одну из этих команд:

```
what does this project do?
```

## Что делает Qwen Code для вас

- **Создание функций по описанию**: Расскажите Qwen Code, что вы хотите создать, простым языком. Он составит план, напишет код и убедится, что всё работает.
- **Отладка и исправление проблем**: Опишите ошибку или вставьте сообщение об ошибке. Qwen Code проанализирует вашу кодовую базу, определит проблему и реализует исправление.
- **Навигация по любой кодовой базе**: Задавайте любые вопросы о кодовой базе вашей команды и получайте продуманные ответы. Qwen Code сохраняет осведомлённость о всей структуре вашего проекта, может находить актуальную информацию в интернете, а с помощью [MCP](../users/features/mcp) — извлекать данные из внешних источников, таких как Google Drive, Figma и Slack.
- **Автоматизация рутинных задач**: Исправляйте мелкие проблемы с линтером, разрешайте конфликты слияния и пишите заметки к релизам. Всё это можно сделать одной командой с ваших машин разработчиков или автоматически в CI.

## Почему разработчикам нравится Qwen Code

- **Работает в вашем терминале**: Не очередное окно чата. Не очередная IDE. Qwen Code работает там, где вы уже работаете, с теми инструментами, которые вы уже любите.
- **Выполняет действия**: Qwen Code может напрямую редактировать файлы, запускать команды и создавать коммиты. Нужно больше? [MCP](../users/features/mcp) позволяет Qwen Code читать ваши проектные документы в Google Drive, обновлять задачи в Jira или использовать _ваши_ собственные инструменты разработчика.
- **Философия Unix**: Qwen Code является компонуемым и скриптуемым. `tail -f app.log | qwen -p "Отправь мне сообщение в Slack, если в этом потоке логов появятся аномалии"` — _работает_. Ваш CI может выполнить `qwen -p "Если есть новые текстовые строки, переведи их на французский и создай PR для проверки участникам @lang-fr-team"`.
