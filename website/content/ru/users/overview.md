## Начните за 30 секунд

### Установка Qwen Code:

**Linux / macOS**

```sh
curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh | bash
```

**Windows (запустите командную строку от имени администратора)**

```sh
curl -fsSL -o %TEMP%\install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%\install-qwen.bat
```

> [!note]
>
> Рекомендуется перезапустить терминал после установки, чтобы изменения в переменных окружения вступили в силу. Если установка завершилась неудачно, обратитесь к разделу [Установка вручную](./quickstart#manual-installation) в руководстве «Быстрый старт».

### Начните использовать Qwen Code:

```bash
cd your-project
qwen
```

Выберите аутентификацию **Qwen OAuth (Бесплатно)** и следуйте инструкциям для входа в систему. Затем начнём с изучения вашей кодовой базы. Попробуйте одну из этих команд:

```
что делает этот проект?
```

При первом использовании вас попросят войти в систему. Вот и всё! [Продолжить с краткого руководства (5 минут) →](./quickstart)

## Что делает для вас Qwen Code

- **Создание функций по описанию**: Расскажите Qwen Code на простом языке, что вы хотите создать. Он составит план, напишет код и убедится, что он работает.
- **Отладка и устранение проблем**: Опишите ошибку или вставьте сообщение об ошибке. Qwen Code проанализирует вашу кодовую базу, определит проблему и реализует исправление.
- **Навигация по любой кодовой базе**: Задайте любой вопрос о кодовой базе вашей команды и получите вдумчивый ответ. Qwen Code отслеживает структуру всего вашего проекта, может находить актуальную информацию из интернета и с помощью [MCP](./features/mcp) получать данные из внешних источников, таких как Google Drive, Figma и Slack.
- **Автоматизация рутинных задач**: Исправляйте мелкие проблемы с линтером, разрешайте конфликты слияния и пишите заметки к релизам. Выполните всё это одной командой прямо с ваших рабочих машин разработчика или автоматически в CI.

## Почему разработчики любят Qwen Code

- **Работает в вашем терминале**: Не ещё одно окно чата. Не ещё одна IDE. Qwen Code интегрируется там, где вы уже работаете, и использует те инструменты, которыми вы уже пользуетесь.
- **Выполняет действия**: Qwen Code может напрямую редактировать файлы, запускать команды и создавать коммиты. Нужно больше возможностей? [MCP](./features/mcp) позволяет Qwen Code читать ваши документы по проектированию в Google Drive, обновлять задачи в Jira или использовать _ваши_ собственные инструменты для разработчиков.
- **Философия Unix**: Qwen Code составной и поддерживает скриптование. Команда `tail -f app.log | qwen -p "Отправь мне уведомление в Slack, если в этом потоке логов появятся аномалии"` _работает_. Ваш CI может выполнять команду `qwen -p "Если появились новые текстовые строки, переведи их на французский и создай pull request для проверки командой @lang-fr-team"`.
