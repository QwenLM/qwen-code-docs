# Обзор Qwen Code

[![@qwen-code/qwen-code версия](https://img.shields.io/npm/v/@qwen-code/qwen-code.svg)](https://www.npmjs.com/package/@qwen-code/qwen-code)

> Узнайте о Qwen Code, инструменте для программирования с агентом Qwen, который живет в вашем терминале и помогает вам превращать идеи в код быстрее, чем раньше.

## Начните за 30 секунд

Предварительные требования:

- Учетная запись [Qwen Code](https://chat.qwen.ai/auth?mode=register)
- Требуется [Node.js 20+](https://nodejs.org/zh-cn/download), вы можете использовать `node -v`, чтобы проверить версию. Если он не установлен, используйте следующую команду для установки.

### Установите Qwen Code:

**NPM**(рекомендуется)

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew**(macOS, Linux)

```bash
brew install qwen-code
```

### Начните использовать Qwen Code:

```bash
cd your-project
qwen
```

Выберите аутентификацию **Qwen OAuth (Бесплатно)** и следуйте инструкциям для входа. Затем начните с изучения своей кодовой базы. Попробуйте одну из следующих команд:

```
what does this project do?
```

При первом использовании вас попросят войти в систему. Вот и всё! [Продолжить с быстрого старта (5 мин) →](./quickstart)

## Что делает Qwen Code для вас

- **Создание функций из описаний**: Расскажите Qwen Code, что вы хотите создать, простым языком. Он составит план, напишет код и обеспечит его работоспособность.
- **Отладка и исправление проблем**: Опишите ошибку или вставьте сообщение об ошибке. Qwen Code проанализирует вашу кодовую базу, найдет проблему и реализует исправление.
- **Навигация по любой кодовой базе**: Задавайте любые вопросы о кодовой базе вашей команды и получайте обдуманные ответы. Qwen Code отслеживает структуру всего вашего проекта, может находить актуальную информацию из веба и с помощью [MCP](./features/mcp) может извлекать данные из внешних источников, таких как Google Drive, Figma и Slack.
- **Автоматизация рутинных задач**: Исправляйте мелкие проблемы с линтером, разрешайте конфликты слияния и пишите заметки о выпуске. Выполняйте все это одной командой с ваших разработческих машин или автоматически в CI.

## Почему разработчики любят Qwen Code

- **Работает в вашем терминале**: Не еще одно окно чата. Не еще одна IDE. Qwen Code встречает вас там, где вы уже работаете, с инструментами, которые вы уже любите.
- **Выполняет действия**: Qwen Code может напрямую редактировать файлы, запускать команды и создавать коммиты. Нужно больше? [MCP](./features/mcp) позволяет Qwen Code читать ваши технические документы в Google Drive, обновлять заявки в Jira или использовать _ваши_ пользовательские инструменты разработчика.
- **Философия Unix**: Qwen Code компонуем и скриптуем. `tail -f app.log | qwen -p "Slack мне, если ты видишь какие-либо аномалии в этом потоке логов"` _работает_. Ваш CI может запускать `qwen -p "Если есть новые текстовые строки, переведи их на французский и создай PR для проверки @lang-fr-team"`
