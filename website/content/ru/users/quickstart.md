# Быстрый старт

> 👏 Добро пожаловать в Qwen Code!

Это руководство по быстрому старту позволит вам начать использовать ИИ-ассистента для программирования всего за несколько минут. К концу вы поймёте, как применять Qwen Code для типовых задач разработки.

## Перед началом

Убедитесь, что у вас есть:

- Открытый **терминал** или командная строка
- Проект с исходным кодом, над которым вы будете работать
- Учётная запись [Qwen Code](https://chat.qwen.ai/auth?mode=register)

## Шаг 1. Установка Qwen Code

Для установки Qwen Code воспользуйтесь одним из следующих способов:

### Быстрая установка (рекомендуется)

**Linux / macOS**

```sh
curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh | bash
```

**Windows (запустите командную строку от имени администратора)**

```sh
curl -fsSL -o %TEMP%\install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%\install-qwen.bat
```

> [!note]
>
> Рекомендуется перезапустить терминал после установки, чтобы изменения в переменных окружения вступили в силу.

### Установка вручную

**Предварительные требования**

Убедитесь, что у вас установлен Node.js версии 20 или новее. Скачайте его с сайта [nodejs.org](https://nodejs.org/ru/download).

**NPM**

```bash
npm install -g @qwen-code/qwen-code@latest
```

**Homebrew (macOS, Linux)**

```bash
brew install qwen-code
```

## Шаг 2: Войдите в свою учётную запись

Для использования Qwen Code требуется учётная запись. При запуске интерактивного сеанса с помощью команды `qwen` вам будет предложено выполнить вход:

```bash
# При первом использовании появится запрос на вход
qwen
```

Следуйте инструкциям для входа в систему с помощью вашей учётной записи. Выполните `/auth`, выберите `Qwen OAuth`, войдите в свою учётную запись и следуйте инструкциям для подтверждения. После входа в систему ваши учётные данные сохраняются, и повторный вход не потребуется.

> [!note]  
>  
> При первом подключении Qwen Code к вашей учётной записи Qwen автоматически создаёт рабочее пространство с именем `.qwen`. Это рабочее пространство обеспечивает централизованный учёт и управление затратами на использование Qwen Code в вашей организации.

> [!tip]  
>  
> Если вам нужно снова войти в систему или переключиться на другую учётную запись, используйте команду `/auth` внутри Qwen Code.

## Шаг 3. Запустите первую сессию

Откройте терминал в любом каталоге проекта и запустите Qwen Code:

```bash  
# необязательно  
cd /путь/к/вашему/проекту  

# запуск qwen  
qwen  
```

Вы увидите стартовый экран Qwen Code с информацией о текущей сессии, последними диалогами и новейшими обновлениями. Введите `/help`, чтобы получить список доступных команд.

## Общение с Qwen Code

### Задайте первый вопрос

Qwen Code проанализирует ваши файлы и предоставит краткое резюме. Вы также можете задавать более конкретные вопросы:

```
объясни структуру папок
```

Вы также можете спросить Qwen Code о его собственных возможностях:

```
что умеет Qwen Code?
```

> [!note]
>
> Qwen Code читает ваши файлы по мере необходимости — добавлять контекст вручную не требуется. Кроме того, Qwen Code имеет доступ к собственной документации и может отвечать на вопросы о своих функциях и возможностях.

### Внесите первое изменение в код

Теперь заставим Qwen Code выполнить реальную программистскую задачу. Попробуйте простую операцию:

```
добавь функцию «hello world» в основной файл
```

Qwen Code выполнит следующие действия:

1. Найдёт подходящий файл  
2. Покажет предложенные изменения  
3. Запросит ваше подтверждение  
4. Внесёт правку  

> [!note]
>
> Qwen Code всегда запрашивает разрешение перед изменением файлов. Вы можете подтверждать изменения по отдельности или включить режим «Принять все» для текущей сессии.

### Работа с Git в Qwen Code

Qwen Code делает операции с Git диалоговыми:

```
какие файлы я изменил?
```

```
зафиксируй мои изменения с содержательным сообщением
```

Также можно запросить выполнение более сложных операций Git:

```
создай новую ветку с именем feature/quickstart
```

```
покажи последние 5 коммитов
```

```
помоги разрешить конфликты слияния
```

### Исправление ошибки или добавление функциональности

Qwen Code эффективно справляется с отладкой и реализацией новых функций.

Опишите требуемое на естественном языке:

```
добавь проверку входных данных в форму регистрации пользователя
```

Или укажите существующую проблему:

```
есть ошибка: пользователи могут отправлять пустые формы — исправь её
```

Qwen Code выполнит следующие действия:

- Найдёт соответствующий код  
- Поймёт контекст  
- Реализует решение  
- Запустит тесты, если они доступны

## Основные команды

Вот самые важные команды для ежедневного использования:

| Команда                 | Что делает                                      | Пример                          |
| ----------------------- | ------------------------------------------------ | ------------------------------- |
| `qwen`                  | запуск Qwen Code                                 | `qwen`                          |
| `/auth`                 | изменение метода аутентификации                  | `/auth`                         |
| `/help`                 | отображение справочной информации по доступным командам | `/help` или `/?`                |
| `/compress`             | замена истории чата на сводку для экономии токенов | `/compress`                     |
| `/clear`                | очистка содержимого терминала                    | `/clear` (ярлык: `Ctrl+L`)      |
| `/theme`                | изменение визуальной темы Qwen Code             | `/theme`                        |
| `/language`             | просмотр или изменение языковых настроек        | `/language`                     |
| → `ui [язык]`           | установка языка интерфейса пользователя         | `/language ui zh-CN`            |
| → `output [язык]`       | установка языка вывода LLM                      | `/language output Chinese`      |
| `/quit`                 | немедленный выход из Qwen Code                  | `/quit` или `/exit`             |

Полный список команд см. в [справочнике CLI](./features/commands).

## Полезные советы для начинающих

**Формулируйте запросы конкретно**

- Вместо: «исправьте ошибку»  
- Попробуйте: «исправьте ошибку входа, из-за которой пользователи видят пустой экран после ввода неверных учётных данных»

**Используйте пошаговые инструкции**

- Разбивайте сложные задачи на шаги:

```
1. создать новую таблицу базы данных для профилей пользователей
2. создать API-эндпоинт для получения и обновления профилей пользователей
3. разработать веб-страницу, позволяющую пользователям просматривать и редактировать свои данные
```

**Сначала дайте Qwen Code возможность исследовать код**

- Прежде чем вносить изменения, дайте Qwen Code понять структуру вашего кода:

```
проанализировать схему базы данных
```

```
создать дашборд, отображающий товары, которые чаще всего возвращают наши клиенты из Великобритании
```

**Экономьте время с помощью сочетаний клавиш**

- Нажмите `?`, чтобы увидеть все доступные сочетания клавиш  
- Используйте клавишу Tab для автодополнения команд  
- Нажмите ↑ для просмотра истории команд  
- Введите `/`, чтобы увидеть все команды, начинающиеся с косой черты
