# Visual Studio Code

> Расширение VS Code (бета) позволяет видеть изменения от Qwen в реальном времени через нативный графический интерфейс, встроенный прямо в вашу IDE, что упрощает доступ и взаимодействие с Qwen Code.

### Особенности

- **Нативный интерфейс IDE**: Выделенная боковая панель Qwen Code, доступная по иконке Qwen
- **Режим автоматического применения правок**: Автоматически применять изменения от Qwen по мере их внесения
- **Управление файлами**: Упоминание файлов через @ или прикрепление файлов и изображений через системный выбор файлов
- **История бесед**: Доступ к прошлым перепискам
- **Несколько сеансов**: Одновременный запуск нескольких сеансов Qwen Code

### Требования

- VS Code 1.85.0 или выше

### Установка

Загрузите и установите расширение из [Visual Studio Code Extension Marketplace](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion).

## Устранение неполадок

### Расширение не устанавливается

- Убедитесь, что у вас VS Code 1.85.0 или выше
- Проверьте, что VS Code имеет разрешение на установку расширений
- Попробуйте установить напрямую с сайта Marketplace

### Qwen Code не отвечает

- Проверьте подключение к интернету
- Начните новый разговор, чтобы проверить, сохраняется ли проблема
- [Создайте issue на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема не исчезает
