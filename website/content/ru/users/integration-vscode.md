# Visual Studio Code

> Расширение VS Code (бета-версия) позволяет в реальном времени отслеживать изменения, вносимые Qwen, через собственный графический интерфейс, интегрированный непосредственно в вашу IDE. Это упрощает доступ к Qwen Code и взаимодействие с ним.

### Возможности

- **Родной опыт работы в IDE**: выделанная боковая панель Qwen Code, доступная по щелчку на значке Qwen
- **Режим автоматического принятия правок**: автоматическое применение изменений, вносимых Qwen, по мере их создания
- **Управление файлами**: упоминание файлов с помощью символа `@` или прикрепление файлов и изображений через системный проводник файлов
- **История диалогов**: доступ к предыдущим диалогам
- **Несколько сессий**: одновременный запуск нескольких сессий Qwen Code

### Требования

- VS Code версии 1.85.0 или выше

### Установка

Скачайте и установите расширение из [рынка расширений Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion).

## Устранение неполадок

### Расширение не устанавливается

- Убедитесь, что у вас установлена VS Code версии 1.85.0 или выше.
- Проверьте, разрешено ли в VS Code устанавливать расширения.
- Попробуйте установить расширение напрямую с веб-сайта Marketplace.

### Qwen Code не отвечает

- Проверьте подключение к интернету.
- Начните новый диалог, чтобы проверить, сохраняется ли проблема.
- [Сообщите о проблеме на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема не исчезнет.
