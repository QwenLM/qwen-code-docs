# Visual Studio Code

> Расширение для VS Code (бета-версия) позволяет видеть изменения Qwen в реальном времени через нативный графический интерфейс, интегрированный непосредственно в вашу IDE, что облегчает доступ к Qwen Code и взаимодействие с ним.

### Возможности

- **Нативный опыт работы в IDE**: Выделенная боковая панель Qwen Code, доступ к которой осуществляется через значок Qwen
- **Режим автоматического принятия изменений**: Автоматическое применение изменений Qwen по мере их внесения
- **Управление файлами**: Упоминание файлов с помощью @ или прикрепление файлов и изображений с использованием системного средства выбора файлов
- **История переписки**: Доступ к предыдущим разговорам
- **Несколько сессий**: Возможность одновременного запуска нескольких сессий Qwen Code

### Требования

- VS Code 1.85.0 или выше

### Установка

1. Установите Qwen Code CLI:

   ```bash
   npm install -g qwen-code
   ```

2. Загрузите и установите расширение из [магазина расширений Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion).

## Устранение неполадок

### Расширение не устанавливается

- Убедитесь, что у вас установлен VS Code версии 1.85.0 или выше
- Проверьте, разрешено ли VS Code устанавливать расширения
- Попробуйте установить расширение напрямую с веб-сайта Marketplace

### Qwen Code не отвечает

- Проверьте подключение к интернету
- Начните новый диалог, чтобы проверить, сохраняется ли проблема
- [Сообщите о проблеме на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема не исчезнет
