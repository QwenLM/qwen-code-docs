# Visual Studio Code

> Расширение для VS Code (бета) позволяет отслеживать изменения Qwen в реальном времени через нативный графический интерфейс, встроенный непосредственно в вашу IDE, что упрощает доступ к Qwen Code и взаимодействие с ним.

### Возможности

- **Нативная интеграция в IDE**: Отдельная боковая панель Qwen Code, открываемая через значок Qwen
- **Режим автоматического принятия правок**: Автоматическое применение изменений от Qwen по мере их создания
- **Управление файлами**: @-упоминание файлов или прикрепление файлов и изображений через системный диалог выбора
- **История диалогов**: Доступ к предыдущим разговорам
- **Несколько сессий**: Одновременный запуск нескольких сессий Qwen Code

### Требования

- VS Code версии 1.85.0 или выше

### Установка

Скачайте и установите расширение из [Visual Studio Code Extension Marketplace](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion).

## Устранение неполадок

### Расширение не устанавливается

- Убедитесь, что у вас установлена VS Code версии 1.85.0 или выше
- Проверьте, что у VS Code есть разрешения на установку расширений
- Попробуйте установить расширение напрямую с сайта Marketplace

### Qwen Code не отвечает

- Проверьте подключение к интернету
- Начните новый диалог, чтобы проверить, сохраняется ли проблема
- [Создайте issue на GitHub](https://github.com/qwenlm/qwen-code/issues), если проблема не исчезнет
