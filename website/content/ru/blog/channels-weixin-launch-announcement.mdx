## 🎯 Зачем вам нужен канал WeChat?

![Диаграмма основной концепции канала WeChat](https://img.alicdn.com/imgextra/i4/O1CN01p9WUdg27Z2P6L8UTB_!!6000000007810-0-tps-1376-768.jpg)

Как разработчик, часто ли вы сталкиваетесь с такими сценариями:

- **Во время поездки** вы внезапно придумываете решение бага и хотите, чтобы ИИ немедленно проверил ваш подход
- **Между встречами** нужно быстро проверить использование API, но не хочется открывать компьютер
- **Поздно ночью** вдохновение приходит в постели, и вы хотите, чтобы ИИ сгенерировал для вас каркас кода
- **В путешествии** возникает срочная онлайн-проблема, и нужно удалённо помочь коллегам с устранением неполадок

Раньше вам нужно было открыть компьютер и запустить терминал, чтобы использовать Qwen Code. Теперь **ваш WeChat—это ваша среда разработки**.

## 🚀 Краткое руководство по началу работы

### Предварительные требования

Убедитесь, что вы:

- ✅ Установили preview-версию Qwen Code `npm i @qwen-code/qwen-code@0.14.0-preview.0 -g`
- ✅ Подготовили аккаунт WeChat, который может получать сообщения

> В эпоху ИИ, кто ещё настраивает параметры вручную? Вам нужно просто выполнить `npx skills add https://github.com/QwenLM/qwen-code-examples --skill weixin-channel-setup` чтобы установить этот навык weixin-channel-setup, затем открыть Qwen и спросить: `{'/'}skills weixin-channel-setup Пожалуйста, помогите мне настроить`

Если вы предпочитаете традиционный метод настройки, обратитесь к трём шагам ниже:

![Диаграмма настройки в три шага](https://img.alicdn.com/imgextra/i4/O1CN01aWJWAF1YwqRJ6ryra_!!6000000003124-0-tps-1376-768.jpg)

### Шаг 1: Сканирование для подключения

Выполните в терминале:

```bash
qwen channel configure-weixin
```

В терминале есть "URL QR-кода". Скопируйте ссылку и откройте в браузере, затем отсканируйте с помощью WeChat.

- Возможно, вам будет предложено установить WeChat
  После успешного сканирования вы увидите:

```
Connected to WeChat successfully!
Credentials saved. You can now start a weixin channel with:
  qwen channel start <name>
```

На этом этапе ваши учётные данные WeChat были безопасно сохранены локально.

### Шаг 2: Настройка канала

Отредактируйте `~/.qwen/settings.json` и добавьте следующую конфигурацию:

```json
"channels": {
    "my-weixin": {
      "type": "weixin",
      "senderPolicy": "open",
      "sessionScope": "user",
      "model": "qwen3.5-plus",
      "instructions": "You are a concise coding assistant responding via WeChat. Keep responses under 500 characters. Use plain text only."
    }
  }
```

### Шаг 3: Запуск и тестирование

```bash
qwen channel start my-weixin
```

После появления сообщения `[Channel] "my-weixin" is running.` откройте WeChat:

1. Найдите своего бота-ассистента (того, которого вы только что отсканировали)
2. Отправьте сообщение: "Привет, помоги мне написать Hello World на Python"
3. Дождитесь ответа...

🎉 **Поздравляем! Теперь вы можете писать код с помощью WeChat!**

## 🔧 Продвинутые советы

### 1. Настройка персоны ИИ

Через поле `instructions` вы можете создать своего собственного ИИ-ассистента:

```json
{
  "instructions": "Вы старший TypeScript-инженер. При ответе на вопросы:\n1. Приоритизируйте выполнимые примеры кода\n2. Объясняйте ключевые проектные решения\n3. Указывайте на потенциальные граничные случаи\n4. Сохраняйте профессиональный, но дружелюбный тон"
}
```

### 2. Автозапуск при загрузке

Честно говоря, лучше спросить Qwen Code—пусть он поможет вам настроить～
