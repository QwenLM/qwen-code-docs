## Всё ещё работаете по ночам над маркетинговыми материалами?

Наверняка вы с этим сталкивались:

Новый продукт готовится к запуску. Маркетинговая команда просит мобильный сплэш-экран, три рекламных баннера, набор изображений для страницы товара, 15-секундное видео презентации продукта, плюс тексты для соцсетей и статью в WeChat — **дедлайн послезавтра**.

Вы открываете Photoshop, ждёте 30 секунд загрузки и обнаруживаете, что дизайнер в отпуске. Открываете видеоредактор, полчаса перетаскиваете клипы и экспортируете в неправильном разрешении. Открываете документ для текста, пишете две строки и замираете перед экраном.

**Работаете до 2 часов ночи. Материалы наконец готовы. Следующий запуск — всё заново.**

А что, если всё может работать так:

- Набрать одно предложение в терминале, **сгенерировать мобильный сплэш-экран за 10 секунд**
- Сфотографировать продукт на телефон, отправить в группу Feishu @боту, **получить профессиональное фото товара за 1 минуту**
- Сказать ИИ «напиши мне обзор этого пауэрбанка», **черновик готов за 30 секунд**

Разве это не то, как должна выглядеть работа в 2026 году?

В этой статье я покажу на 3 реальных сценариях, как использовать **Qwen Code + Bailian CLI** для всего — от генерации изображений и ретуши фото до написания текстов. Каждый сценарий содержит пошаговые инструкции, которые можно применить сразу.

## Сначала об инструментах: что такое Qwen Code и Bailian CLI?

Простыми словами:

- **Qwen Code** — это ИИ-помощник для программирования, но он умеет не только писать код — это универсальный ИИ-агент, который понимает инструкции на естественном языке и помогает вызывать различные инструменты для выполнения задач
- **Bailian CLI** (команда `bl`) — это инструмент командной строки платформы Model Studio от Alibaba Cloud, объединяющий возможности текст-в-изображение, изображение-в-изображение, текст-в-видео, изображение-в-видео и многое другое

Вместе они образуют **линию производства маркетинговых материалов, которая понимает обычный язык**.

### Что вам понадобится

- Установить Qwen Code CLI: [Обзор Qwen Code \| Qwen Code Docs](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)
- Установить Bailian CLI: [Alibaba Cloud Bailian CLI \| Все возможности ИИ одной командой](https://bailian.console.aliyun.com/cli?source_channel=cn-beijing-cli&&utm_content=se_1023346373)
- API-ключ Alibaba Cloud Model Studio (получить в [Консоли Model Studio](https://bailian.console.aliyun.com))

Готовы? Начинаем.

## Сценарий 1: Генерация маркетинговых изображений одной фразой 🎨

При каждой распродаже или запуске продукта нужно просить дизайнера сделать главные изображения, графику для страниц товара и промо-баннеры. График дизайнера забит, даже мелкие правки занимают полдня. **Когда срочно — просто нет времени ждать.**

Теперь достаточно описать нужное изображение одним предложением, и ИИ его сгенерирует. Главные изображения на белом фоне, сцены, промо-атмосфера, техно-стиль — всё доступно.

Есть продукт? Начнём с генерации изображений товара:

```
帮我生成 3 张充电宝的电商白底主图，要求：淡米色背景、专业棚拍风格、柔和灯光、底部轻微阴影、构图居中干净，3:4 尺寸
```

Довольны изображениями? Хотите добавить текст с аргументами продажи? Просто добавьте ещё одну строку:

```
已经生成了商品图，现在给这些图片加上宣传语，以及卖点内容。
```

Не устраивает? Подкорректируйте

Картинка на 80%, но нужна доработка? Просто скажите Qwen Code:

```
把第一张图的背景换成紫红渐变，加金色点缀，更适合双 11 氛围
```

Хотите изменить текст? Не нужно начинать заново. Итерируйте на существующем изображении — чем больше уточнений, тем точнее результат. Поскольку HTML-дизайн управляет текстом отдельно, изменение текста не требует повторной генерации изображения, экономя токены. Просто скажите Qwen Code:

```
把 20000mAh，修改为10000mAh
```

### Поддерживаемые типы изображений

| Тип | Размер | Применение |
| --- | --- | --- |
| Главное изображение e-commerce | 1024×1024 (Квадрат 1:1) | Главные изображения товаров Taobao/JD, промо-изображения |
| Страница деталей товара | 1024×1366 (Портрет 3:4) | Отображение одного экрана страницы деталей |
| Мобильный сплэш-экран | 1080×1920 (Портрет 9:16) | Реклама при запуске приложения |
| Рекламный баннер | 1344×756 (Альбомный 16:9) | PC-баннер |

Все размеры поддерживают **изображения товаров с маркетинговым текстом** — ИИ сначала генерирует чистое визуальное базовое изображение, затем с помощью HTML точно накладывает аргументы продажи, слоганы, цены и другой текст, экспортируя готовый PNG. Текст остаётся чётким без искажений, а изменение текста занимает всего 2 секунды для повторного рендеринга без повторной генерации изображения.

## Сценарий 2: Сфотографируйте в Feishu, автоматически получите изображения и видео товара 📸

Фотосъёмка товаров для e-commerce — дорогое удовольствие: аренда студии, найм фотографа, постобработка — весь пакет начинается от нескольких тысяч юаней, срок — одна-две недели. Маленькие команды просто не могут себе это позволить. А теперь все платформы продвигают короткие видео, так что видео товаров — ещё одна статья расходов.

Лучший подход сейчас: **Сделайте фото на телефон, отправьте в группу Feishu @боту, и ИИ автоматически определит ваш товар, затем в одном потоке сгенерирует изображения на белом фоне, маркетинговые изображения с текстом и 5-секундное видео-презентацию.**

Компьютер для всего процесса не нужен.

**Шаг 1: Сфотографируйте и отправьте в группу**

Сделайте быстрый снимок товара на телефон (любой фон — на столе, на диване, где угодно), отправьте в группу Feishu и @Qwen Code:

```
使用 open1v-product-visual 技能帮我生成2张商品图
```

Бот автоматически делает 3 вещи:

1. **Определяет товар** — автоматически распознаёт, что сфотографировано
2. **Генерирует стандартные изображения товара** — создаёт изображения на основе определённого товара, сохраняет в папку проекта
3. **Возвращает результаты** — возвращает ссылки на изображения в облачном хранилище

**Шаг 2: Выберите изображение, добавьте маркетинговый текст**

Выберите понравившееся изображение на белом фоне и скажите:

```
现在我需要加上营销文字，小红书3:4尺寸，布局你可以自选，我要白底的。
```

Бот извлекает маркетинговый текст на основе распознанной информации о товаре и накладывает на изображение: главный заголовок, подзаголовок, теги функций, данные спецификаций — готовое изображение можно сразу размещать.

Хотите изменить текст или макет? Просто скажите:

```
把图片整体向右侧移动一点，产品主体居中，文字和排版不变
```

Изменение текста не требует повторной генерации изображения — **2 секунды** на повторный рендеринг и готово.

**Шаг 3: Получите ссылки на готовые материалы**

```
现在我需要你生成图片的链接，上传到云空间并返回给我链接，只需要终稿版本
```

Видео полного рабочего процесса 👇

### Вы получаете полную папку проекта

```
product-charger-0610/
├── raw/          ← Ваши оригинальные фото
├── product/      ← Сгенерированные стандартные изображения товара (белый фон, сцена)
├── index.html    ← Исходный файл макета текста (отредактируйте текст вручную, повторный экспорт за 2 секунды)
├── final/        ← Готовые изображения с текстом (готовы к размещению)
└── video/        ← 5-секундное видео-презентация
```

От фото до полного набора материалов — **менее 10 минут**. Раньше фотограф + дизайнер тратили на это целую неделю.

> Не используете Feishu? В терминале Qwen Code всё работает точно так же — просто отправьте путь к фото.

## Сценарий 3: ИИ пишет ваши маркетинговые тексты ✍️

Написание текстов — одна из самых трудоёмких задач. Для WeChat нужны длинные сторителлинговые статьи, для соцсетей — короткие яркие посты, для страниц товаров — описания функций — **один продукт, три канала, три разных стиля письма**. Каждый раз начинать с нуля, в итоге идеи заканчиваются, одни и те же фразы по кругу.

Теперь достаточно сообщить ИИ информацию о продукте и каналы распространения. WeChat получает сторителлинг, соцсети — лайфстайл-посты, страницы товаров — описания функций — один ввод, мультиканальный вывод.

**Пример: Сгенерировать тексты для портативного пауэрбанка для 3 каналов**

В Qwen Code введите:

```
帮我写一篇充电宝的推广文案。
产品名：PowerMax 迷你充电宝
卖点：1. 仅重 120g，比一颗鸡蛋还轻  2. 10000mAh 大容量  3. 支持 65W 快充，30 分钟充满 iPhone
目标人群：经常出差的商务人士
渠道：公众号、小红书、详情页
```

ИИ выдаёт три отдельные версии. Смотрите демо-видео:

### Не устраивает? Продолжайте диалог и публикуйте прямо в WeChat

Не довольны содержанием? Просто скажите Qwen Code, что нужно исправить, например:

```
公众号版本内容再轻松一些，直接帮我发不到公众号后台
```

Сообщите Qwen Code AppID и AppSecret вашего официального аккаунта WeChat, добавьте ваш IP в белый список на платформе разработчиков WeChat, и после однократной настройки вы сможете публиковать прямо в папку черновиков — никаких проблем с форматированием:

## Соедините все три сценария — получите полный конвейер запуска продукта

Вернёмся к начальному сценарию — новый продукт запускается, и вам нужен полный набор маркетинговых материалов.

Теперь рабочий процесс выглядит так:

```
Сделайте быстрое фото продукта
    ↓
Сценарий 1: Сгенерируйте сплэш-экраны, баннеры, постеры → 10 минут на все маркетинговые изображения
    ↓
Сценарий 2: Отретушируйте фото + сгенерируйте видео    → 20 минут на изображения и видео товара
    ↓
Сценарий 3: Сгенерируйте тексты WeChat/соцсети/детали  → 10 минут на тексты для всех каналов
```

**40 минут, один человек, полный набор материалов.** Раньше для этого нужны были дизайнер + фотограф + копирайтер, работающие вместе целую неделю.

## Как начать?

### Подготовка окружения

1. **Установить Qwen Code**: Следуйте [официальной документации](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)
2. **Установить Bailian CLI**: Следуйте [документации Bailian CLI](https://bailian.console.aliyun.com/cli)
3. **Запустить Qwen Code**: Сообщите Qwen Code ваш API-ключ и ваши потребности

### Установка пакета навыков

Откройте Qwen Code и скажите:

```
帮我安装 open1v-skills 技能包：git clone https://github.com/joeytoday/open1v-skills.git ~/.agents/skills/open1v-skills，然后安装依赖
```

После установки одна фраза запускает полный рабочий процесс:

- **Сгенерировать изображения товара**: `帮我生成一张充电宝的白底主图`
- **Написать маркетинговые тексты**: `帮我写一篇充电宝的推广文案，渠道：公众号、小红书、详情页`
- **Опубликовать в WeChat**: `把公众号文案发布到草稿箱`

Вы также можете настроить эти навыки под свои продукты.

### Подключение к Feishu (опционально)

Хотите использовать @бота в группе Feishu для всего вышеперечисленного? Спросите Qwen Code:

```
帮我配置飞书频道，我想在飞书群里 @机器人 生成商品图和文案，并一步步告诉我步骤
```

Qwen Code проведёт вас через настройку — 5 минут и готово.

## В заключение

ИИ не заменяет дизайнеров и копирайтеров. Он **даёт каждому возможность «выпустить первый вариант».**

С черновиком от ИИ дизайнеры могут доводить работу с 80 до 95 баллов вместо того, чтобы каждый раз начинать с нуля. Копирайтеры могут сосредоточиться на идеальных формулировках и эмоциональном отклике, а не мучиться с базовой структурой.

Это правильный подход к инструментам ИИ — **не замена, а ускорение**.
