Самое раздражающее при написании для официальных аккаунтов WeChat — это не сам контент, а поиск обложки после.

Каждый раз, когда я заканчиваю статью и хочу опубликовать, я мучаюсь с обложкой. До ИИ я искал на Unsplash (который теперь грузится мучительно медленно). Даже пробовал делать обложки в Keynote, вот так:

![](https://gw.alicdn.com/imgextra/i3/O1CN01B7V0w21BvFuqOBDiX_!!6000000000007-2-tps-2350-1000.png)

На самом деле всё просто — определённое изображение в сочетании с текстом заголовка. Раз так, ИИ вполне может справиться с этим сейчас. Но при прямой генерации ИИ часто выдаёт «технически правильное, но странное». Стили моего контента сильно различаются — есть эссе, есть инструментальные статьи. Для одних тем нужны тёмные тона, для технических инструментов — минималистичный абстрактный стиль, для культурной критики — редакторское журнальное ощущение. Каждый раз нужно заново продумывать композицию, цветовую палитру и промпты. Довольно утомительно.

Поэтому я просто **создал Skill, чтобы Qwen Code + Model Studio CLI делали это за меня**.

## Что он умеет

**Дайте ему заголовок статьи или тему, и он автоматически анализирует тональность, выбирает стиль, уточняет заголовок, строит промпты и вызывает Model Studio для генерации изображений.**

Четыре стиля, покрывающих мои основные сценарии в WeChat. Посмотрите — ещё многое можно улучшить, я постоянно итерирую:

![](https://gw.alicdn.com/imgextra/i2/O1CN01AXTaRR1O9hnhEBLKc_!!6000000001663-2-tps-2848-1214.png)

- **01 Плакат Bold**: Тёмный кинематографический фон + левый градиентный оверлей + крупный белый текст с одним ключевым словом, выделенным красным. Отлично для глубоких комментариев, исторических тем, социальных наблюдений.
- **02 Журнал**: Текст слева, изображение в рамке справа в двухколоночной вёрстке с бумажной текстурой. Отлично для культурной критики, заметок о книгах, портретных нарративов.
- **03 Минималистичная абстракция**: Полноформатная иллюстрация + монохромный цветовой оверлей для иммерсивного одноцветного ощущения. Отлично для технических блогов, презентаций продуктов, объяснения концепций.
- **04 Пиксель-арт**: 16-битная пиксельная графика + тёмно-фиолетовый оверлей + золотой пиксельный текст. Отлично для обзоров инструментов, лёгкого контента, ностальгических тем.

## Стек: Qwen Code + Model Studio CLI

Генерация изображений обеспечивается **командной утилитой Alibaba Cloud Model Studio**. Model Studio CLI — это мультимодальный ИИ-инструментарий в вашем терминале: чат, генерация изображений, генерация видео, синтез речи, поиск знаний — всё в командной строке. Здесь мы в основном используем `image generate`.

Почему именно это:

- **Qwen Code вызывает напрямую** — в Skill прописаны шаблоны команд, ИИ подставляет параметры и запускает, мне не нужно ничего делать
- **Есть бесплатная квота** — у qwen-image-2.0 есть бесплатный лимит использования, включите «остановить при исчерпании» чтобы избежать списаний
- **Точный контроль размера** — `--size '1344*572'` выдаёт соотношение 2.35:1 широкоформатное, рендеринг Playwright @2x экспортирует HD PNG 2688×1144

После установки моего навыка достаточно одного промпта:

```
/open1v-mpcover-gen Сгенерируй обложку WeChat на основе @твоя-статья.md, <можно указать стиль, использовать журнальный стиль>
```

## Как использовать

❶ Установить Qwen Code, см. https://qwenlm.github.io/qwen-code-docs/en/users/quickstart/

```
npm install -g @qwen-code/qwen-code@latest
```

❷ Получить API Key Model Studio: https://bailian.console.aliyun.com/cn-beijing?tab=model#/api-key

<Callout type="warning">
Перейдите на страницу использования модели и включите «Остановить при исчерпании бесплатной квоты», чтобы избежать неожиданных списаний: https://bailian.console.aliyun.com/cn-beijing?tab=model#/model-usage
</Callout>

❸ Установить навык обложек. Запустите Qwen Code и введите:

```
Help me install this skill: https://github.com/joeytoday/open1v-mpcover-gen
```

❹ Начать использовать: После установки просто скажите Qwen Code:

> Сделай мне обложку WeChat. Заголовок статьи — «Qwen Code × Model Studio CLI Руководство по генерации обложек»

ИИ проанализирует тональность контента, спросит какой стиль вы предпочитаете (или выберет автоматически), уточнит заголовок, построит английские промпты, вызовет Model Studio для генерации изображений и выдаст две альтернативы.

## Больше примеров использования

Указать стиль напрямую:

> Сделай минималистичную обложку, тема — безопасность API

Дать больше контекста:

Пакетная генерация:

> На этой неделе я написал три статьи, заголовки xxx / yyy / zzz, сделай по одной обложке для каждой

## Философия дизайна

Почему бы просто не попросить ИИ сгенерировать изображение с текстом заголовка? Потому что китайские иероглифы, сгенерированные ИИ, — это по сути абракадабра. Поэтому процесс разделён на два шага: **Model Studio генерирует фон → HTML накладывает текст → скриншот Playwright экспортирует PNG**. Изображение отвечает за атмосферу; текст точно контролируется CSS.

Преимущество — крайне низкая стоимость итерации: изменить заголовок — просто отредактировать текст в HTML и перерендерить (2 секунды), не нужно перегенерировать изображение (30 секунд). Оверлей слишком тяжёлый или лёгкий, цвета неправильные — всё решается изменением одной строки CSS.

Четыре стиля выбраны не наугад — они категоризированы по **ощущению читателей в ленте**:

- «Есть позиция» → Тёмный фон с жирным текстом, визуальный удар
- «Есть вкус» → Разделённые колонки с пустым пространством, редакторская эстетика
- «Очень ясно» → Один тон, минимальная когнитивная нагрузка
- «Весело» → Пиксель-арт ломает условности

Каждый стиль соответствует **паттерну распределения текст-изображение** — полный оверлей, разделённая колонка, цветовой оверлей — гарантируя читаемость текста независимо от фонового изображения.
