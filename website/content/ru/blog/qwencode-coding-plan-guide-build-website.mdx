Нужна презентация проекта для выпускной работы, но вы правите CSS до лысины? Для курсовых проектов нужны демо-видео, которые вы режете до трех часов ночи? Хотите поучаствовать в опенсорс-проектах, но не знаете, как сделать первый PR?

## Персональный сайт-портфолио за 5 минут

Нужна хорошая страница резюме, чтобы показать портфолио при поиске работы? Вместо бесконечной переделки шаблонов лучше пусть ИИ поможет вам сгенерировать её.

Я просто использовал команду `@` в Qwen Code, чтобы сослаться на своё резюме, и позволил ему создать личный сайт с красивыми анимациями (симуляция, ПС он действительно выглядел великолепно). Один лишь промпт:

```
/skills react-native-design build a personal resume website based on @resume-website/resume.md
```

## Участие в опенсорс-проектах начиная с первого PR

Хотите поучаствовать в опенсорсе, но не знаете с чего начать? Qwen Code может автоматизировать весь процесс.

Киньте ему ссылку на Issue, и он сделает следующее:

1. Автоматически поймет описание проблемы
2. Исследует структуру репозитория
3. Обнаружит местоположение бага
4. Предоставит исправляющий код
5. Самостоятельно отправит PR

<video class="mt-5" src="https://cloud.video.taobao.com/vod/JV-7WNvq-GF9ZRHhrr1igplZDtHaFeNNoJ412hgkKYI.mp4" controls></video>

Никакого ручного участия не требуется. Конечно, в первый раз рекомендую вручную проверить код, прежде чем доверять всё автоматизации. (Не спрашивайте, откуда знаю...)

Особенно полезно студентам, желающим набраться опыта в опенсорсе. Можно начать с простых исправлений багов и постепенно переходить к разработке более сложных функций.

(Искренне советую студентам технических специальностей попробовать — опыт в опенсорсе действительно помогает при трудоустройстве)

## Генерация демо-видео с помощью кода, прощай, софт для монтажа

Курсовые презентации, защита проектов, дорожные шоу конкурсов — да и предыдущие опенсорс-проекты тоже — всегда нуждались в демо-видео. Изучение Premiere и видеомонтаж требует кучу времени. (Кто занимался монтажом — знает!)

Основанный на Remotion (библиотека для создания видео с помощью React), Qwen Code способен генерировать профессиональные демо-видео с помощью кода.

Например, если нужно создать видео-презентацию продукта:

1. Установить соответствующие навыки Remotion
2. Написать сценарий и описание сториборда
3. Позволить ему сгенерировать видеокод
4. Выполнить рендеринг выходного файла

**Подходящие сценарии:**

- Видео защиты курсовых проектов

Преимущества Remotion заключаются в том, что вы можете создавать видео с помощью компонентов React, поддерживаются анимации, переходы, субтитры, аудиообработка, а также контроль версий (да, видео тоже можно управлять через Git — довольно круто).

## Как начать?

1. Получить ключ API из консоли BaiLian
2. Скачать Qwen Code и запустить с помощью `qwen`

```bash
# Linux/macOS
curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh | bash

# Windows
curl -fsSL -o %TEMP%\install-qwen.bat https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.bat && %TEMP%\install-qwen.bat
```

3. Введите `/auth`, выберите «Alibaba Cloud Coding Plan», вставьте ключ API
4. Используйте `/model` для выбора модели, затем начинайте!

У Qwen Code есть рынок навыков (похожий на рынок плагинов VS Code) с различными готовыми пакетами навыков. Используйте команду `/extensions`, чтобы выбрать и установить расширения, либо исследовать больше плагинов и навыков.

Не стремитесь сразу делать большие проекты. Начните с простых случаев, таких как:

- Попросите ИИ написать веб-страницу Hello World
- Пусть объяснит непонятный код
- Заставить его сгенерировать шаблон конспекта занятия

Освоившись, постепенно переходите к более сложным задачам.
