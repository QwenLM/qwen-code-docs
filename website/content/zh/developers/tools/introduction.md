# Qwen Code 工具

Qwen Code 内置了一系列工具，模型通过这些工具与本地环境交互、获取信息并执行操作。这些工具扩展了 CLI 的能力，使其不仅限于文本生成，还能协助完成各种任务。

## Qwen Code 工具概览

在 Qwen Code 中，工具是模型可以请求执行的特定函数或模块。例如，当你让模型”总结 `my_document.txt` 的内容”时，模型会识别出需要读取该文件，并请求执行 `read_file` 工具。

核心组件（`packages/core`）负责管理这些工具：向模型提供工具定义（schema）、在模型请求时执行工具，并将结果返回给模型，由模型进一步处理后呈现给用户。

这些工具提供以下能力：

- **访问本地信息：** 工具允许模型访问本地文件系统，读取文件内容、列出目录等。
- **执行命令：** 通过 `run_shell_command` 等工具，模型可以运行 shell 命令（附带适当的安全措施和用户确认）。
- **与 Web 交互：** 工具可以从 URL 获取内容。
- **执行操作：** 工具可以修改文件、写入新文件，或在系统上执行其他操作（通常也有相应的安全保障）。
- **提升响应质量：** 通过工具获取实时或特定的本地数据，响应可以更准确、更贴近你的实际上下文。

## 如何使用 Qwen Code 工具

向 CLI 提供提示词即可使用 Qwen Code 工具。整个流程如下：

1.  你向 CLI 提供提示词。
2.  CLI 将提示词发送给 core。
3.  core 连同你的提示词和对话历史，将可用工具列表及其描述/schema 发送给已配置的模型 API。
4.  模型分析你的请求。如果判断需要使用工具，其响应中会包含一个请求，要求以特定参数执行某个工具。
5.  core 接收该工具请求，进行验证，并在（通常是对敏感操作征得用户确认后）执行该工具。
6.  工具输出结果被发送回模型。
7.  模型根据工具输出生成最终答案，再经由 core 传回 CLI 并展示给你。

通常，CLI 中会显示工具调用状态的提示信息，告知工具是否执行成功。

## 安全性与确认机制

许多工具，尤其是可能修改文件系统或执行命令的工具（`write_file`、`edit`、`run_shell_command`），在设计上都考虑了安全性。Qwen Code 通常会：

- **要求确认：** 在执行潜在敏感操作前进行提示，向你展示即将执行的操作内容。
- **使用沙箱：** 所有工具均受沙箱强制限制（参见 [Qwen Code 沙箱](./sandbox.md)）。这意味着在沙箱环境下运行时，你希望使用的所有工具（包括 MCP server）必须在沙箱环境_内部_可用。例如，若要通过 `npx` 运行 MCP server，则 `npx` 可执行文件必须已安装在沙箱的 Docker 镜像中，或在 `sandbox-exec` 环境中可用。

在允许工具继续执行前，务必仔细查看确认提示。

## 深入了解 Qwen Code 工具

Qwen Code 的内置工具大致可分为以下几类：

- **[文件系统工具](./file-system.md)：** 用于与文件和目录交互（读取、写入、列出、搜索等）。
- **[Shell 工具](./shell.md)（`run_shell_command`）：** 用于执行 shell 命令。
- **[Monitor 工具](./monitor.md)（`monitor`）：** 用于运行长时间运行的 shell 命令，以后台任务通知的形式流式返回输出。
- **[Web Fetch 工具](./web-fetch.md)（`web_fetch`）：** 用于从 URL 获取内容。
- **[Todo Write 工具](./todo-write.md)（`todo_write`）：** 用于在编码会话中创建和管理结构化任务列表。
- **[Agent 工具](./task.md)（`agent`）：** 用于将复杂任务委托给专门的子 agent。
- **[Exit Plan Mode 工具](./exit-plan-mode.md)（`exit_plan_mode`）：** 用于退出计划模式并继续执行实现步骤。

此外，这些工具还整合了：

- **[MCP server](./mcp-server.md)**：MCP server 充当模型与本地环境或其他服务（如 API）之间的桥梁。
  - **[MCP 用户指南](../../users/features/mcp.md)**：在 Qwen Code 中配置和管理 MCP server
  - **[通过 MCP 进行 Web 搜索](./web-search.md)**：通过 MCP 接入 Web 搜索服务（百炼、Tavily、GLM）
- **[沙箱](./sandbox.md)**：沙箱将模型及其变更与你的环境隔离，以降低潜在风险。
