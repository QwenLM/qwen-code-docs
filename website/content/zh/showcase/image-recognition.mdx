## 概述

Qwen Code 的图片识别能力让它能够像人类一样理解和分析图片内容。无论是 UI 设计稿、错误信息截图、系统架构图、流程图还是代码片段截图，Qwen Code 都能准确识别其中的关键信息，并根据你的需求提供相应的帮助。这种多模态能力打破了传统代码助手的局限，让开发者可以通过更直观的方式与 AI 交互。从分析设计稿生成代码，到解读错误日志提供解决方案，图片识别极大地扩展了 AI 辅助编程的应用场景。

## 操作步骤

<Steps>
### 上传图片

在对话中点击上传按钮，或者直接将图片文件拖拽到对话窗口。支持 PNG、JPG、GIF、WebP 等常见图片格式。上传后，Qwen Code 会立即分析图片内容，并在对话中显示图片预览。你可以同时上传多张图片，Qwen Code 会综合分析所有图片的信息。

### 提问分析

基于图片内容，向 Qwen Code 提出你的问题。如果是 UI 设计稿，可以询问如何实现某个组件；如果是错误截图，可以请求分析错误原因和解决方案；如果是架构图，可以要求解释设计思路。Qwen Code 会深入理解图片细节，给出精准的分析和建议。

### 基于图片编码

对于设计稿类图片，可以直接要求 Qwen Code 生成对应的代码。它会分析设计稿的布局、样式、交互等要素，生成符合设计规范的代码。生成的代码会尽可能还原设计稿的效果，并提供必要的注释说明。你可以在生成的代码基础上继续优化和调整。
</Steps>

<Callout type="info">
图片识别支持 OCR 文字提取，即使是纯文字截图也能准确识别内容。对于复杂的架构图和流程图，建议分步提问，逐步深入理解。
</Callout>
