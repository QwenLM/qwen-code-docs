## 概述

在离线环境或需要使用自定义 Skill 时，通过文件夹安装是最灵活的方式。只需将 Skill 文件放入指定的目录，Qwen Code 就会自动识别并加载这些扩展。这种方式特别适合企业内网环境、需要定制的 Skill，或者想要分享自己开发的 Skill 给团队成员使用。整个过程简单直观，无需复杂的配置，让你的 AI 能力库可以随时扩展和定制。

## 操作步骤

<Steps>

### 下载或创建 Skill

首先获取你想要安装的 Skill 文件。你可以从官方仓库下载，也可以使用自己开发的 Skill。确保 Skill 文件结构完整，包含必要的配置文件和代码。

```bash
# 下载示例 Skill
git clone https://github.com/example/skill-repo.git
```

### 放入指定目录

将 Skill 文件复制到 Qwen Code 的 Skills 目录中。默认位置是 `~/.qwen/skills/`，如果目录不存在可以手动创建。

```bash
cp -r skill-repo ~/.qwen/skills/my-skill
```

### 重新加载 Skills

在 Qwen Code 对话中输入重新加载命令，让系统识别新添加的 Skill。Qwen Code 会扫描目录并加载所有可用的 Skill 扩展。

```bash
/skills reload
```

</Steps>

<Callout type="info">
文件夹安装方式支持自定义 Skill，你可以根据自己的需求开发专属的 AI 能力扩展。记得定期更新 Skill 文件以获得最新功能和修复。
</Callout>
