## 概述

百炼 Coding Plan 模式是 Qwen Code 的高级功能，专为复杂编程任务设计。通过智能的多模型协作，它能够将大型任务拆解为可执行的步骤，并自动规划最优的执行路径。无论是重构大型代码库还是实现复杂功能，Coding Plan 都能显著提升任务完成的质量和效率。

## 操作步骤

<Steps>

### 进入设置

在 Qwen Code 中进入设置界面，找到 Coding Plan 相关的配置选项。你可以通过命令或菜单导航来访问设置。确保你的 API Key 已正确配置，这是使用 Coding Plan 模式的前提条件。

### 选择 Coding Plan

在设置中启用百炼 Coding Plan 模式。这个模式会激活多模型协作机制，让不同的 AI 模型各司其职，共同完成复杂任务。启用后，系统会根据任务类型自动选择最合适的模型组合。

```bash
/settings enable coding-plan
```

### 开始复杂任务

描述你的编程任务，Coding Plan 会自动规划执行步骤。你可以详细说明任务的目标、约束条件和预期结果。AI 会分析任务需求，生成一个结构化的执行计划，包括每个步骤的具体操作和预期产出。

```bash
我需要重构这个项目的数据层，将数据库从 MySQL 迁移到 PostgreSQL，同时保持 API 接口不变
```

</Steps>

<Callout type="info">
Coding Plan 模式特别适合需要多步骤协作的复杂任务，如架构重构、功能迁移等。
</Callout>
