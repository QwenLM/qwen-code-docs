## 概述

Qwen Code 支持灵活的语言切换，你可以独立设置 UI 界面语言和 AI 输出语言。无论是中文、英文还是其他语言，都能找到适合的设置。多语言支持让你能够在熟悉的语言环境中工作，提高沟通效率和舒适度。

## 操作步骤

<Steps>

### 切换 UI 语言

使用 `/language ui` 命令切换界面语言，支持中文、英文等多种语言。UI 语言切换后，菜单、提示信息、帮助文档等都会使用你选择的语言显示。

```bash
/language ui zh-CN
```

### 切换输出语言

使用 `/language output` 命令切换 AI 的输出语言。AI 会使用你指定的语言回答问题、生成代码注释、提供解释等，让交流更加自然顺畅。

```bash
/language output zh-CN
```

### 混合语言使用

你还可以为 UI 和输出设置不同的语言，实现混合语言使用。例如，UI 使用中文，AI 输出使用英文，适合需要阅读英文技术文档的场景。

</Steps>

<Callout type="info">
语言设置会自动保存，下次启动时会使用你上次选择的语言。
</Callout>
