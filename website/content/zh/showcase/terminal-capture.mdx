## 概述

终端输出捕获是产品演示和文档编写的必备技能。通过这个 Skill，Qwen Code 能够自动执行测试命令、捕获终端输出信息，并生成精美的截图。无论是展示测试结果、记录错误日志，还是分享命令行操作，都能轻松实现专业的视觉效果。

## 操作步骤

<Steps>

### 安装 Skill

使用命令或对话方式安装终端捕获 Skill，这是社区贡献的实用工具。安装过程简单快速，安装后即可立即使用。Skill 会自动处理终端输出的捕获和格式化。

```bash
/skills terminal-capture
```

### 执行测试命令

让 AI 执行项目测试命令，它会运行并监控终端输出。你可以执行任何命令，包括测试、构建、部署等，AI 会实时捕获所有输出信息。

```bash
npm test
```

### 自动捕获输出

AI 会自动捕获终端输出信息，包括成功/失败状态、错误日志、性能数据等。捕获的内容会被格式化处理，确保清晰易读。你可以指定需要捕获的输出类型和格式。

### 生成截图分享

使用捕获的内容生成精美的截图，可以分享到社交媒体或文档中。截图支持多种样式和主题，你可以根据需要选择最适合的展示方式。

```bash
生成测试报告截图
```

</Steps>

<Callout type="info">
终端捕获 Skill 特别适合用于生成技术文档、教程和产品演示材料。
</Callout>
