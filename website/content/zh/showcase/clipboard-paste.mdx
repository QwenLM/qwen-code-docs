## 概述

剪贴板图片粘贴功能让你能够将任何截图直接粘贴到 Qwen Code 的对话窗口中，无需先保存文件再上传。无论是错误截图、UI 设计稿、代码片段截图还是任何其他图片，Qwen Code 都能即时理解其内容，并根据你的需求提供帮助。这种便捷的交互方式极大地提升了沟通效率，让你能够在遇到问题时快速获得 AI 的分析和建议。从截图到获得解决方案，只需一次粘贴操作。

## 操作步骤

<Steps>
### 复制图片

使用系统截图工具截取需要的图片，或者在图片编辑软件中复制图片。在 macOS 上使用 `Cmd+C`，在 Windows 或 Linux 上使用 `Ctrl+C` 将图片复制到剪贴板。确保剪贴板中只有一张图片，以获得最佳的识别效果。

### 粘贴到对话

在 Qwen Code 的对话输入框中，使用 `Cmd+V`（macOS）或 `Ctrl+V`（Windows/Linux）将图片粘贴。Qwen Code 会自动识别图片内容，并在对话中显示图片预览。此时你可以继续输入文字描述，或者直接询问关于图片的问题。

### 描述需求

根据图片内容，向 Qwen Code 描述你的需求。例如，如果是错误截图，可以询问错误原因和解决方案；如果是 UI 设计稿，可以请求生成对应的代码；如果是代码截图，可以要求解释或优化。Qwen Code 会基于图片内容和你的文字描述，提供精准的帮助。
</Steps>

<Callout type="info">
粘贴的图片会自动上传并进行 OCR 文字识别，即使是纯文字截图也能准确提取内容。支持 PNG、JPG、GIF 等常见图片格式。
</Callout>
