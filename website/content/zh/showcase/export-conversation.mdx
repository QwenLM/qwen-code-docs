## 概述

导出对话记录功能让你能够将重要的对话内容保存到本地，方便归档、分享和后续查阅。支持多种导出格式，包括 Markdown 适合文档编写，HTML 适合网页展示，JSON 适合程序处理。导出的内容包含完整的对话历史，包括用户提问、AI 回复、代码片段和文件操作等。这对于知识沉淀、团队协作、案例分享都非常有用。你可以定期导出重要的对话，构建自己的知识库，或者将解决方案分享给团队成员。

## 操作步骤

<Steps>
### 查看会话列表

使用 `/list` 命令查看所有历史会话。列表会显示每个会话的 ID、创建时间、主题摘要等信息，帮助你快速找到需要导出的对话。可以使用关键词搜索或按时间排序，精确定位目标会话。

```bash
/list
```

### 选择导出格式

确定要导出的会话后，使用 `/export` 命令并指定格式和会话 ID。支持三种格式：`markdown`、`html` 和 `json`。Markdown 格式适合在文档工具中使用，HTML 格式可以直接在浏览器中打开查看，JSON 格式适合程序化处理。

```bash
/export html <session-id>
```

### 分享和归档

导出的文件会保存到指定的目录。Markdown 和 HTML 文件可以直接分享给同事或在团队文档中使用。JSON 文件可以导入到其他工具进行二次处理。建议定期归档重要的对话，构建团队的知识库。导出的文件也可以作为备份，防止重要信息丢失。
</Steps>

<Callout type="info">
导出的 HTML 格式包含完整的样式，可以在任何浏览器中直接打开查看。JSON 格式保留所有元数据，适合进行数据分析和处理。
</Callout>
