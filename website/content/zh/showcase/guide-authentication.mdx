## 概述

认证登录是使用 Qwen Code 的第一步，通过简单的认证流程，你可以快速解锁全部功能。认证过程安全可靠，支持多种登录方式，确保你的账号和数据安全。完成认证后，你可以享受个性化的 AI 编程体验，包括自定义模型配置、历史记录保存等功能。

## 操作步骤

<Steps>

### 触发认证

启动 Qwen Code 后输入 `/auth` 命令，系统会自动启动认证流程。认证命令可以在命令行或 VS Code 扩展中使用，操作方式一致。系统会检测当前的认证状态，如果未登录则引导你完成登录。

```bash
/auth
```

### 浏览器登录

系统会自动打开浏览器，跳转到认证页面。在认证页面，你可以选择使用阿里云账号或其他支持的登录方式。登录过程快速便捷，只需几秒钟即可完成。

### 确认登录

登录成功后，浏览器会自动关闭或显示成功提示。回到 Qwen Code 界面，你会看到认证成功的消息，表示你已经成功登录并解锁全部功能。认证信息会自动保存，无需重复登录。

</Steps>

<Callout type="info">
认证信息安全存储在本地，无需每次重新登录。
</Callout>
