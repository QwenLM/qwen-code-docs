## 概述

Skills 是 Qwen Code 的扩展机制，让你能够通过自然语言安装和使用各种功能模块。无需手动下载和配置，只需告诉 AI 你需要什么 Skill，它就会自动搜索、下载并安装。这种即装即用的体验大大简化了功能扩展的流程。

## 操作步骤

<Steps>

### 请求安装 Skill

用自然语言告诉 AI 你想要安装某个 Skill。你可以描述 Skill 的功能或名称，AI 会理解你的需求并开始搜索。如果不确定 Skill 的确切名称，也可以描述它的功能，AI 会帮你找到合适的 Skill。

```bash
帮我安装 find-skills 这个 skill
```

### AI 自动执行

Qwen Code 会自动搜索并安装指定的 Skill。整个过程完全自动化，你只需要等待安装完成。AI 会处理所有依赖关系和配置，确保 Skill 能够正常工作。

```bash
Please first check if find-skills exists, if not, please use npx skills add https://github.com/vercel-labs/skills --skill find-skills to install this skill
```

### 使用已安装的 Skill

安装完成后即可在对话中使用该 Skill 的功能。你可以直接调用 Skill 提供的能力，无需额外的配置或设置。AI 会自动将你的请求路由到相应的 Skill。

```bash
现在帮我搜索一下有哪些适合 React 开发的 skills
```

</Steps>
