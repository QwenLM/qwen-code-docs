## 概述

想要创作视频但不会写代码？通过自然语言描述你的创意，Qwen Code 就能使用 Remotion Skill 生成视频代码。AI 会理解你的创意描述，自动生成 Remotion 项目代码，包括场景设置、动画效果、文字排版等。无论是产品宣传、社交媒体内容还是教学视频，都能快速生成专业的视频内容，让视频创作变得简单有趣。

## 操作步骤

<Steps>

### 安装 Remotion Skill

首先安装 Remotion 相关的 Skill，这个 Skill 包含了视频创作的最佳实践和模板。

```bash
npx skills add nicepkg/agent-skills@remotion-best-practices
```

### 描述你的创意

用自然语言详细描述你想要创作的视频内容，包括主题、风格、时长、关键场景等。

```bash
我想制作一个 30 秒的产品宣传视频，风格简洁现代，展示产品的主要功能
```

### 预览和渲染

Qwen Code 会根据你的描述生成 Remotion 项目代码。启动开发服务器预览效果，满意后渲染成最终视频。

```bash
npm run dev
```

</Steps>

<Callout type="info">
提示词方式适合快速原型和创意探索。你可以不断调整描述，让 AI 优化视频效果。生成的代码可以进一步手动编辑，实现更复杂的功能。
</Callout>
