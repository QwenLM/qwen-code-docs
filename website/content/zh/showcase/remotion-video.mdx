## 概述

Remotion 是一个强大的代码生成视频框架，Qwen Code 通过 Remotion Skill 让你能够用自然语言描述视频创意，自动生成专业的演示视频。无论是产品介绍、功能演示，还是品牌宣传，都能快速制作出高质量的视频内容。

## 操作步骤

<Steps>

### 安装 Remotion Skill

使用 npx skills 命令安装 Remotion Best Practices Skill，这是视频创作的核心工具。安装过程简单快速，安装后即可开始创作视频。

```bash
npx skills add nicepkg/agent-skills@remotion-best-practices
```

### 描述创意需求

用自然语言告诉 AI 你想制作什么类型的视频，包括风格、时长、内容等。你可以描述视频的主题、目标受众、关键信息等，AI 会根据你的描述生成视频脚本和代码。

```bash
帮我制作一个产品介绍视频，包含 logo 动画和功能演示，时长 30 秒
```

### 预览和调整

AI 会生成 Remotion 代码，在浏览器中预览效果并提出调整意见。你可以实时查看视频效果，对动画、转场、文字等进行调整，直到满意为止。

```bash
npm run dev
```

### 渲染导出

满意后使用 build 命令渲染最终视频，支持 MP4、WebM 等格式。渲染过程自动完成，生成的视频可以直接用于分享和发布。

```bash
npm run build
```

</Steps>

<Callout type="info">
Remotion 视频创作特别适合制作产品演示、教程视频和品牌宣传内容。
</Callout>
