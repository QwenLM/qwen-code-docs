## 概述

想要更直观地创作视频？通过网页界面配合 Remotion Skill，你可以可视化地编辑和预览视频内容。Qwen Code 会生成 Remotion 项目并启动网页编辑器，让你在浏览器中直接操作时间轴、调整动画、添加文字和图片。这种可视化方式让视频创作更加直观和高效，特别适合需要精细控制视频效果的场景。

## 操作步骤

<Steps>

### 安装 Remotion Skill

安装 Remotion 网页编辑器相关的 Skill，这会为你配置完整的可视化编辑环境。

```bash
npx skills add nicepkg/agent-skills@remotion-web-editor
```

### 打开网页编辑器

启动开发服务器，Qwen Code 会自动打开网页编辑器，让你在浏览器中进行视频创作。

```bash
npm run dev
```

### 可视化编辑视频

在网页编辑器中，你可以直观地添加场景、调整时间轴、设置动画效果、添加文字和图片，实时预览视频效果。

```bash
在编辑器中添加一个新的场景
```

### 导出最终视频

编辑完成后，使用渲染命令将视频导出为 MP4 或其他格式，可以直接分享或发布。

```bash
npm run build
```

</Steps>

<Callout type="info">
网页编辑器提供了丰富的可视化工具，让你无需编写代码就能创作专业视频。编辑器支持实时预览，所见即所得，大大提升了创作效率。
</Callout>
