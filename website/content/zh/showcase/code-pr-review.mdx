## 概述

代码审查是保证代码质量的重要环节，但往往耗时且容易遗漏问题。Qwen Code 可以帮你对 Pull Request 进行智能代码审查，自动分析代码变更、发现潜在 bug、检查代码规范，并提供详细的审查报告。AI 会深入理解代码逻辑，识别出可能的问题和改进建议，让代码审查更全面、更高效，同时减轻开发者的负担。

## 操作步骤

<Steps>

### 指定要审查的 PR

告诉 Qwen Code 你想要审查的 Pull Request，可以提供 PR 编号或链接。AI 会自动获取 PR 的详细信息和代码变更。

<Callout type="tip">
你可以使用相关的 pr review 技能来帮助你分析 PR，安装技能参考：[安装技能](./guide-skill-install.mdx)。

</Callout>


```bash
/skill pr-review 帮我 review 这个 PR：<pr 链接或编号>
```

### 运行测试和检查

Qwen Code 会分析代码变更，识别需要运行的测试用例，并执行相关测试来验证代码的正确性。

```bash
运行测试，检查这个 PR 是否通过所有测试用例
```

### 查看审查报告

审查完成后，Qwen Code 会生成详细的审查报告，包括发现的问题、潜在风险、改进建议等。你可以根据报告决定是否批准 PR。

```bash
查看审查报告，列出所有发现的问题
```

</Steps>

<Callout type="info">
Qwen Code 的代码审查不仅关注语法和规范，还会深入分析代码逻辑，识别潜在的性能问题、安全漏洞和设计缺陷，提供更全面的审查意见。
</Callout>
