## 操作步骤

<Steps>

### 全局安装 Qwen Code

在终端中执行安装命令，npm 会自动下载并安装 Qwen Code 到你的系统。这个过程只需要几秒钟，取决于你的网络速度。

```bash
npm install -g @qwen-code/qwen-code
```

### 启动 Qwen Code 应用

安装完成后，直接输入启动命令即可打开 Qwen Code。应用会自动初始化并准备好与你对话。

```bash
qwen
```

### 开始你的第一次对话

Qwen Code 启动后，你就可以开始与它对话了。试着输入你的第一个问题或需求，比如"帮我写一个 Hello World 程序"，体验 AI 编程的便捷。

</Steps>

<Callout type="info">
确保你的系统已安装 Node.js 和 npm，这是运行 Qwen Code 的必要环境。推荐使用 Node.js 16 或更高版本以获得最佳体验。
</Callout>
