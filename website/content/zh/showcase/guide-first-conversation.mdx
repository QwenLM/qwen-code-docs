## 操作步骤

<Steps>

### 启动 Qwen Code

在终端中输入 `qwen` 命令启动应用。首次启动时，系统会进行初始化配置，这个过程只需几秒钟。启动后，你会看到一个友好的命令行界面，准备开始与 AI 的对话。

```bash
qwen
```

### 发起对话

在输入框中输入你的问题，开始与 AI 交流。你可以从简单的自我介绍开始，比如询问 Qwen Code 是什么，或者它能够帮你做什么。AI 会用清晰易懂的语言回答你的问题，并引导你探索更多功能。

```bash
what is qwen code?
```

### 探索能力

尝试让 AI 帮你完成一些简单任务，比如解释代码、生成函数、回答技术问题等。通过实际操作，你会逐渐熟悉 Qwen Code 的各种能力，并发现它在不同场景下的应用价值。

```bash
帮我写一个 Python 函数，计算斐波那契数列
```

</Steps>

<Callout type="info">
Qwen Code 支持多轮对话，你可以基于之前的回答继续追问，深入探讨感兴趣的话题。
</Callout>
