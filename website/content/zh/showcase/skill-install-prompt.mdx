## 概述

想要快速扩展 Qwen Code 的能力？通过自然语言对话即可一键安装所需的 Skill，无需手动查找和配置。只需告诉 Qwen Code 你需要什么功能，它会自动识别并安装对应的 Skill 扩展，整个过程流畅自然，就像和一位智能助手对话一样简单。这种即装即用的方式，让你能够快速获得各种强大的 AI 能力，大幅提升工作效率。

## 操作步骤

<Steps>

### 描述你的需求

直接在对话中告诉 Qwen Code 你需要安装什么 Skill。例如，你可以说"帮我安装 find-skills"或"我需要能够查找技能的扩展"。Qwen Code 会理解你的意图并识别出对应的 Skill。

```bash
帮我安装 find-skills
```

### 自动安装 Skill

Qwen Code 会自动执行安装命令，从官方 Skill 库中下载并安装所需的 Skill 扩展。整个过程完全自动化，你无需手动输入任何命令或配置参数。

```bash
npx skills add qwen-code-examples@find-skills
```

### 确认并开始使用

安装完成后，Qwen Code 会确认 Skill 已成功加载，并告诉你如何使用它。现在你可以直接在对话中使用这个新安装的 Skill 功能了。

</Steps>

<Callout type="info">
提示词安装方式最适合在线环境，可以快速获取最新的 Skill 扩展。如果你在离线环境或需要使用自定义 Skill，可以考虑通过文件夹方式安装。
</Callout>
