## 概述

Headless 模式让你能够在无 GUI 环境下使用 Qwen Code，特别适合远程服务器、CI/CD 流水线和自动化脚本场景。通过命令行参数和管道输入，你可以将 Qwen Code 无缝集成到现有的工作流程中，实现自动化的 AI 辅助编程。

## 操作步骤

<Steps>

### 使用 prompt 模式

使用 `--p` 参数直接在命令行中传递提示词，Qwen Code 会返回答案后自动退出。这种方式适合一次性查询，可以轻松集成到脚本中。

```bash
qwen --p 'what is qwen code?'
```

### 管道输入

通过管道将其他命令的输出传递给 Qwen Code，实现自动化处理。你可以将日志文件、错误信息等直接发送给 AI 进行分析。

```bash
cat error.log | qwen --p '分析错误'
```

### 脚本集成

在 Shell 脚本中集成 Qwen Code，实现复杂的自动化工作流程。你可以根据 AI 的返回结果执行不同的操作，构建智能化的自动化脚本。

```bash
#!/bin/bash
result=$(qwen --p '检查代码是否有问题')
if [[ $result == *"有问题"* ]]; then
  echo "发现代码问题"
fi
```

</Steps>

<Callout type="info">
Headless 模式支持所有 Qwen Code 功能，包括代码生成、问题解答、文件操作等。
</Callout>
