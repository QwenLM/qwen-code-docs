## 概述

每周写产品更新周报是一项重复且耗时的工作。使用 Qwen Code 的定制技能，你可以让 AI 自动爬取本周的产品更新信息，按照标准模板生成结构化的周报。只需一行命令，AI 就会收集数据、分析内容、整理成文档，大幅提升你的工作效率。你可以专注于产品本身，让 AI 处理繁琐的文档工作。

## 操作步骤

<Steps>

### 配置数据源

首先告诉 Qwen Code 你需要收集哪些数据源的信息，比如 GitHub 仓库更新、产品发布日志、用户反馈等。AI 会根据你的需求配置相应的数据爬取规则。

```bash
帮我配置周报数据源，包括 GitHub 提交记录和产品更新日志
```

### 生成周报内容

配置完成后，输入生成命令，Qwen Code 会自动爬取本周的更新信息，并按照产品周报模板整理成结构化的文档。

```bash
生成本周的产品更新报告
```

### 导出和分享

生成的周报可以直接导出为 Markdown 格式，方便你进一步编辑或在团队中分享。你也可以让 AI 帮你调整格式和内容重点。

```bash
/export markdown weekly-report
```

</Steps>

<Callout type="info">
首次使用时需要配置数据源，配置一次后可以重复使用。你可以设置定时任务，让 Qwen Code 每周自动生成周报，彻底解放你的双手。
</Callout>
