本周发布 v0.10.0、v0.11.0 主要功能版本和 5 个 bugfix 版本。

- 发布两个主要功能版本：v0.10.0 与 v0.11.0，以及 5 个 bugfix 版本
- OAuth 认证，模型调用升级到 qwen3.5-plus，每天免费 1000 次调用
- 新增 Insight 洞察分析
- 剪贴板图片支持
- Terminal Capture 自动化验证功能并生产截图

## ✨ 新增特性

### Insight 洞察

官方内置的命令，可以深入分析本地 Session 会话记录，从而为用户提供个性化编程洞察和使用分析报告。

![](https://gw.alicdn.com/imgextra/i1/O1CN01fU1ONc1cQ6aODsN3J_!!6000000003594-2-tps-1698-954.png)

使用方式：输入 `/insight` 开始调用 LLM 进行并行分析

产出一份详尽的 HTML 报告，分为 7 个章节：
- **What You Work On** - 你所从事的工作
- **How You Use Qwen Code** - 你如何使用 Qwen Code
- **Impressive Things** - 令人印象深刻的事情
- **Where Things Go Wrong** - 事情出错的地方
- **Features to Try** - 值得尝试的功能
- **New Usage Patterns** - 新的使用模式
- **On the Horizon** - 未来展望

最后可导出一份用于社交媒体分享的图片。

![](https://gw.alicdn.com/imgextra/i4/O1CN01xJKhpq1WNSbVJyveF_!!6000000002776-2-tps-2400-1332.png)

### 剪贴板图片支持

跨平台支持：macOS / Windows / Linux。详见 PR [#1612](https://github.com/QwenLM/qwen-code/pull/1612)

功能特性：
- **图片捕获**：从系统剪贴板捕获图片
- **临时存储**：保存到 clipboard 临时目录
- **LRU 清理**：超过 100 张自动删除最旧 50 张
- **附件 UI**：`↑` 进入附件模式，`←/→` 导航，`Delete` 删除

![](https://gw.alicdn.com/imgextra/i2/O1CN01OsEDov1z4nJto1CfQ_!!6000000006661-2-tps-1694-956.png)

### Terminal-Capture

用途：CLI 视觉验证的自动化终端截图工具，自动验证功能然后产出演示截图。

架构：node-pty (伪终端) → xterm.js (渲染) → Playwright (截图)，详见 PR [#1840](https://github.com/QwenLM/qwen-code/pull/1840)

### MCP 工具进度更新

功能：MCP 工具支持实时进度通知

- **TUI 模式**：显示 `⏳ [50/100] Processing...`
- **SDK 模式**：新增 `tool_progress` 流事件类型

详见 PR [#1756](https://github.com/QwenLM/qwen-code/pull/1756)
