## 🚀 快速上手指南

### 准备工作

确保你已:

- ✅ 安装体验预览版本的 Qwen Code `npm i @qwen-code/qwen-code@0.14.0-preview.0 -g`
- ✅ 准备一个可接收消息的微信账号

> AI 时代下，谁还自己配置参数哇，你只需要 `npx skills add https://github.com/QwenLM/qwen-code-examples --skill weixin-channel-setup` 安装这个 weixin-channel-setup Skill，然后打开qwen，问它 `{'/'}skills weixin-channel-setup 请帮我配置`

如果你喜欢古法配置，不妨参考下面三步走

![三步配置流程图](https://img.alicdn.com/imgextra/i4/O1CN01aWJWAF1YwqRJ6ryra_!!6000000003124-0-tps-1376-768.jpg)

### 第一步:扫码连接

在终端中运行:

```bash
qwen channel configure-weixin
```

终端中有「QR code URL」，复制链接在浏览器中打开，打开后用微信扫描。

- 可能需要提醒你安装微信
  扫码成功后,你会看到:

```
Connected to WeChat successfully!
Credentials saved. You can now start a weixin channel with:
  qwen channel start <name>
```

此时,你的微信凭证已安全保存到本地。

### 第二步:配置通道

编辑 `~/.qwen/settings.json`,添加以下配置:

```json
"channels": {
    "my-weixin": {
      "type": "weixin",
      "senderPolicy": "open",
      "sessionScope": "user",
      "model": "qwen3.5-plus",
      "instructions": "You are a concise coding assistant responding via WeChat. Keep responses under 500 characters. Use plain text only."
    }
  }
```

### 第三步:启动并测试

```bash
qwen channel start my-weixin
```

看到 `[Channel] "my-weixin" is running.` 提示后,打开微信:

1. 找到你的机器人助手(就是你刚才扫码的那个)
2. 发送消息:"你好,帮我写一个 Python 的 Hello World"
3. 等待回复...

🎉 **恭喜你!现在开始就可以用微信写代码了!**
