本周发布 **v0.14.3**、**v0.14.4**、**v0.14.5** 三个功能版本。

- v0.14.3 是核心体验优化版本：`/review` 命令全面重写，支持确定性分析、自动修复和安全加固；`/statusline` 命令支持自定义状态栏显示；`/settings` 对话框新增模型选择器和设置描述；CJK 输入优化支持按词粒度移动光标；French 多语言支持上线。

## ✨ 新增特性

### /review 代码审查：确定性分析 + 自动修复 + 安全加固

`/review` 命令经过全面重写，支持确定性分析、自动修复建议和安全漏洞检测。

**你能用它做什么：**

- 对当前改动执行结构化代码审查，自动识别潜在 bug
- 一键生成修复建议，直接应用到 PR 中
- 扫描常见安全风险（如硬编码密钥、注入风险）

详见 PR [#2932](https://github.com/QwenLM/qwen-code/pull/2932)

### 可定制状态栏：/statusline 命令自由组合信息

通过 `/statusline` 命令可以自定义终端状态栏显示内容，支持显示模型名称、Token 用量、沙箱状态等。

**你能用它做什么：**

- 实时查看当前使用的模型和 Token 消耗
- 显示沙箱开关状态、权限模式等关键信息
- 按个人偏好精简或扩展显示内容

详见 PR [#2923](https://github.com/QwenLM/qwen-code/pull/2923)

### /settings 界面：Fast Model 模型选择器 + 设置描述展示

`/settings` 对话框中的 Fast Model 从文本输入框改为下拉模型选择器；同时每个活跃设置项下方会显示说明文字。

**你能用它做什么：**

- 直接在下拉列表中选择 Fast Model，无需手动输入模型名
- 查看每个设置项的作用说明，减少配置困惑

详见 PR [#3120](https://github.com/QwenLM/qwen-code/pull/3120)

### 输入法优化：CJK 分词 + Ctrl+ 方向键跳转

为中文、日文、韩文用户优化了终端输入体验，支持按词粒度移动光标，配合 Ctrl+ 方向键可以快速跳转。

**你能用它做什么：**

- 按词（而非逐字符）移动光标，编辑长句更高效
- 用 Ctrl+ 方向键在输入框中快速跳转，减少按键次数

详见 PR [#2942](https://github.com/QwenLM/qwen-code/pull/2942)

### 输入队列编辑：↑ 弹出待发送消息编辑

**你能用它做什么：**

- 发送前发现错误，按 ↑ 直接修改队列中的消息
- 多条消息排队时，逐条弹出修改，提升编辑效率
