本周发布 v0.11.1 正式版，包含多项稳定性修复和功能增强。与此同时，v0.12.0 开发分支持续推进，带来几个值得关注的新特性。

- 发布正式版：v0.11.1
- v0.12.0 开发中，已合并多个新功能 PR

## ✨ 新增特性

### HTML 导出：工具调用内容查看器

导出会话为 HTML 文件后，**点击 Shell 工具调用的 IN / OUT 标签，现在会弹出一个模态窗口显示完整内容**。此前这两个标签点击没有任何响应。详见 PR [#2085](https://github.com/QwenLM/qwen-code/pull/2085)

![](https://gw.alicdn.com/imgextra/i4/O1CN01jLoKVP29zihmemlWK_!!6000000008139-2-tps-1610-858.png)

### 终端流式截图与 GIF 生成

terminal-capture 工具新增流式捕获模式，可以在命令执行过程中按固定间隔截图，适合录制进度条、spinner 动画等动态终端输出。捕获完成后自动调用 ffmpeg 合成为循环 GIF。详见 PR [#2116](https://github.com/QwenLM/qwen-code/pull/2116)

支持配置截图间隔、延迟启动时间，以及输出稳定后自动停止。未安装 ffmpeg 时会跳过 GIF 生成并继续执行，不影响截图流程。

之前只能截静态图，**现在可以录制完整的命令执行过程**，适合做功能演示和教程。

### GitHub 工作流内置命令

新增四个内置自定义命令，覆盖常见 GitHub 操作场景，通过 `/qc:<命令名>` 调用：详见 PR [#2117](https://github.com/QwenLM/qwen-code/pull/2117)

- **`/qc:code-review`**：拉取 PR 详情和 diff，生成覆盖正确性、风格、性能、安全性的代码审查报告
- **`/qc:commit`**：分析暂存区变更，生成符合 Conventional Commits 规范的提交信息，自动处理分支（避免直接提交到 main）
- **`/qc:create-issue`**：根据代码库上下文起草 GitHub Issue，自动选择 feature request 或 bug report 模板
- **`/qc:create-pr`**：创建结构完整的 Pull Request，遵循项目 PR 模板

![](https://gw.alicdn.com/imgextra/i1/O1CN01yjkkmf1PU94inB5NU_!!6000000001843-1-tps-860-584.gif)

### AGENTS.md 作为默认上下文文件

项目根目录的 `AGENTS.md` 现在会被自动加载为上下文文件，与 `GEMINI.md`、`CLAUDE.md` 等行为一致。在多 Agent 框架下工作时不需要额外配置。详见 PR [#2018](https://github.com/QwenLM/qwen-code/pull/2018)

![](https://gw.alicdn.com/imgextra/i1/O1CN018tD4ZB1HuEiAze39i_!!6000000000817-2-tps-1902-1144.png)

### Ctrl+Y 重试失败请求

请求失败后，按 `Ctrl+Y` 可以直接重试，不需要重新输入内容。详见 PR [#2011](https://github.com/QwenLM/qwen-code/pull/2011)

![](https://gw.alicdn.com/imgextra/i1/O1CN01YtZAzm1ShNMbf8YTt_!!6000000002278-2-tps-1694-956.png)

### 认证对话框布局优化

登录对话框重新设计为三选项布局，选项之间的区分更清晰。详见 PR [#2030](https://github.com/QwenLM/qwen-code/pull/2030)

![](https://gw.alicdn.com/imgextra/i2/O1CN011eGoRs1Pf8a4hXQ0K_!!6000000001867-2-tps-1700-952.png)

### 代码高亮 Tab 宽度支持

代码高亮组件新增 `tabWidth` 配置，Tab 字符会被替换为对应数量的空格，解决代码块中 Tab 显示宽度不一致的问题。详见 PR [#2077](https://github.com/QwenLM/qwen-code/pull/2077)

![](https://gw.alicdn.com/imgextra/i4/O1CN016C78pF1aCFN1nJhiI_!!6000000003293-2-tps-2868-1346.png)
