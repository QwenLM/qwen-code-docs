# 上下文提示

Qwen Code 内置了上下文提示系统，帮助你发现新功能并随时了解会话状态。

## 启动提示

每次启动 Qwen Code 时，标题区域都会显示一条提示。提示会优先按优先级筛选，然后在相同优先级的提示中使用 LRU（最近最少使用）策略跨会话轮换，确保你每次看到的提示都不同。

新用户在前几次会话中会看到侧重于引导的提示：

| 会话次数 | 提示示例                                             |
| -------- | ---------------------------------------------------- |
| < 5      | 斜杠命令（`/`）、Tab 自动补全                        |
| < 10     | `QWEN.md` 项目上下文、`--continue` / `--resume`      |
| < 15     | 带 `!` 前缀的 Shell 命令                             |

之后，提示将轮换展示 `/compress`、`/approval-mode`、`/insight`、`/btw` 等通用功能。

## 响应后提示

在对话过程中，Qwen Code 会监控上下文窗口使用情况，并在需要操作时显示提示：

| 上下文使用率 | 触发条件                       | 提示内容                                          |
| ------------- | ------------------------------ | ------------------------------------------------- |
| 50-80%        | 会话中进行数次提示后           | 建议使用 `/compress` 释放上下文空间               |
| 80-95%        | —                              | 警告上下文即将占满                                |
| >= 95%        | —                              | 紧急：立即运行 `/compress` 或使用 `/new` 继续对话 |

响应后提示设有单条提示的冷却时间，以避免重复显示。

## 提示历史

提示显示历史记录会持久化保存在 `~/.qwen/tip_history.json` 中。该文件用于记录：

- 会话次数（用于为新用户选择提示）
- 已显示的提示及其显示时间（用于 LRU 轮换和冷却控制）

你可以安全地删除此文件以重置提示历史。

## 禁用提示

要隐藏所有提示（包括启动提示和响应后提示），请在 `~/.qwen/settings.json` 中将 `ui.hideTips` 设置为 `true`：

```json
{
  "ui": {
    "hideTips": true
  }
}
```

你也可以通过 `/settings` 命令打开设置对话框来切换此选项。

启用屏幕阅读器模式时，提示也会自动隐藏。
