# 上下文提示

Qwen Code 包含一个上下文提示系统，帮助您发现功能并随时了解会话状态。

## 启动提示

每次启动 Qwen Code 时，标题区域会显示一条提示。提示会优先按优先级选择，然后在相同优先级的提示之间，使用 LRU（最近最少使用）调度算法在多次会话中轮换，因此每次您都会看到不同的提示。

新用户在前几次会话中会看到引导性提示：

| 会话次数 | 示例提示                                   |
| -------- | ---------------------------------------- |
| 小于 5   | 斜杠命令（`/`）、Tab 自动补全                |
| 小于 10  | `QWEN.md` 项目上下文、`--continue` / `--resume` |
| 小于 15  | 带 `!` 前缀的 Shell 命令                    |

之后，提示会在常规功能间轮换，例如 `/compress`、`/approval-mode`、`/insight`、`/btw` 等。

## 响应后提示

在对话过程中，Qwen Code 会监控您的上下文窗口使用情况，并在可能需要操作时显示提示：

| 上下文使用率 | 条件                          | 提示                                                |
| ------------ | ----------------------------- | -------------------------------------------------- |
| 50-80%       | 会话中经过几次提示后           | 建议运行 `/compress` 以释放上下文                     |
| 80-95%       | —                             | 警告上下文即将占满                                   |
| >= 95%       | —                             | 紧急：立即运行 `/compress` 或 `/new` 以继续           |

响应后提示每条都有自己的冷却时间，避免重复显示。

## 提示历史

提示显示历史记录保存在 `~/.qwen/tip_history.json` 文件中。该文件跟踪：

- 会话次数（用于新用户提示选择）
- 哪些提示已经显示以及显示时间（用于 LRU 轮换和冷却）

您可以安全地删除此文件以重置提示历史。

## 禁用提示

要隐藏所有提示（包括启动提示和响应后提示），请在 `~/.qwen/settings.json` 中将 `ui.hideTips` 设置为 `true`：

```json
{
  "ui": {
    "hideTips": true
  }
}
```

您也可以通过 `/settings` 命令在设置对话框中切换此选项。

当屏幕阅读器模式启用时，提示也会自动隐藏。
