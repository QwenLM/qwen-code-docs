# Qwen Code：服务条款与隐私声明

Qwen Code 是由 Qwen Code 团队维护的一款开源 AI 编程助手工具。本文档说明了在使用 Qwen Code 的身份验证方法及 AI 模型服务时所适用的服务条款与隐私政策。

## 如何确定您的认证方式

Qwen Code 支持三种认证方式以访问 AI 模型。您的认证方式将决定适用于您使用的相关服务条款与隐私政策：

1. **Qwen OAuth** — 使用您的 qwen.ai 账户登录（享有免费的每日配额）
2. **阿里云 Coding 计划** — 使用来自阿里云的 API 密钥
3. **API 密钥** — 自行提供 API 密钥

针对每种认证方式，由于底层服务提供商不同，所适用的服务条款和隐私声明也可能不同。

| 认证方式                  | 提供方         | 服务条款                                                                 | 隐私声明                                                                 |
| :------------------------ | :------------- | :----------------------------------------------------------------------- | :----------------------------------------------------------------------- |
| Qwen OAuth                | Qwen AI        | [Qwen 服务条款](https://qwen.ai/termsservice)                            | [Qwen 隐私政策](https://qwen.ai/privacypolicy)                           |
| 阿里云 Coding 计划        | 阿里云         | 详见[下方说明](#2-if-you-are-using-alibaba-cloud-coding-plan)           | 详见[下方说明](#2-if-you-are-using-alibaba-cloud-coding-plan)           |
| API 密钥                  | 各类提供商     | 取决于您选择的 API 提供商（如 OpenAI、Anthropic 等）                     | 取决于您选择的 API 提供商                                                |

## 1. 如果你使用 Qwen OAuth 认证

当你使用 qwen.ai 账户进行身份验证时，以下《服务条款》和《隐私声明》文档适用：

- **服务条款**：你的使用受 [Qwen 服务条款](https://qwen.ai/termsservice) 约束。  
- **隐私声明**：关于你数据的收集与使用，请参阅 [Qwen 隐私政策](https://qwen.ai/privacypolicy)。

有关身份验证配置、配额及支持功能的详细信息，请参阅 [身份验证配置](../configuration/settings)。

## 2. 如果你使用阿里云百炼计划

当你使用阿里云提供的 API 密钥进行身份验证时，适用阿里云的服务条款和隐私声明。

阿里云百炼计划在以下两个区域提供：

- **阿里云百炼（aliyun.com）** — [bailian.console.aliyun.com](https://bailian.console.aliyun.com)  
- **阿里云（alibabacloud.com）** — [bailian.console.alibabacloud.com](https://bailian.console.alibabacloud.com)

> [!important]
>
> 使用阿里云百炼计划时，你须遵守阿里云的服务条款与隐私政策。请查阅其相关文档，了解有关数据使用、存储期限及隐私实践的具体细节。

## 3. 如果你使用自己的 API 密钥

当你使用其他服务商提供的 API 密钥进行身份验证时，适用的服务条款和隐私声明取决于你选择的服务商。

> [!important]
>
> 使用你自己的 API 密钥时，你需遵守所选 API 服务商的条款与隐私政策，而非 Qwen Code 的条款。请查阅服务商的文档，了解有关数据使用、存储期限及隐私实践的具体细节。

Qwen Code 支持多种兼容 OpenAI 的服务商。有关详细信息，请参阅你所用服务商的服务条款与隐私政策。

## 使用统计与遥测数据

Qwen Code 可能会收集匿名的使用统计信息及[遥测数据](../../developers/development/telemetry)，以提升用户体验与产品质量。该数据收集为可选行为，可通过配置设置进行控制。

### 收集哪些数据

启用后，Qwen Code 可能会收集以下数据：

- 匿名使用统计信息（执行的命令、性能指标）
- 错误报告和崩溃数据
- 功能使用模式

### 按认证方式划分的数据收集

- **Qwen OAuth：** 使用统计信息受 Qwen 隐私政策约束。您可通过 Qwen Code 的配置设置选择退出。
- **阿里云 Coding 计划：** 使用统计信息受阿里云隐私政策约束。您可通过 Qwen Code 的配置设置选择退出。
- **API Key：** 除您所选 API 提供商自行收集的数据外，Qwen Code 不额外收集任何数据。

## 常见问题（FAQ）

### 1. 我的代码（包括提示词和回答）是否会被用于训练 AI 模型？

您的代码（包括提示词和回答）是否会被用于训练 AI 模型，取决于您所使用的身份验证方式以及具体的 AI 服务提供商：

- **Qwen OAuth**：数据使用受 [Qwen 隐私政策](https://qwen.ai/privacy) 约束。请查阅其政策以了解有关数据收集及模型训练实践的具体细节。

- **阿里云编码计划**：数据使用受阿里云隐私政策约束。请查阅其政策以了解有关数据收集及模型训练实践的具体细节。

- **API Key**：数据使用完全取决于您选择的 API 提供商。各提供商均有各自的数据使用政策。请务必查阅您所用提供商的隐私政策及服务条款。

**重要提示**：Qwen Code 本身不会将您的提示词、代码或响应用于模型训练。任何用于训练目的的数据使用均受您所认证的 AI 服务提供商相关政策约束。

### 2. 什么是使用统计信息？“退出”控制项的作用是什么？

**使用统计信息** 设置用于控制 Qwen Code 是否可选地收集数据，以提升用户体验和产品质量。

启用后，Qwen Code 可能会收集以下信息：

- 匿名遥测数据（执行的命令、性能指标、功能使用情况）
- 错误报告与崩溃数据
- 通用使用模式

**Qwen Code 不会收集以下内容：**

- 您的代码内容
- 发送给 AI 模型的提示词（prompts）
- AI 模型返回的响应
- 个人身份信息

“使用统计信息”设置仅控制 Qwen Code 自身的数据收集行为。它不会影响您所选用的 AI 服务提供商（如 Qwen、OpenAI 等）依据其自身隐私政策可能收集的数据。

### 3. 如何在不同认证方式之间切换？

你可随时在 Qwen OAuth、阿里云 Coding 计划和你自己的 API key 之间切换：

1. **启动时**：在提示时选择你偏好的认证方式  
2. **在 CLI 中**：使用 `/auth` 命令重新配置你的认证方式  
3. **环境变量**：设置 `.env` 文件以实现 API key 的自动认证  

详细操作说明请参阅 [认证配置](../configuration/settings#environment-variables-for-api-access) 文档。
