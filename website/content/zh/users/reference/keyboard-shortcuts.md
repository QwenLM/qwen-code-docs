# Qwen Code 键盘快捷键

本文档列出了 Qwen Code 中可用的键盘快捷键。

## 通用

| 快捷键                         | 说明                                                                                                                   |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | 关闭对话框和建议。                                                                                                     |
| `Ctrl+C`                       | 取消当前请求并清空输入。按两次可退出应用。                                                                             |
| `Ctrl+D`                       | 输入为空时退出应用。按两次确认。                                                                                       |
| `Ctrl+L`                       | 清屏。                                                                                                                 |
| `Ctrl+O`                       | 切换紧凑模式（隐藏/显示工具输出和思考过程）。                                                                          |
| `Ctrl+S`                       | 允许长响应完整打印，禁用截断。使用终端的回滚功能查看完整输出。                                                         |
| `Ctrl+T`                       | 切换工具描述的显示。                                                                                                   |
| `Shift+Tab` (`Tab` on Windows) | 循环切换审批模式（`plan` → `default` → `auto-edit` → `yolo`）                                                          |

## 输入提示

| 快捷键                                             | 说明                                                                                                                           |
| -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `!`                                                | 输入为空时切换 Shell 模式。                                                                                                    |
| `?`                                                | 输入为空时切换键盘快捷键显示。                                                                                                 |
| `\` (at end of line) + `Enter`                     | 插入换行符。                                                                                                                   |
| `Down Arrow`                                       | 向下浏览输入历史。                                                                                                             |
| `Enter`                                            | 提交当前提示。                                                                                                                 |
| `Meta+Delete` / `Ctrl+Delete`                      | 删除光标右侧的单词。                                                                                                           |
| `Tab`                                              | 如果存在建议，自动补全当前建议。                                                                                               |
| `Up Arrow`                                         | 向上浏览输入历史。                                                                                                             |
| `Ctrl+A` / `Home`                                  | 将光标移至行首。                                                                                                               |
| `Ctrl+B` / `Left Arrow`                            | 将光标向左移动一个字符。                                                                                                       |
| `Ctrl+C`                                           | 清空输入提示。                                                                                                                 |
| `Esc` (double press)                               | 清空输入提示。                                                                                                                 |
| `Ctrl+D` / `Delete`                                | 删除光标右侧的字符。                                                                                                           |
| `Ctrl+E` / `End`                                   | 将光标移至行尾。                                                                                                               |
| `Ctrl+F` / `Right Arrow`                           | 将光标向右移动一个字符。                                                                                                       |
| `Ctrl+H` / `Backspace`                             | 删除光标左侧的字符。                                                                                                           |
| `Ctrl+K`                                           | 删除从光标到行尾的内容。                                                                                                       |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`   | 将光标向左移动一个单词。                                                                                                       |
| `Ctrl+N`                                           | 向下浏览输入历史。                                                                                                             |
| `Ctrl+P`                                           | 向上浏览输入历史。                                                                                                             |
| `Ctrl+R`                                           | 反向搜索输入/Shell 历史。                                                                                                      |
| `Ctrl+Y`                                           | 重试上一次失败的请求。                                                                                                         |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F` | 将光标向右移动一个单词。                                                                                                       |
| `Ctrl+U`                                           | 删除从光标到行首的内容。                                                                                                       |
| `Ctrl+V` (Windows: `Alt+V`)                        | 粘贴剪贴板内容。如果剪贴板包含图片，将保存图片并在提示中插入其引用。                                                           |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`     | 删除光标左侧的单词。                                                                                                           |
| `Ctrl+X` / `Meta+Enter`                            | 在外部编辑器中打开当前输入。                                                                                                   |

## 建议

| 快捷键          | 说明                               |
| --------------- | ---------------------------------- |
| `Down Arrow`    | 向下浏览建议。                     |
| `Tab` / `Enter` | 接受选中的建议。                   |
| `Up Arrow`      | 向上浏览建议。                     |

## 单选按钮选择

| 快捷键             | 说明                                                                                                     |
| ------------------ | -------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` | 向下移动选择。                                                                                           |
| `Enter`            | 确认选择。                                                                                               |
| `Up Arrow` / `k`   | 向上移动选择。                                                                                           |
| `1-9`              | 通过数字选择对应项。                                                                                     |
| (multi-digit)      | 对于编号大于 9 的项，快速连续按下对应数字即可选择。                                                      |

## IDE 集成

| 快捷键   | 说明                             |
| -------- | -------------------------------- |
| `Ctrl+G` | 查看 IDE 传递给 CLI 的上下文。   |
