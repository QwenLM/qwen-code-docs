# Qwen Code 键盘快捷键

本文档列出了 Qwen Code 中可用的键盘快捷键。

## 通用

| 快捷键                         | 描述                                                                                                                                                                                                                                                                                                             |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | 关闭对话框和建议。                                                                                                                                                                                                                                                                                                |
| `Ctrl+C`                       | 取消正在进行的请求并清空输入。按两次退出应用程序。                                                                                                                                                                                                                                                                |
| `Ctrl+D`                       | 如果输入为空则退出应用程序。按两次确认。                                                                                                                                                                                                                                                                         |
| `Ctrl+L`                       | 清屏。                                                                                                                                                                                                                                                                                                           |
| `Ctrl+O`                       | 切换紧凑模式（隐藏/显示工具输出和思考过程）。                                                                                                                                                                                                                                                                    |
| `Ctrl+S`                       | 允许长响应完全打印，禁用截断。使用终端的回滚查看完整输出。                                                                                                                                                                                                                                                       |
| `Ctrl+T`                       | 切换工具描述的显示。                                                                                                                                                                                                                                                                                              |
| `Ctrl+B`                       | 当前台 shell 命令正在运行时：将其提升为后台任务。子进程继续运行，智能体的回合解除阻塞，shell 出现在 `/tasks` 和后台任务对话框中。当没有 shell 执行时无操作——`Ctrl+B` 随后会回退到其提示区域绑定（光标左移）。                                                                                           |
| `Alt/Option+M`                 | 在富渲染预览和原始/源码模式之间切换 Markdown 输出。在 macOS 上，终端必须将 Option 作为 Meta 发送。                                                                                                                                                                                                               |
| `Shift+Tab`（Windows 上为 `Tab`） | 循环切换审批模式（`plan` → `default` → `auto-edit` → `auto` → `yolo`）                                                                                                                                                                                                                                           |

## 建议

| 快捷键                 | 描述                        |
| ---------------------- | --------------------------- |
| `下箭头` / `Ctrl+N`      | 向下浏览建议。              |
| `Tab` / `Enter`        | 接受选中的建议。            |
| `上箭头` / `Ctrl+P`      | 向上浏览建议。              |

## 单选按钮选择

| 快捷键                          | 描述                                                                                 |
| ------------------------------- | ------------------------------------------------------------------------------------ |
| `下箭头` / `j` / `Ctrl+N`         | 向下移动选择。                                                                       |
| `Enter`                         | 确认选择。                                                                           |
| `上箭头` / `k` / `Ctrl+P`         | 向上移动选择。                                                                       |
| `1-9`                           | 通过编号选择项目。                                                                   |
| （多位数字）                    | 对于编号大于 9 的项目，快速连续按下数字以选择对应项目。                              |

## 历史回滚

仅在启用 `ui.useTerminalBuffer` 时生效（设置 → UI → 虚拟化历史）。在该模式下，对话历史会在应用内的视口中渲染，而不是宿主终端的回滚区，因此以下按键将取代终端的原生滚动。

| 快捷键        | 描述                                          |
| ------------- | --------------------------------------------- |
| `Shift+上箭头` | 向上滚动一行历史记录。                         |
| `Shift+下箭头` | 向下滚动一行历史记录。                         |
| `PgUp`        | 向上滚动一页历史记录（视口高度）。             |
| `PgDn`        | 向下滚动一页历史记录（视口高度）。             |
| `Ctrl+Home`   | 跳转到对话顶部。                               |
| `Ctrl+End`    | 跳转到底部（并重新激活实时自动跟随）。          |
| **鼠标滚轮**  | 滚动历史记录（每次滚动 3 行）。                |

当 `ui.useTerminalBuffer` 开启时，终端会将鼠标事件转发给 qwen-code，因此滚轮可以驱动应用内视口。作为副作用，**原生的点击拖拽文本选择会被程序拦截**——拖拽时按住 `Shift`（在 macOS Terminal / iTerm 上为 `Option`）可以绕过鼠标捕获，以常规方式选择文本。

### tmux 触控板滚动

在 tmux 内部，某些终端会在 qwen-code 接收到触控板或滚轮手势之前就将其转换为普通的 `上箭头` 和 `下箭头` 序列。这些字节与真实的方向键按压完全相同，因此 qwen-code 无法判断您是想滚动视口还是浏览提示历史。

如果 tmux 中的触控板滚动改变了提示历史，请启用 `ui.useTerminalBuffer`；然后使用 `Shift+上箭头` / `Shift+下箭头`，或者当 tmux 向应用转发滚轮事件时使用鼠标滚轮。如果您更倾向于使用主机回滚，请调整 tmux 中滚轮事件的鼠标绑定。
