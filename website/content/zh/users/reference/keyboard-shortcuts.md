# Qwen Code 键盘快捷键

本文档列出了 Qwen Code 中可用的键盘快捷键。

## 通用

| 快捷键                       | 描述                                                                                                                                                                                                                                                                                               |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | 关闭对话框和建议。                                                                                                                                                                                                                                                                            |
| `Ctrl+C`                       | 取消正在进行的请求并清空输入。按两次退出应用程序。                                                                                                                                                                                                                                      |
| `Ctrl+D`                       | 输入为空时退出应用程序。按两次确认。                                                                                                                                                                                                                                       |
| `Ctrl+L`                       | 清屏。                                                                                                                                                                                                                                                                         |
| `Ctrl+O`                       | 切换紧凑模式（隐藏/显示工具输出和思考过程）。                                                                                                                                                                                                                                                 |
| `Ctrl+S`                       | 允许长响应完整输出，禁用截断。使用终端的滚动缓冲区查看完整输出。                                                                                                                                                                                     |
| `Ctrl+T`                       | 切换工具描述的显示。                                                                                                                                                                                                                                                  |
| `Ctrl+B`                       | 在前台 shell 命令运行时：将其提升为后台任务。子进程继续运行，agent 的回合解除阻塞，该 shell 显示在 `/tasks` 和后台任务对话框中。未执行 shell 时为空操作——`Ctrl+B` 此时回退到其提示区绑定（光标左移）。 |
| `Alt/Option+M`                 | 在富文本渲染预览和原始/源码模式之间切换 Markdown 输出。在 macOS 上，终端必须将 Option 键发送为 Meta。                                                                                                                                                               |
| `Shift+Tab`（Windows 上为 `Tab`） | 循环切换审批模式（`plan` → `default` → `auto-edit` → `auto` → `yolo`）                                                                                                                                                                                                                                 |

| ----------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `!`                                                   | 输入为空时切换 shell 模式。                                                                                          |

| `Ctrl+Enter` / `Cmd+Enter` / `Shift+Enter` / `Ctrl+J` | 插入换行符。                                                                                                                   |
| `Down Arrow`                                          | 向下移动一行，然后跳到末尾，然后切换到下一条历史记录。                                                                      |
| `Enter`                                               | 提交当前提示。                                                                                                          |
| `Meta+Delete` / `Ctrl+Delete`                         | 删除光标右侧的单词。                                                                                         |
| `Tab`                                                 | 如果存在当前建议，则自动补全。                                                                                  |
| `Up Arrow`                                            | 向上移动一行，然后跳到开头，然后切换到上一条历史记录。                                                                      |
| `Ctrl+A` / `Home`                                     | 将光标移动到行首。                                                                                       |
| `Ctrl+B` / `Left Arrow`                               | 将光标向左移动一个字符。                                                                                          |
| `Ctrl+C`                                              | 清空输入提示。                                                                                                              |
| `Esc`（双击）                                  | 清空输入提示。                                                                                                             |
| `Ctrl+D` / `Delete`                                   | 删除光标右侧的字符。                                                                                    |
| `Ctrl+E` / `End`                                      | 将光标移动到行尾。                                                                                             |
| `Ctrl+F` / `Right Arrow`                              | 将光标向右移动一个字符。                                                                                         |
| `Ctrl+H` / `Backspace`                                | 删除光标左侧的字符。                                                                                     |
| `Ctrl+K`                                              | 删除从光标到行尾的内容。                                                                                      |
| `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`      | 将光标向左移动一个单词。                                                                                               |
| `Ctrl+N`                                              | 向下移动一行，然后跳到末尾，然后切换到下一条历史记录。                                                                      |
| `Ctrl+P`                                              | 向上移动一行，然后跳到开头，然后切换到上一条历史记录。                                                                      |
| `Ctrl+R`                                              | 反向搜索输入/shell 历史记录。                                                                                         |
| `Ctrl+Y`                                              | 重试上一次失败的请求。                                                                                       |
| `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F`    | 将光标向右移动一个单词。                                                                                              |
| `Ctrl+U`                                              | 删除从光标到行首的内容。                                                                                |
| `Ctrl+V`（Windows 上为 `Alt+V`）                           | 粘贴剪贴板内容。如果剪贴板包含图片，将会保存该图片并在提示中插入对它的引用。 |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`        | 删除光标左侧的单词。                                                                                          |
| `Ctrl+X` / `Meta+Enter`                               | 在外部编辑器中打开当前输入。                                                                       |

## 建议

| 快捷键                | 描述                            |
| ----------------------- | -------------------------------------- |
| `Down Arrow` / `Ctrl+N` | 向下浏览建议。 |
| `Tab` / `Enter`         | 接受选中的建议。        |
| `Up Arrow` / `Ctrl+P`   | 向上浏览建议。   |

## 单选按钮选择

| 快捷键                      | 描述                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `Down Arrow` / `j` / `Ctrl+N` | 向下移动选择。                                                                                          |
| `Enter`                       | 确认选择。                                                                                            |
| `Up Arrow` / `k` / `Ctrl+P`   | 向上移动选择。                                                                                            |
| `1-9`                         | 按编号选择项目。                                                                                 |
| （多位数字）                 | 对于编号大于 9 的项目，快速连续按下数字键以选择对应项目。 |

## 历史记录滚动

仅在启用 `ui.useTerminalBuffer` 时生效（设置 → UI → 虚拟化历史记录）。在该模式下，对话历史渲染在应用内视口中，而不是宿主终端的滚动缓冲区，因此以下按键替代终端的原生滚动。

| 快捷键        | 描述                                          |
| --------------- | ---------------------------------------------------- |
| `Shift+Up`      | 历史记录向上滚动一行。                          |
| `Shift+Down`    | 历史记录向下滚动一行。                        |
| `PgUp`          | 历史记录向上滚动一页（视口高度）。        |
| `PgDn`          | 历史记录向下滚动一页（视口高度）。      |
| `Ctrl+Home`     | 跳转到对话顶部。                 |
| `Ctrl+End`      | 跳转到底部（并重新启用实时自动跟随）。 |
| **鼠标滚轮** | 滚动历史记录（每格 3 行）。                   |

当 `ui.useTerminalBuffer` 开启时，终端将鼠标事件转发给 qwen-code，使滚轮可以驱动应用内视口。副作用是**原生点击拖拽文本选择会被程序捕获**——拖拽时按住 `Shift`（或在 macOS Terminal / iTerm 上按住 `Option`）可绕过鼠标捕获，以通常方式选择文本。

### tmux 触控板滚动

在 tmux 内部，某些终端会在 qwen-code 接收到事件之前，将触控板或滚轮手势转换为普通的 `Up Arrow` 和 `Down Arrow` 序列。这些字节与真实的方向键按下完全相同，因此 qwen-code 无法判断你是要滚动视口还是浏览提示历史记录。

如果在 tmux 中触控板滚动改变了提示历史记录，请启用 `ui.useTerminalBuffer`；然后使用 `Shift+Up` / `Shift+Down`，或在 tmux 将滚轮事件转发给应用时使用鼠标滚轮。如果你偏好宿主终端滚动，请调整 tmux 中滚轮事件的鼠标绑定。
