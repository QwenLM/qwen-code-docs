# Qwen Code 键盘快捷键

本文档列出了 Qwen Code 中可用的键盘快捷键。

## 常用快捷键

| 快捷键                         | 说明                                                                                                                 |
| ------------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| `Esc`                          | 关闭对话框和建议。                                                                                                   |
| `Ctrl+C`                       | 取消当前请求并清空输入；连续按两次可退出应用。                                                                       |
| `Ctrl+D`                       | 当输入为空时退出应用；连续按两次以确认。                                                                             |
| `Ctrl+L`                       | 清屏。                                                                                                               |
| `Ctrl+O`                       | 切换调试控制台的显示状态。                                                                                           |
| `Ctrl+S`                       | 允许长响应完整输出（禁用截断），可通过终端的滚动回溯功能查看全部输出内容。                                           |
| `Ctrl+T`                       | 切换工具描述的显示状态。                                                                                             |
| `Shift+Tab`（Windows 上为 `Tab`） | 循环切换审批模式（`plan` → `default` → `auto-edit` → `yolo`）。                                                     |

## 输入提示

| 快捷键                                           | 说明                                                                                                                         |
| ------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| `!`                                              | 当输入框为空时，切换至 Shell 模式。                                                                                          |
| `?`                                              | 当输入框为空时，切换显示键盘快捷键列表。                                                                                      |
| `\`（行末）+ `Enter`                             | 插入换行符。                                                                                                                 |
| `向下箭头`                                       | 在输入历史中向下导航。                                                                                                       |
| `Enter`                                          | 提交当前提示。                                                                                                               |
| `Meta+Delete` / `Ctrl+Delete`                    | 删除光标右侧的整个单词。                                                                                                     |
| `Tab`                                            | 若存在建议项，则自动补全当前建议。                                                                                           |
| `向上箭头`                                       | 在输入历史中向上导航。                                                                                                       |
| `Ctrl+A` / `Home`                                | 将光标移至行首。                                                                                                             |
| `Ctrl+B` / `左箭头`                              | 将光标向左移动一个字符。                                                                                                     |
| `Ctrl+C`                                         | 清空输入提示。                                                                                                               |
| `Esc`（双击）                                    | 清空输入提示。                                                                                                               |
| `Ctrl+D` / `Delete`                              | 删除光标右侧的一个字符。                                                                                                     |
| `Ctrl+E` / `End`                                 | 将光标移至行尾。                                                                                                             |
| `Ctrl+F` / `右箭头`                              | 将光标向右移动一个字符。                                                                                                     |
| `Ctrl+H` / `Backspace`                           | 删除光标左侧的一个字符。                                                                                                     |
| `Ctrl+K`                                         | 删除从光标位置到行尾的所有内容。                                                                                             |
| `Ctrl+左箭头` / `Meta+左箭头` / `Meta+B`         | 将光标向左移动一个单词。                                                                                                     |
| `Ctrl+N`                                         | 在输入历史中向下导航。                                                                                                       |
| `Ctrl+P`                                         | 在输入历史中向上导航。                                                                                                       |
| `Ctrl+R`                                         | 反向搜索输入/Shell 历史记录。                                                                                                 |
| `Ctrl+Y`                                         | 重试上一次失败的请求。                                                                                                       |
| `Ctrl+右箭头` / `Meta+右箭头` / `Meta+F`         | 将光标向右移动一个单词。                                                                                                     |
| `Ctrl+U`                                         | 删除从光标位置到行首的所有内容。                                                                                             |
| `Ctrl+V`（Windows：`Alt+V`）                     | 粘贴剪贴板内容；若剪贴板中为图片，则自动保存该图片，并在提示中插入其引用。                                                   |
| `Ctrl+W` / `Meta+Backspace` / `Ctrl+Backspace`   | 删除光标左侧的整个单词。                                                                                                     |
| `Ctrl+X` / `Meta+Enter`                          | 在外部编辑器中打开当前输入内容。                                                                                             |

## 建议

| 快捷键         | 描述                     |
| -------------- | ------------------------ |
| `向下箭头`     | 在建议列表中向下导航。   |
| `Tab` / `Enter` | 接受当前选中的建议。     |
| `向上箭头`     | 在建议列表中向上导航。   |

## 单选按钮选择

| 快捷键             | 说明                                                                                                       |
| ------------------ | ---------------------------------------------------------------------------------------------------------- |
| `向下箭头` / `j`   | 向下移动选择项。                                                                                           |
| `Enter`            | 确认当前选择。                                                                                             |
| `向上箭头` / `k`   | 向上移动选择项。                                                                                           |
| `1-9`              | 按数字直接选择对应编号的选项。                                                                             |
| （多位数字）       | 对于编号大于 9 的选项，快速连续输入对应数字即可选择该选项。                                               |

## IDE 集成

| 快捷键   | 描述                             |
| -------- | -------------------------------- |
