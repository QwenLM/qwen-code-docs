# Qwen Code Beta : Utilisez l'IA pour écrire du code dans votre canal WeChat !

> **Bonne nouvelle pour les développeurs !** À partir d'aujourd'hui, Qwen Code prend officiellement en charge la fonctionnalité de canal WeChat. Plus besoin d'ouvrir un terminal ou un IDE—envoyez simplement un message sur WeChat, et laissez l'assistant IA vous aider à écrire du code, consulter la documentation et résoudre des problèmes techniques. Attention, attention ! Ce n'est pas une écrevisse, c'est la fonctionnalité Channel de Qwen Code ! Tout juste sortie du four : https://github.com/QwenLM/qwen-code/pull/2628

## 🎯 Pourquoi avez-vous besoin d'un canal WeChat ?

![Diagramme du concept principal du canal WeChat](https://img.alicdn.com/imgextra/i4/O1CN01p9WUdg27Z2P6L8UTB_!!6000000007810-0-tps-1376-768.jpg)

En tant que développeur, rencontrez-vous souvent ces scénarios :

- **Pendant votre trajet**, vous pensez soudain à une solution de bug et voulez que l'IA vérifie votre approche immédiatement
- **Entre deux réunions**, vous devez rapidement vérifier l'utilisation d'une API mais ne voulez pas ouvrir votre ordinateur
- **Tard dans la nuit**, l'inspiration vous frappe au lit, et vous voulez que l'IA génère un framework de code pour vous
- **En voyage**, vous rencontrez un problème urgent en ligne et devez aider vos collègues à distance

Auparavant, vous deviez ouvrir votre ordinateur et démarrer un terminal pour utiliser Qwen Code. Maintenant, **votre WeChat est votre environnement de développement**.

## 🚀 Guide de démarrage rapide

### Prérequis

Assurez-vous d'avoir :

- ✅ Installé la version preview de Qwen Code `npm i @qwen-code/qwen-code@0.14.0-preview.0 -g`
- ✅ Préparé un compte WeChat capable de recevoir des messages

> À l'ère de l'IA, qui configure encore les paramètres manuellement ? Vous avez juste besoin de `npx skills add https://github.com/QwenLM/qwen-code-examples --skill weixin-channel-setup` pour installer ce skill weixin-channel-setup, puis ouvrez Qwen et demandez : `/skills weixin-channel-setup Aidez-moi à configurer`

Si vous préférez la méthode de configuration traditionnelle, reportez-vous aux trois étapes ci-dessous :

![Diagramme des trois étapes de configuration](https://img.alicdn.com/imgextra/i4/O1CN01aWJWAF1YwqRJ6ryra_!!6000000003124-0-tps-1376-768.jpg)

### Étape 1 : Scanner pour se connecter

Exécutez dans le terminal :

```bash
qwen channel configure-weixin
```

Il y a une "URL de code QR" dans le terminal. Copiez le lien et ouvrez-le dans votre navigateur, puis scannez avec WeChat.

- Vous pouvez être invité à installer WeChat
  Après un scan réussi, vous verrez :

```
Connected to WeChat successfully!
Credentials saved. You can now start a weixin channel with:
  qwen channel start <name>
```

À ce stade, vos identifiants WeChat ont été sauvegardés en toute sécurité localement.

### Étape 2 : Configurer le canal

Éditez `~/.qwen/settings.json` et ajoutez la configuration suivante :

```json
"channels": {
    "my-weixin": {
      "type": "weixin",
      "senderPolicy": "open",
      "sessionScope": "user",
      "model": "qwen3.5-plus",
      "instructions": "You are a concise coding assistant responding via WeChat. Keep responses under 500 characters. Use plain text only."
    }
  }
```

### Étape 3 : Démarrer et tester

```bash
qwen channel start my-weixin
```

Après avoir vu le message `[Channel] "my-weixin" is running.`, ouvrez WeChat :

1. Trouvez votre assistant bot (celui que vous venez de scanner)
2. Envoyez un message : "Bonjour, aidez-moi à écrire un Hello World en Python"
3. Attendez la réponse...

🎉 **Félicitations ! Vous pouvez maintenant écrire du code avec WeChat !**

