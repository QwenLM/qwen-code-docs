## Qu'est-ce que Qwen Code Cowork ?

En simple, c'est un **assistant IA de bureau gratuit et open source**, construit sur la base du SDK Qwen Code.

Mais ce n'est pas juste une boîte de chat, c'est une IA qui peut **vraiment agir pour vous** :

| Capacité | Description |
|------|------|
| 📝 Écrire et éditer du code | Supporte tous les langages de programmation |
| 📁 Gérer les fichiers | Créer, déplacer, renommer, organiser |
| ⚡ Exécuter des commandes | Construire, tester, déployer |
| 💬 Répondre aux questions | Sur votre code ou n'importe quel sujet |
| 🔧 Automatiser les tâches | Tant que vous pouvez les décrire en langage naturel |

Le point clé : **totalement gratuit, open source et contrôlable, données stockées localement**.

En comparaison, Claude Code n'est disponible que pour les abonnés Max, à 100-200 $ par mois, tandis que Qwen Code Cowork offre des capacités équivalentes à zéro coût.

## Guide complet des cas d'utilisation

Voici quelques scénarios ultra-pratiques que j'ai découverts lors de mon utilisation, chacun accompagné d'une démonstration concrète.

### Scénario 1 : Organiser un bureau en pagaille / dossier de téléchargements

Le bureau et le dossier de téléchargements sont de véritables décharges numériques, où se mélangent captures d'écran, documents, installateurs et fichiers temporaires. À chaque fois que vous cherchez quelque chose, vous devez fouiller pendant des heures. C'est stressant à regarder, mais l'organisation manuelle prend trop de temps.

**Comment Cowork vous aide** :

Dites-lui simplement : « Aide-moi à organiser mon bureau, classer par type de fichier, les captures d'écran dans le dossier captures d'écran, les documents dans le dossier documents, les installateurs dans le dossier installateurs »

Il va :

1. D'abord scanner votre bureau et lister tous les fichiers
2. Vous présenter un aperçu du plan d'organisation
3. Après votre confirmation, exécuter automatiquement le classement et le déplacement

 
<Callout>
    **Astuce** :
    Utilisez le mode Plan pour laisser l'IA planifier d'abord, confirmez le plan avant d'exécuter, c'est plus sûr et contrôlable.
</Callout>

### Scénario 2 : Renommer des fichiers en masse (photos par date)

Les photos exportées depuis le téléphone ont des noms chaotiques, IMG_0001, IMG_0002... On ne peut pas savoir quand elles ont été prises. Vous voulez les renommer au format « 2024-01-15_Coucher de soleil à la plage », mais renommer manuellement des centaines de photos prendrait une éternité.

**Comment Cowork vous aide** :

Dites-lui : « Renomme les photos dans ce dossier par date de prise de vue, au format AAAA-MM-JJ_Numéro »

Il va :

1. Lire les informations EXIF de chaque photo (date de prise de vue)
2. Générer les nouveaux noms de fichiers selon le format spécifié
3. Prévisualiser toutes les opérations de renommage
4. Exécuter en masse après confirmation

### Scénario 3 : Vérifier l'espace de stockage local, trouver les gros fichiers

Votre ordinateur indique à nouveau que l'espace disque est insuffisant, mais vous ne savez pas ce qui occupe tout cet espace. Vous voulez nettoyer mais craignez de supprimer des fichiers importants par erreur, et vous hésitez à utiliser des logiciels de nettoyage par peur pour votre vie privée.

**Comment Cowork vous aide** :

Dites-lui : « Aide-moi à vérifier l'utilisation de l'espace de stockage de mon ordinateur, trouve les 10 dossiers et fichiers les plus volumineux, explique brièvement leur contenu, et indique s'ils peuvent être supprimés et les conséquences éventuelles »

Il va :

1. Analyser l'utilisation de votre disque
2. Lister les fichiers et dossiers les plus volumineux
3. Analyser ceux qui peuvent être supprimés en toute sécurité (comme les caches, fichiers temporaires)
4. Donner des recommandations de nettoyage

<Callout type="error">
    **Attention** :
    Confirmez toujours avant de supprimer des fichiers ! Il est recommandé de les déplacer d'abord vers la corbeille, puis de les supprimer définitivement après vérification.
</Callout>

### Scénario 4 : Intégration et formatage de notes

Vos notes sont dispersées un peu partout : favoris WeChat, mémos, Notion, Obsidian... Vous voulez les regrouper, mais les formats ne sont pas uniformes, et copier-coller manuellement tout en ajustant le formatage est trop fastidieux.

**Comment Cowork vous aide** :

Dites-lui : « Intègre tous les fichiers docx de ce dossier en un seul fichier Markdown, ordonnés par date de création, avec un formatage uniforme, les noms de fichiers affichés comme titres de niveau 2 »

Il va :

1. Lire le contenu de tous les fichiers
2. Identifier et uniformiser les formats
3. Trier et intégrer selon les règles spécifiées
4. Générer un fichier Markdown avec un formatage standard

### Scénario 5 : Traitement de documents en masse (extraction de contenu PDF)

Vous avez téléchargé une pile de documents PDF et voulez en extraire le texte pour prendre des notes, mais ouvrir et copier un par un prend trop de temps. Certains PDF sont même des scans, et le texte copié est illisible.

**Comment Cowork vous aide** :

Dites-lui : « Extrais le contenu de tous les fichiers PDF de ce dossier et sauvegarde-les dans des fichiers markdown correspondants »

Il va :

1. Parcourir tous les PDF du dossier
2. Extraire le contenu textuel
3. Générer un fichier txt pour chaque PDF
4. Rapporter les résultats du traitement

### Installation depuis les sources (avancé)

Si vous voulez construire depuis les sources, vous devez d'abord installer Bun :

```bash
# Installer Bun
curl -fsSL https://bun.sh/install | bash
exec /bin/zsh

# Cloner le dépôt
git clone https://github.com/QwenLM/qwen-code-examples

# Entrer dans le répertoire et installer
cd qwen-code-examples/apps/qwen-cowork
bun install

# Démarrer
bun run dev
```

## Astuces d'utilisation

### 1. Utiliser judicieusement le mode Plan

Pour les opérations de fichiers irréversibles, il est recommandé de laisser l'IA planifier d'abord :

```
« Aide-moi d'abord à analyser la situation de ce dossier, donne-moi un plan d'organisation, n'exécute pas directement »
```

Confirmez que le plan est correct avant de lui demander d'exécuter.

### 2. Spécifier le répertoire de travail

Vous pouvez spécifier le répertoire de travail lors de la création d'une session, ainsi l'IA n'opérera que dans ce répertoire, ce qui est plus sûr.

### 3. Confirmation requise pour les opérations sensibles

Cowork demandera votre confirmation avant les opérations sensibles comme la suppression ou le déplacement, pas de souci pour les erreurs de manipulation.
