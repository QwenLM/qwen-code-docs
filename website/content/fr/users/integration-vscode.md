# Visual Studio Code

> L'extension VS Code (bêta) vous permet de visualiser les modifications de Qwen en temps réel via une interface graphique native intégrée directement à votre IDE, facilitant ainsi l'accès et l'interaction avec Qwen Code.

### Fonctionnalités

- **Expérience IDE native** : Panneau latéral dédié Qwen Code accessible via l'icône Qwen
- **Mode d'acceptation automatique des modifications** : Applique automatiquement les modifications de Qwen au fur et à mesure
- **Gestion des fichiers** : Mentionnez des fichiers avec @ ou joignez des fichiers et images via le sélecteur de fichiers système
- **Historique des conversations** : Accès aux conversations précédentes
- **Sessions multiples** : Exécutez plusieurs sessions Qwen Code simultanément

### Prérequis

- VS Code 1.85.0 ou version ultérieure

### Installation

Téléchargez et installez l'extension depuis le [Visual Studio Code Extension Marketplace](https://marketplace.visualstudio.com/items?itemName=qwenlm.qwen-code-vscode-ide-companion).

## Dépannage

### L'extension ne s'installe pas

- Vérifiez que vous utilisez VS Code 1.85.0 ou une version ultérieure
- Vérifiez que VS Code dispose des autorisations nécessaires pour installer des extensions
- Essayez d'installer l'extension directement depuis le site web du Marketplace

### Qwen Code ne répond pas

- Vérifiez votre connexion Internet
- Démarrez une nouvelle conversation pour vérifier si le problème persiste
- [Signalez un problème sur GitHub](https://github.com/qwenlm/qwen-code/issues) si le problème persiste
