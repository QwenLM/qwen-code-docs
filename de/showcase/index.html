Tägliche Nutzung

Präsentation: PPT erstellen

PPT aus Produktdemo-Screenshots erstellen—genau das, was Sie gerade sehen. Geben Sie einfach Screenshots an, geben Sie den Befehl ein, warten Sie, und voila—bling bling fertig.

Programmierung

Papiere lesen

Papiere direkt aus dem Web lesen und herunterladen, wie dieses 'attention is all you need' Paper. Dann können Sie KI-Gespräche zum Lernen führen und sogar leicht verständliche Kernfragen-Kartenstile für intuitiveres Leseverständnis generieren.

Programmierung

Code-Lernen

Repositories direkt klonen zum Lernen und Verstehen—lassen Sie Qwen Code Ihnen sagen, wie Sie zu Open-Source-Projekten beitragen können, Schritt-für-Schritt-Operationen, vollständige Prozessverfolgung.

Programmierung

Issues lösen

Basierend auf dem oben genannten Plan können Sie beginnen, Issues für Open-Source-Projekte zu lösen—dies kann auch mit Qwen Code durchgeführt werden, und Sie können sogar Terminal-Screenshot-Uploads simulieren.

Programmierung

PR Review

Nach dem Lösen eines Issues und automatischen Einreichen eines PR können Sie Qwen Code auch direkt für PR-Tests und Reviews verwenden. Wenn es Ihr Projekt ist und jemand einen PR einreicht, können Sie auch Qwen Code damit reviewen lassen. Erhöhen Sie die Projektiterationsgeschwindigkeit erheblich.
