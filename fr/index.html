Bienvenue dans la documentation de Qwen Code

Cette documentation fournit un guide complet pour installer, utiliser et développer Qwen Code. Cet outil vous permet d’interagir avec des modèles d’IA via une interface en ligne de commande.

Aperçu

Qwen Code apporte les capacités de modèles de code avancés directement dans votre terminal, au sein d’un environnement interactif de type Read-Eval-Print Loop (REPL). Qwen Code se compose d’une application côté client (packages/cli) qui communique avec un serveur local (packages/core). Qwen Code inclut également divers outils pour effectuer des opérations sur le système de fichiers, exécuter des commandes shell, et récupérer des données depuis le web, tous gérés par packages/core.

Navigation dans la documentation

Cette documentation est organisée en plusieurs sections :

  • Exécution et déploiement : Informations pour exécuter Qwen Code.
  • Aperçu de l’architecture : Comprenez la conception de haut niveau de Qwen Code, y compris ses composants et leur interaction.
  • Utilisation du CLI : Documentation pour packages/cli.
    • Introduction au CLI : Aperçu de l’interface en ligne de commande.
    • Commandes : Description des commandes CLI disponibles.
    • Configuration : Informations sur la configuration du CLI.
    • Checkpointing : Documentation de la fonctionnalité de checkpointing.
    • Extensions : Comment étendre le CLI avec de nouvelles fonctionnalités.
    • Intégration IDE : Connectez le CLI à votre éditeur.
    • Télémétrie : Aperçu de la télémétrie dans le CLI.
  • Détails du Core : Documentation pour packages/core.
    • Introduction au Core : Aperçu du composant principal.
    • API des outils : Informations sur la façon dont le core gère et expose les outils.
  • Outils :
    • Aperçu des outils : Aperçu des outils disponibles.
    • Outils du système de fichiers : Documentation des outils read_file et write_file.
    • Outil de lecture multi-fichiers : Documentation de l’outil read_many_files.
    • Outil Shell : Documentation de l’outil run_shell_command.
    • Outil de récupération web : Documentation de l’outil web_fetch.
    • Outil de recherche web : Documentation de l’outil web_search.
    • Outil de mémoire : Documentation de l’outil save_memory.
  • Guide de contribution et de développement : Informations pour les contributeurs et développeurs, y compris l’installation, la construction, les tests et les conventions de codage.
  • Workspaces NPM et publication : Détails sur la gestion et la publication des packages du projet.
  • Guide de dépannage : Trouvez des solutions aux problèmes courants et FAQ.
  • Conditions d’utilisation et politique de confidentialité : Informations sur les conditions d’utilisation et les mentions de confidentialité applicables à votre utilisation de Qwen Code.

Nous espérons que cette documentation vous aidera à tirer le meilleur parti de Qwen Code !
