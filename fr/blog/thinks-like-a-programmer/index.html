Qwen Code : Un assistant de programmation IA qui pense comme un développeur

Qwen Team
30 janvier 2026

Si l’IA doit changer le monde, la première étape consiste à transformer le monde numérique. Qwen Code n’est pas qu’un outil en ligne de commande ; c’est un compagnon développeur que vous pouvez invoquer depuis votre terminal à tout moment.

Au cours de l’année écoulée, le paradigme de la programmation IA a considérablement évolué. Du mode Chat initial au mode de complétion de Copilot, jusqu’au mode Agent autonome d’aujourd’hui. Aujourd’hui, nous sommes fiers de présenter Qwen Code, un outil de programmation IA développé par Alibaba Tongyi Lab.

Il est open-source, gratuit et propulsé par le puissant modèle Qwen3-Coder. Non seulement il peut écrire du code via des interactions conversationnelles dans le terminal pour donner vie instantanément à vos inspirations, mais il s’intègre également profondément dans votre flux de développement pour corriger des bugs, refactoriser du code et gérer des tâches complexes avec aisance. De plus, c’est un compagnon polyvalent capable de rédiger des articles, dessiner des diagrammes et s’intégrer harmonieusement dans votre travail quotidien.

Qu’est-ce que Qwen Code ?

Nous avons réalisé le site de présentation du produit Qwen Code simplement par conversation. Tout au long du processus, Qwen Code pense et agit comme un véritable développeur :

  • Déconstruire les tâches : Crée une liste de tâches complète.
  • Écrire du code : Lit/écrit automatiquement des fichiers et exécute des scripts.
  • Auto-correction : Débogue automatiquement en cas d’erreurs.
  • Livrer des résultats : Livre non seulement du code, mais des applications Web complètes ou de la documentation.

Vous n’avez plus besoin de le guider pas à pas avec des instructions comme “écris cette fonction” ou “change cette variable”. Ce n’est plus un outil nécessitant de nombreux manuels ; c’est un véritable partenaire de codage, autonome et efficace.

Pourquoi choisir Qwen Code ?

1. Cœur puissant : Propulsé par Qwen3-Coder

Qwen Code est construit sur le puissant modèle Qwen3-Coder et est profondément optimisé pour celui-ci. Lors de la phase d’entraînement, nous avons permis au modèle de comprendre les outils, et le framework a été adapté au modèle. Il démontre des capacités exceptionnelles en génération de code, raisonnement et compréhension de contextes longs.

2. Véritable Agentic Workflow

Qwen Code offre des capacités avancées comme Skills, SubAgents et Plan Mode, rendant l’Agent plus autonome et fiable. Le Mode d’Approbation assure la sécurité du travail en permettant une confirmation étape par étape, ou vous pouvez activer le Mode YOLO pour laisser l’IA opérer avec pleine autonomie.

3. Couverture complète des scénarios : Partout où vous en avez besoin

Bien que Qwen Code soit né dans le terminal, il n’est pas limité par celui-ci. Il est suffisamment petit mais aussi suffisamment puissant pour exister dans cinq environnements distincts :

  • Environnement Terminal : Le choix des geeks. Lancez-le dans votre Terminal pour coder et créer dans une interface interactive élégante.
  • Environnement IDE : Le choix privilégié de la plupart des développeurs. Il prend en charge les éditeurs principaux comme VS Code, Qoder, Zed et JetBrains, offrant à la fois la commodité de l’interface graphique et la puissance de CodeAgent.
  • Environnement CI/CD : Utilisez qwen -p pour démarrer le Mode Headless. Vous pouvez mentionner @QwenCode dans GitHub Actions pour lui faire répondre automatiquement aux issues, reviewer du code ou soumettre des PRs.
  • Environnement Navigateur : Une fonctionnalité expérimentale qui intègre Qwen Code dans une extension de navigateur, lui permettant de percevoir l’état des pages web et d’analyser les paramètres – un excellent assistant pour le débogage et le scraping de données.
  • Environnement de Programmation : Utilisez le SDK Qwen Code pour apporter des capacités de codage intelligentes dans votre propre environnement, prenant actuellement en charge Node.js et Java.

4. Engagement envers l’Open Source et l’accès gratuit

Nous nous engageons dans une stratégie open-source et gratuite. Dans cette ère d’itération rapide du codage IA, nous croyons que l’open source offre la tranquillité d’esprit et que l’accès gratuit encourage les utilisateurs à explorer davantage de possibilités.

  • Utilisation gratuite : Connectez-vous via Qwen OAuth pour profiter d’un quota quotidien de 2 000 requêtes gratuites.
  • Compatibilité des protocoles : En tant que projet open-source, nous sommes compatibles avec les protocoles principaux comme Anthropic, Google Gemini et OpenAI.

Étude de cas : Qwen Code au-delà du codage

L’imagination de Qwen Code va bien au-delà de l’écriture de code. C’est également un assistant complet. Jetez un œil à une fonctionnalité communautaire populaire : Skills, et voyez comment Qwen Code devient une “aubaine” pour les créateurs de contenu :

Avec une Skill personnalisée, Qwen Code peut automatiquement télécharger des sous-titres YouTube, extraire des résumés, générer des illustrations et produire un article de blog complet.

Commencer maintenant

Qwen Code est désormais entièrement open-source sur GitHub avec 17,4k Stars. Démarrez en trois étapes simples :

  1. Installer (nécessite Node.js 20+) : 
    npm install -g @qwen-code/qwen-code@latest
  2. Lancer : 
    qwen
  3. Authentifier : Tapez /auth dans la session et sélectionnez Qwen OAuth pour commencer votre voyage de programmation IA.
