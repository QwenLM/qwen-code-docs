Qwen Code Hebdomadaire : Analytique Insight, Images du Presse-papiers, Terminal Capture

Qwen Team
2026-03-03

Cette semaine, nous avons publié les versions majeures v0.10.0, v0.11.0 et 5 versions de correction de bugs.

  • Publication de deux versions majeures : v0.10.0 et v0.11.0, plus 5 versions de correction de bugs
  • Authentification OAuth avec mise à niveau vers qwen3.5-plus, 1000 appels gratuits par jour
  • Ajout de l’analytique Insight
  • Support des images du presse-papiers
  • Terminal Capture validation automatisée et génération de captures d’écran

✨ Nouvelles Fonctionnalités

Analytique Insight

Commande intégrée officielle qui analyse en profondeur les enregistrements de Session locaux pour fournir des informations de programmation personnalisées et des rapports d’analyse d’utilisation.

Utilisation : Tapez /insight pour démarrer l’analyse LLM parallèle

Génère un rapport HTML détaillé divisé en 7 sections :

  • What You Work On - Votre focus de travail
  • How You Use Qwen Code - Comment vous utilisez Qwen Code
  • Impressive Things - Réalisations notables
  • Where Things Go Wrong - Zones problématiques
  • Features to Try - Fonctionnalités à essayer
  • New Usage Patterns - Nouveaux modèles d’utilisation
  • On the Horizon - Perspectives futures

Enfin, vous pouvez exporter une image pour le partage sur les réseaux sociaux.

👇 Démo de Fonctionnalité

Support des Images du Presse-papiers

Support multiplateforme : macOS / Windows / Linux. Voir PR #1612 

Fonctionnalités :

  • Capture d’Image : Capturer des images depuis le presse-papiers système
  • Stockage Temporaire : Sauvegarder dans le répertoire temporaire du presse-papiers
  • Nettoyage LRU : Suppression automatique des 50 images les plus anciennes lors du dépassement de 100
  • UI de Pièce Jointe : ↑ pour entrer en mode pièce jointe, ←/→ pour naviguer, Delete pour supprimer

Terminal-Capture

Objectif : Outil de capture d’écran de terminal automatisé pour la vérification visuelle CLI, valide automatiquement les fonctionnalités et produit des captures d’écran de démonstration.

Architecture : node-pty (pseudo-terminal) → xterm.js (rendu) → Playwright (capture d’écran), voir PR #1840 

👇 Démo de Fonctionnalité

Mises à Jour de Progression des Outils MCP

Fonctionnalité : Les outils MCP supportent les notifications de progression en temps réel

  • Mode TUI : Affichage de ⏳ [50/100] Processing...
  • Mode SDK : Nouveau type d’événement de flux tool_progress

Voir PR #1756
